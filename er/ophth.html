眼科急症

視力威脅性急症、眼眶感染與常見紅眼鑑別。化學灼傷 → 立刻大量沖洗優先於一切評估。

首頁 › 急診 › 眼科急症
眼部化學灼傷 Chemical Eye Injury
眼部化學灼傷是真正的眼科急症——立刻大量沖洗（2 公升以上）優先於所有評估和病史詢問。鹼性物質（石灰、氨水、NaOH）比酸性物質更危險（液化壞死 → 持續穿透眼球；酸性物質形成蛋白質凝固層限制穿透）。沖洗目標：結膜囊 pH 達到 7.0-7.4 並維持至少 30 分鐘後。
緊急處置
化學灼傷緊急處置
立刻大量沖洗！不要先量視力、不要先問病史。沖洗 = 最重要的治療。

立刻沖洗流程

  • 沖洗液：生理食鹽水（首選）；自來水（無 NS 時立刻使用）；Lactated Ringer's；不要用酸中和鹼（反應放熱 → 加重損傷）
  • 沖洗方式：點 Proparacaine（局部麻醉）→ 翻開眼瞼（含穹窿部 Fornix）→ 大量沖洗（Morgan lens 或 IV 管）× 至少 30 分鐘；翻開上下眼瞼沖洗穹窿部（化學物質積聚處）
  • 沖洗量：鹼性 → ≥2 L；酸性 → ≥1 L；不確定類型 → ≥2 L
  • pH 監測：沖洗停止後 5-10 分鐘測量結膜囊 pH；目標 pH 7.0-7.4；若未達目標 → 繼續沖洗
  • 移除隱形眼鏡（若有）；移除顆粒狀異物（石灰）用棉棒

Roper-Hall 分級（沖洗後評估）

分級角膜結膜 / 鞏膜缺血預後
I（輕度）角膜上皮損傷；基質清澈無缺血良好
II（中度）霧狀基質；虹膜可見<1/3 缺血良好
III（重度）全基質霧化；虹膜難見1/3-1/2 缺血不確定
IV（極重度）完全不透明；虹膜不可見>1/2 缺血差

沖洗後治療（眼科會診）

  • 局部類固醇（↓ 發炎）；局部抗生素（預防感染）
  • 散瞳劑（Cycloplegic，↓ 睫狀肌痙攣疼痛）
  • 局部 Ascorbic acid（Vit C）眼藥水（↓ 膠原酶活性，↑ 角膜癒合）
  • Grade III-IV → 住院眼科；Grade I-II 處置後可門診追蹤
急性閉角型青光眼 Acute Angle-Closure Glaucoma
急性閉角型青光眼（AACG）= 前房引流角突然關閉 → 眼內壓（IOP）急速升高（可達 40-80 mmHg，正常 <21 mmHg）→ 視神經缺血性損傷。典型表現：單側眼紅 + 眼痛 + 視力模糊 + 「看燈有彩虹光環（Halos）」+ 噁心嘔吐。容易被誤診為偏頭痛或腸胃炎！中等散大固定瞳孔是關鍵診斷線索。每小時都在傷害視神經——立刻降眼壓，緊急眼科會診。
機轉說明
病生理 + 臨床表現 + 誘因

病生理

  • 解剖危險因素：遠視（眼球小）；淺前房；窄引流角；老年（水晶體增大 → 前房更淺）；亞洲人（比白人高 3-4 倍）
  • 機轉：瞳孔中等散大（3-6 mm）→ 虹膜與水晶體貼合（瞳孔阻滯）→ 房水無法流入前房 → 後房壓力↑ → 虹膜向前膨出 → 引流角關閉 → IOP 急速升高

誘因

  • 暗室（瞳孔散大）；情緒激動；長時間低頭閱讀
  • 藥物誘發（散瞳）：抗膽鹼藥（Atropine、TCA、抗組織胺）；擬交感藥（Pseudoephedrine）；局部散瞳劑；Topiramate（特殊機轉）

臨床表現

  • 眼部：突發嚴重單側眼痛；視力模糊；「彩虹光環」（角膜水腫 → 稜鏡效果）；眼睛紅（睫狀充血）；眼球摸起來硬如石頭
  • 全身：噁心嘔吐（迷走神經反應，IOP 極高）；額頭痛；易誤診為偏頭痛或腸胃炎（關鍵：看眼睛！）
  • 身體檢查：結膜充血（睫狀充血，角膜緣周圍明顯）；角膜霧狀水腫；前房淺；中等散大固定瞳孔（4-6 mm，對光反應遲鈍）
  • IOP 測量：指觸法（眼球硬度）；Tonopen 或 Schiotz 眼壓計；正常 <21 mmHg；AACG 通常 >40 mmHg
用藥指引
緊急降眼壓治療
緊急眼科會診。多管齊下降眼壓，不要等單一藥物起效。目標 IOP <30 mmHg。

藥物降眼壓（同時使用多種）

體位 + 根治性治療

  • 仰臥位：重力使晶體後移 → 開放引流角（IOP ↓）；請患者平躺
  • 止吐止痛：Ondansetron（止吐）；Analgesics（疼痛控制）
  • 根治性治療（眼科）：雷射虹膜切開術（Laser Iridotomy）= 首選 → 在虹膜上開一個孔 → 消除瞳孔阻滯；藥物控制後盡快進行；對側眼預防性 Iridotomy（另一眼也有發作風險）
  • AACG 若漏診或延遲治療 → 數天內永久性視神經損傷；早期識別和緊急處置可完全恢復視力
視網膜剝離 + 視網膜動脈阻塞 Retinal Detachment & Arterial Occlusion
兩者均為無痛性急性視力喪失的重要病因。視網膜剝離（RD）：飛蚊症突增 + 閃光感 + 視野缺損（「窗簾遮住視野」）——累及黃斑部前緊急手術。視網膜中央動脈阻塞（CRAO）：突發無痛性單眼視力完全喪失（光感消失）；「Cherry-red spot」是特徵性眼底表現；治療視窗極短（<6-24 小時）；必須立刻排除 GCA（顳動脈炎）——若懷疑 GCA → 立刻大劑量類固醇，不要等活檢。
緊急處置
視網膜剝離（Retinal Detachment）

分類與危險因素

  • 孔源性（最常見，~85%）：視網膜裂孔 → 液體進入視網膜下方；高度近視（最常見危險因素）；老年；白內障術後；外傷；家族史
  • 牽引性：纖維血管增生膜牽拉（糖尿病視網膜病變、鐮刀型貧血）
  • 滲出性：無裂孔；炎症 / 腫瘤 / 中心性漿液性視網膜病變

臨床表現（依進展）

  • 前驅症狀：突然增多的飛蚊症（Floaters）；閃光感（Photopsia，玻璃體牽拉視網膜）
  • 視網膜剝離：周邊視野缺損（「窗簾 / 幕簾感」，從周邊向中心推進）；無痛性
  • 黃斑部累及：中心視力急速下降；閱讀困難；影像變形（Metamorphopsia）

分診與處置

  • 診斷：眼底鏡（直接或間接）；床邊超音波（B-scan）可在屈光介質混濁時診斷（視網膜剝離呈「海鷗徵」）；正式眼底檢查（眼科散瞳後）
  • 黃斑部未累及（Macula-on）：緊急（數小時內）眼科手術 → 視力恢復機率高
  • 黃斑部已累及（Macula-off）：急診（24 小時內）手術；功能性視力恢復機率↓（術後視力恢復有限）
  • 手術方式：玻璃體切除術（Vitrectomy）；鞏膜扣壓術（Scleral buckle）；氣體填充（Pneumatic retinopexy）
  • 單純飛蚊症增加 + 無視野缺損 → 玻璃體後脫離（PVD）可能；仍需眼科評估排除裂孔（裂孔可進展為 RD）
緊急處置
視網膜動脈阻塞（CRAO / BRAO）

分類

  • 視網膜中央動脈阻塞（CRAO）：完全視力喪失（僅存光感或無光感）；整個視網膜缺血；最嚴重
  • 視網膜分支動脈阻塞（BRAO）：部分視野缺損；預後較好
  • 睫狀視網膜動脈：約 20-25% 人有此變異血管（源自睫狀後動脈）→ CRAO 時可保留中心視力

病因

  • 栓塞（最常見）：頸動脈粥狀硬化斑塊（Hollenhorst plaque）；心臟栓子（AF、心內膜炎）
  • 巨細胞動脈炎（GCA）：佔 CRAO 1-2%，但雙眼視力喪失風險高 → 必須排除
  • 血栓；高凝狀態；血管痙攣（偏頭痛、可卡因）

眼底表現

  • Cherry-red spot：視網膜蒼白水腫（缺血）+ 黃斑部中心仍呈紅色（黃斑部由脈絡膜血液供應）= 經典表現
  • 視網膜動脈「車廂樣」（Boxcarring）血流
  • 視盤蒼白（視神經缺血）

急診處置

懷疑 GCA（年齡 >50、頭痛、下顎跛行、ESR↑）→ 立刻類固醇，不要等活檢結果。
  • 眼球按摩：間歇性按壓眼球（5 秒按 / 5 秒放）× 10-15 分 → ↑ 視網膜灌流 + 嘗試移動栓子；證據有限但無害
  • 前房穿刺（Paracentesis）：緊急降 IOP → ↑ 視網膜灌流；眼科手術
  • 吸入 CO₂ / 紙袋重呼吸：↑ PaCO₂ → 視網膜血管擴張；現已少用
  • Acetazolamide 500 mg IV：↓ IOP
  • tPA（靜脈或動脈內）：症狀發作 <4.5 小時；嚴格適應症；出血風險 vs 視力恢復效益；個案決策
  • 排除 GCA（年齡 >50）：詢問頭痛、下顎跛行、頭皮觸痛；查 ESR / CRP；懷疑 → 立刻 Prednisolone 1 mg/kg/天 IV 或 PO（不要等顳動脈活檢）

CRAO = 眼部中風（Ocular Stroke）

  • CRAO 後 90 天內缺血性中風風險 ~10%（類 TIA 風險）
  • 需完整中風評估：ECG（AF）；頸動脈超音波；心臟超音波；TTE；抗血小板治療
  • CRAO 視力預後差：自然病程下僅 1/3 有部分視力恢復；症狀 >24 小時幾乎無法恢復
眼球外傷 Ocular Trauma
眼球外傷分為開放性（Open Globe）和閉合性（Closed Globe）。開放性眼球傷（破裂 / 穿刺傷）= 眼科急症——不要加壓眼球、不要移除嵌入異物、戴眼罩保護（Rigid Shield）、禁食備手術、緊急眼科會診。前房出血（Hyphema）= 血液積聚在前房，需限制活動 + 抬高頭部 + 眼科追蹤（↑ IOP 風險）。
緊急處置
開放性眼球傷（Open Globe Injury）
不要加壓眼球！不要移除嵌入異物！硬式眼罩保護 → 禁食 → 緊急眼科手術。

辨識開放性眼球傷

  • 症狀：眼痛（有時意外地輕微）；視力急速下降；外傷史（高速飛來異物、爆炸、刀刺、槍傷）
  • 身體檢查線索：眼球形狀不規則 / 塌陷；可見眼內容物（玻璃體、葡萄膜、水晶體）脫出；前房消失或極淺；Seidel 試驗陽性（螢光素 + 鈷藍光 → 看到房水從傷口流出稀釋螢光素呈「瀑布狀」）；眼壓極低
  • 若懷疑開放性眼球傷 → 不做 Tonometry（眼壓計）測量（加壓可能擠出眼內容物）

急診處置

  • 保護：硬式眼罩（Rigid Eye Shield，非眼墊）→ 避免任何壓迫；嵌入異物不移除（用杯子保護）
  • 禁食：備緊急手術
  • 止吐：Ondansetron → 避免嘔吐時的 Valsalva → 避免眼內容物脫出
  • 鎮靜：必要時（兒童、不合作患者）；避免用力哭鬧
  • 抗生素：Ciprofloxacin 或 Moxifloxacin（預防眼內炎）；農場外傷或有機物 → 加 Clindamycin（厭氧菌）
  • 破傷風預防；眼球 X 光或 CT（懷疑球內金屬異物；不做 MRI！）
  • CT 眼眶：評估眼眶骨折、眼球位置、球內異物；MRI 禁忌（若有金屬異物 → 移動 → 加重損傷）

用藥指引
前房出血（Hyphema）+ 角膜擦傷 / 異物

前房出血（Hyphema）

  • 分級：Grade 0（微量）→ Grade IV（完全充滿，「八球眼」Black Ball / Eight-Ball）
  • 併發症：↑ IOP → 繼發性青光眼；角膜血染（Grade IV + IOP↑）；再出血（傷後 2-5 天，預後差）；鐮刀型貧血患者特別高危（低 O₂ 張力 → ↑ 鐮刀化）
  • 處置：頭部抬高 30-45°；眼罩保護；限制活動（避免 Valsalva）；Cycloplegic（散瞳）↓ 睫狀肌痙攣；局部 β-blocker / Acetazolamide（IOP↑ 時）；避免 NSAIDs / ASA（↑ 再出血）；鐮刀型貧血 → Acetazolamide 不用（↑ 鐮刀化）
  • Grade I-III → 多數可門診追蹤（眼科每天評估）；Grade IV 或再出血 → 手術沖洗

角膜擦傷（Corneal Abrasion）

  • 症狀：劇烈疼痛（角膜神經豐富）；異物感；畏光；流淚；視力可正常或輕度模糊
  • 診斷：螢光素染色（Fluorescein）+ 鈷藍光 → 擦傷區域呈亮綠色；裂隙燈評估
  • 治療：局部抗生素眼藥水（Tobramycin 或 Ciprofloxacin）× 3-5 天；Cycloplegic（散瞳，↓ 疼痛）；NSAID 眼藥水（↓ 疼痛）；可視需要口服止痛；眼墊（Eye Patch）不建議常規使用（↑ 感染風險；不縮短癒合時間）
  • 隱形眼鏡使用者 → 懷疑 Pseudomonas 感染 → 加 Ciprofloxacin 眼藥水；不要戴回隱形眼鏡直到完全癒合

角膜 / 結膜異物

  • 結膜異物：翻開眼瞼（包含上眼瞼穹窿部）→ 濕棉棒或沖洗移除
  • 角膜異物（非穿透性）：裂隙燈下用 25G 針頭或異物鑷移除；金屬異物 → 48 小時後「鏽環（Rust Ring）」→ 需眼科用旋轉刮除器（Alger brush）
  • 移除後 → 同角膜擦傷處置；抗生素眼藥水 × 3-5 天
眼眶蜂窩組織炎 Orbital Cellulitis
眼眶蜂窩組織炎最重要的鑑別是眶隔前（Preseptal）vs 眶隔後（Postseptal / Orbital）——眶隔是解剖屏障，眶隔後感染可擴散至眼眶、海綿竇、腦膜。鑑別關鍵：眼球突出（Proptosis）、眼球運動受限和疼痛、視力下降 = 眶隔後感染，需立刻 CT 眼眶 + 住院 IV 抗生素。副鼻腔炎（尤其篩竇炎）是最常見病因。海綿竇栓塞性靜脈炎 = 最嚴重的併發症，死亡率極高。

抗生素選擇（眶隔後）

  • 一線（IV）：Ampicillin-sulbactam 3 g IV q6h；或 Piperacillin-tazobactam（嚴重 / 懷疑厭氧菌）
  • MRSA 高風險（社區獲得、鼻腔攜帶、穿透傷）：加 Vancomycin

緊急處置
海綿竇栓塞性靜脈炎（Cavernous Sinus Thrombosis）
  • 來源：面部感染（鼻周「危險三角形」）；眶隔後蜂窩組織炎蔓延；副鼻腔炎（蝶竇炎尤其）；牙源性
  • 「危險三角形」：上脣到鼻根的面部區域 → 靜脈引流入海綿竇（無靜脈瓣膜）→ 擠壓 / 切開此區域的膿瘡 → 菌血症 → 海綿竇血栓

臨床表現

  • 高燒；嚴重頭痛；眼瞼水腫；眼球突出
  • 雙側眼部受累（高度特異）：眶隔後蜂窩組織炎通常單側；海綿竇血栓可擴散至對側
  • 顱神經麻痺（CN III、IV、V1/V2、VI）→ 眼球運動障礙、複視、顏面感覺改變
  • 乳突後腦膜炎徵象（頸部僵硬）
  • 意識改變（腦膜炎 / 敗血症）

診斷與治療

  • MRI + MRV（首選）：直接顯示海綿竇血栓；CT 靜脈攝影替代
  • IV 抗生素：廣效（Vancomycin + Piperacillin-tazobactam 或 Meropenem）；Metronidazole（厭氧菌）；長療程 3-4 週
  • 抗凝（Heparin）：爭議中；部分研究顯示 ↓ 死亡率和神經後遺症；個案決策
  • 類固醇：爭議中；嚴重腦水腫時考慮
  • ICU 入住；神經外科和眼科會診
  • 死亡率歷史上 >30%（現代抗生素後下降）；倖存者常有永久性神經後遺症
葡萄膜炎 + 鞏膜炎 Uveitis & Scleritis
葡萄膜炎（虹膜睫狀體炎）和鞏膜炎是兩種需要鑑別的眼部發炎性疾病，常與全身性自體免疫疾病相關。前葡萄膜炎（虹膜炎）是最常見類型——急性發作：眼痛 + 畏光 + 視力模糊 + 睫狀充血（Ciliary flush）+ 前房細胞和閃輝（Cells and Flare）。鞏膜炎比鞏膜外層炎更嚴重——深層鞏膜受累 + 劇烈疼痛（可放射至頭部 / 下顎）；50% 合併全身性疾病（最常見 RA）。

壞死性鞏膜炎（最嚴重）

  • 鞏膜壞死 → 葡萄膜暴露（藍黑色）；RA 患者高危（「類風濕性鞏膜炎」）

  • 鞏膜修補手術（穿孔時）

紅眼鑑別診斷 Red Eye Differential Diagnosis
紅眼是急診最常見的眼科主訴之一。大多數是良性（結膜炎），但必須排除視力威脅性原因。關鍵鑑別問題：有無視力改變？有無眼痛？瞳孔正常嗎？角膜清澈嗎？「視力正常 + 無痛 + 分泌物」= 通常結膜炎；「視力↓ + 眼痛 + 瞳孔異常 + 角膜混濁」= 需緊急評估。

充血類型鑑別

  • 結膜充血（Conjunctival Injection）：周邊紅、角膜緣較淡；血管可移動；通常結膜炎
  • 睫狀充血（Ciliary Flush / Circumcorneal Injection）：角膜緣周圍深紅 / 紫紅；向周邊漸淡；血管不移動；通常角膜炎 / 葡萄膜炎 / 青光眼
  • 結膜下出血（Subconjunctival Hemorrhage）：鮮紅、清楚邊界；通常 Valsalva、外傷或自發性（老年 / 高血壓）；無需治療（2-3 週自行吸收）；雙側 + 外傷 → 排除顱底骨折

急性視力喪失 Acute Visual Loss
急性視力喪失的評估首先區分：單眼 vs 雙眼、有痛 vs 無痛、暫時性 vs 持續性。一過性黑矇（Amaurosis Fugax）= 暫時性單眼視力喪失（如窗簾拉下），通常數分鐘內恢復，是頸動脈粥狀硬化栓子的警示訊號（眼部 TIA）——中風風險極高，需立刻中風評估。雙眼視力喪失通常是後循環問題（枕葉、腦幹）或全身性疾病（GCA、嚴重低血壓）。相對性瞳孔傳入缺損（RAPD / Marcus Gunn 瞳孔）是視神經或視網膜嚴重病變的客觀指標。

初步評估三問

  • 1. 單眼 vs 雙眼？患者遮住一眼測試
  • 2. 有痛 vs 無痛？有痛 → 青光眼、角膜炎、葡萄膜炎、視神經炎；無痛 → 視網膜血管阻塞、視網膜剝離、玻璃體出血
  • 3. 暫時性 vs 持續性？暫時性 → 血管性（TIA、痙攣）；持續性 → 結構性或完全阻塞

RAPD 檢查（Swinging Flashlight Test）

  • 手電筒交替照兩眼；正常：兩眼瞳孔反應對稱
  • RAPD 陽性：光線照患側眼 → 兩眼瞳孔散大（傳入缺損）→ 視神經 / 嚴重視網膜病變的客觀指標
  • RAPD 陽性 + 視力↓ → 嚴重單側視神經或視網膜疾病；需緊急眼科評估
用藥指引
視神經炎（Optic Neuritis）
  • 典型表現：單側視力急速↓（數天進展）；眼球轉動時疼痛（特徵性）；色覺異常（紅色脫飽和）；RAPD ⊕；中央暗點
  • 眼底：50% 視乳頭腫脹（球後視神經炎無腫脹 = 「醫師什麼都看不見，病人什麼都看不見」）
  • 人口學：女性 > 男性；20-45 歲高峰；常為 MS 首發表現（25%）
  • 相關：MS（最常見）；NMOSD（Anti-AQP4 / Anti-MOG 抗體）；結節病；梅毒；萊姆病；病毒感染後

評估與治療

  • MRI 腦部（含 gadolinium）：評估 MS 病灶（白質高信號）；視神經強化（急性）；排除其他病因
  • 視覺誘發電位（VEP）：P100 延遲 = 脫髓鞘客觀證據；即使無視神經炎症狀也可異常
  • IV 甲基 Prednisolone 1 g/天 × 3 天：↑ 視力恢復速度（2 週）；不改變 6 個月後最終視力；↓ MS 轉化風險（若 MRI 有病灶）
  • 口服 Prednisolone（中等劑量）：反而 ↑ 復發率（避免）
  • NMOSD 視神經炎 → 更積極免疫抑制（見神經科 MS block）
  • 視力恢復：80-90% 在 6 個月內顯著恢復（通常自限）；長期色覺和對比敏感度可能持續異常

巨細胞動脈炎（GCA）視力威脅
  • 年齡：幾乎只發生於 >50 歲（高峰 70-80 歲）；女 > 男（2:1）
  • 視力威脅機轉：睫狀後動脈炎症 → 前部缺血性視神經病變（AION）→ 失明；視網膜動脈阻塞；顱神經缺血

臨床線索

  • 新發頭痛（顳部）；頭皮觸痛（梳頭時疼痛）
  • 下顎跛行（Jaw Claudication）：咀嚼時下顎肌肉疼痛（高特異性）
  • 顳動脈壓痛 / 結節 / 脈搏消失
  • 一過性視力模糊（警示！下一步可能完全失明）
  • 合併風濕性多肌痛（PMR）：肩胛帶 / 骨盆帶疼痛 + 僵硬
  • 實驗室：ESR >50 mm/hr（通常 >100）；CRP ↑；貧血（正常球形）；血小板 ↑

處置

  • 懷疑 GCA + 視力症狀 → 立刻 IV Methylprednisolone 1 g/天 × 3 天（不要等活檢！）
  • 無視力症狀 → Prednisolone 40-60 mg/天 PO
  • 顳動脈活檢：黃金標準診斷；類固醇不影響 48-72 小時內的活檢結果；取至少 2-3 cm 長度（跳躍性病灶）；顳動脈超音波（「暗環」）替代
  • 類固醇長期漸減（12-18 個月）；Tocilizumab（IL-6 抑制劑）↓ 類固醇需求（難治性 GCA）
  • ASA 81 mg（↓ 視覺和中風併發症；爭議中但常使用）
  • 一眼失明後對側眼失明風險 >25%（未治療）；類固醇治療後失明風險大幅降低但已失明的視力通常無法恢復
