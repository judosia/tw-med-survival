熱傷害 Heat Illness
熱傷害從輕微（熱痙攣）到致命（熱中暑）。熱中暑（Heat Stroke）= 核心體溫 >40°C + CNS 功能障礙——死亡率高達 10-80%（未治療）。立刻降溫是最重要的治療，每延遲 1 分鐘死亡率和後遺症風險持續增加。降溫目標：30 分鐘內將體溫降至 38-39°C。勞動型熱中暑（Exertional）多見年輕人；傳統型（Classic）多見老年、慢性病患者和熱浪期間。

熱傷害分類
類型體溫臨床特徵處置
熱痙攣（Heat Cramps） 正常 大量出汗後肌肉痛性痙攣（小腿、腹肌）；低鈉血症 口服或 IV 生理食鹽水；避免純水（加重低鈉）；休息 + 移至陰涼處

熱衰竭（Heat Exhaustion） <40°C 大量出汗；頭痛、頭暈、噁心、虛弱；皮膚濕冷；心搏過速；無 CNS 功能障礙（關鍵！） 移至陰涼處；脫衣服；冷水擦拭；口服或 IV 補液（NS）；監測進展為熱中暑
熱中暑（Heat Stroke） >40°C CNS 功能障礙（混亂、激動、昏迷）+ 高體溫；勞動型：大量出汗；傳統型：皮膚可為乾熱 立刻積極降溫（見下方）；ICU 入住
  • 熱衰竭 vs 熱中暑的關鍵鑑別 = 有無 CNS 功能障礙；兩者體溫可重疊 → 意識狀態是決定因素

熱中暑緊急處置
目標：30 分鐘內將核心體溫降至 38-39°C。降溫優先於其他處置。

降溫方法（依效率排序）

方法降溫速率說明
冷水浸泡（Cold Water Immersion，CWI）最快（0.2°C/分）勞動型熱中暑首選；冷水（1-15°C）浸泡至頸部；傳統型老年患者可能耐受差
蒸發降溫（Evaporative Cooling）快溫水（15°C）噴霧 + 持續強力風扇；適合傳統型熱中暑；院前和急診均可
冰袋放置（腋下、腹股溝、頸部）中等輔助性；大血管區域更有效
冷鹽水 IV（4°C）中等輔助；同時達到補液效果
體內冷卻（胃 / 膀胱 / 腹膜沖洗）有效但侵入性高嚴重難治時使用；體外循環（ECMO）= 最後手段

其他處置

  • ABC：意識改變 → 考慮插管（保護氣道 + 防止誤吸）
  • 補液：IVF（NS 或 LR）；避免過度補液（↑ 腦水腫風險）；監測 UOP
  • 癲癇：BZD 首選；降溫通常可控制癲癇
  • 退燒藥（Paracetamol / NSAIDs）無效：熱中暑是體溫調節失常，非發炎性發燒 → 退燒藥無效
  • Dantrolene：無效（非惡性高體溫 / NMS 機轉）

併發症監測

  • 橫紋肌溶解：↑ CK；Myoglobinuria（尿液深褐色）→ 積極補液維持 UOP；勞動型熱中暑尤其常見
  • DIC：熱損傷直接活化凝血；監測 PT / PTT / Fibrinogen / D-dimer
  • 急性腎衰竭：橫紋肌溶解 + 低灌流 → 監測 Cr；嚴重 → 透析
  • 肝衰竭：熱直接損傷肝細胞；24-72 小時後 ALT/AST 峰值
  • 電解質：低鈉血症（大量出汗後大量喝水）；低鉀血症；低磷血症
  • 核心體溫需用直腸溫度計（耳溫 / 腋溫 / 口溫不準確）
低體溫症 Hypothermia
低體溫定義為核心體溫 <35°C。「不到溫暖就不算死（Not dead until warm and dead）」——心跳停止的低體溫患者應積極復甦至核心體溫 >32-35°C 才能宣告死亡。低體溫可保護大腦（↓ 代謝需求），已有核心體溫 9°C 存活的案例。心律不整（最危險：VF）在 <28°C 時風險大增，Lidocaine 等抗心律不整藥物在低體溫下效果差且可能有毒。
數值判讀
低體溫分級與臨床表現
分級核心體溫臨床表現
輕度（HT I） 32-35°C 清醒；顫抖（最大代謝產熱）；心搏過速；高血壓；血管收縮
中度（HT II） 28-32°C 意識改變（嗜睡 → 混亂）；顫抖停止（代謝耗竭）；肌肉無力；反射↓；心律不整開始（AF 最常見）
重度（HT III） 24-28°C 昏迷；反射消失；肌肉僵硬；Osborn（J）波（ECG）；VF 風險↑；血壓難以測量
極重度（HT IV） <24°C 無脈搏；呼吸停止；VF 或 Asystole；臨床死亡外觀

ECG 表現

  • Osborn（J）波：QRS 波群末端和 ST 段起點之間的正向偏折波；低體溫特徵性但非特異性；體溫越低越明顯
  • PR、QRS、QTc 間隔延長；竇性心動過緩；AF；VF（<28°C 時）；Asystole（極重度）

誘因

  • 環境暴露（寒冷 + 濕衣服 + 風）
  • 酒精（↑ 周邊血管擴張 + ↓ 顫抖感知）
  • 甲狀腺低下；腦下垂體功能低下；腎上腺功能不足
  • 脊椎損傷（失去顫抖能力）；燒傷；感染 / 敗血症
  • 藥物：β-blocker、BZD、鴉片類、吩噻嗪類
緊急處置
復溫治療

一般處置

  • 移至溫暖環境；移除濕衣服；輕柔搬運（避免誘發 VF）
  • 核心體溫測量：直腸溫度計或食道溫度探針（耳溫 / 腋溫不可靠）
  • 心電監護（連續）；IV 建立（溫熱的 NS 或 LR）
  • 低血壓：IVF（38-42°C 加溫輸液）；升壓藥效果差（血管對兒茶酚胺反應↓）

復溫策略

主動體外復溫（AEW） 中度（HT II） 熱毯、加溫墊（放軀幹，非四肢）；熱空氣強制對流毯（Bair Hugger）；溫水浸泡

ECMO（體外循環） 心跳停止 + 無灌注心律（HT IV） 最快復溫方式（1-2°C / 分）；同時維持循環；有 ECMO 的醫院優先轉送

心跳停止的低體溫患者

  • 「不到溫暖就不算死」——積極 CPR 直到核心體溫 >32-35°C
  • CPR：低體溫 → 代謝需求↓ → 大腦耐受缺氧更久；持續 CPR 直到復溫
  • 除顫：<30°C → VF 難以除顫；先復溫到 30°C 再嘗試；每個體溫段最多 1 次電擊（若無效則繼續 CPR + 復溫）
  • 藥物：<30°C 時 Epinephrine 和抗心律不整藥物效果差且蓄積毒性 → 體溫 >30°C 後才給；間隔時間加倍
  • K⁺ >12 mEq/L（低體溫心跳停止）→ 預後極差 → 考慮終止復甦
  • 四肢復溫慎用（「復溫後休克」：周邊冷血回流中央 → 心臟溫度再下降 → VF）→ 優先復溫軀幹
凍傷 Frostbite & Cold Injuries
凍傷是寒冷直接導致組織冰晶形成和血管損傷。最重要的原則：確認患者已離開寒冷環境且不會再暴露後才快速復溫——「解凍後再冷凍」比持續凍傷更嚴重。快速復溫用 37-40°C 溫水浸泡（非過熱水！）。Ibuprofen 和 Aloe Vera 可減少炎症前列腺素。嚴重凍傷（III-IV 度）合併血管血栓 → tPA 溶栓治療可顯著減少截肢率，需在凍傷後 24 小時內給藥。
機轉說明
寒冷損傷分類

非凍傷性寒冷損傷

  • 凍瘡（Chilblains / Pernio）：反覆暴露於寒冷（非冰點）+ 潮濕 → 炎症性皮膚病變；手指、足趾、耳廓出現紅斑、搔癢、疼痛丘疹；通常可逆；治療：保暖、Nifedipine（嚴重）
  • 戰壕腳（Trench Foot / Immersion Foot）：長時間暴露於寒冷（非冰點）+ 潮濕；足部水腫、麻木、水皰；治療：乾燥、復溫、傷口照護

凍傷（Frostbite）分度

分度深度外觀預後
一度（表淺） 表皮 紅斑、水腫、麻木；無水皰；復溫後感覺異常（燒灼感） 完全恢復
二度（部分皮層） 真皮淺層 清澈水皰（含漿液）；周圍紅斑水腫；復溫後劇痛 通常恢復（可能有感覺過敏）
三度（全皮層） 真皮全層 出血性水皰（血皰）；皮膚呈深紅 / 藍紫色；硬如木頭 部分組織壞死；可能截肢
四度（全厚） 皮下組織 + 肌肉 + 骨骼 完全無感覺；黑色乾性壞疽；皮革樣 截肢幾乎必然
  • 凍傷後立即外觀很難預測最終結果——急性期的「黑色」組織不一定是壞死（可能是血管痙攣）→ 等待數週再做截肢決定（「January frostbite, July amputation」）
用藥指引
凍傷現場與急診處置

現場處理原則

  • 確認離開寒冷環境且不會再暴露 → 才進行復溫（再凍比持續凍傷更壞）
  • 移除濕衣物、珠寶；保護凍傷肢體（用輕柔的敷料包裹）
  • 不要：摩擦凍傷部位；在野外打破水皰；用雪摩擦；直接用乾熱（火、電熱器）
  • 若必須步行才能脫困 → 不要解凍腳部（凍傷的腳承重比解凍後好）

急診快速復溫

  • 溫水浸泡（37-40°C）30-45 分鐘：直到組織柔軟、紅潤，感覺恢復；可加 Hexachlorophene 或 Povidone-iodine 抗感染；不要超過 40°C（加重損傷）
  • 鎮痛（復溫過程非常痛）：IV Opioid 或 Ketorolac
  • Ibuprofen 400-600 mg q8-12h PO（↓ 花生四烯酸代謝物 → ↓ 炎症和血管痙攣）
  • Aloe Vera 外用（Thromboxane 抑制劑 → ↓ 血管痙攣）
  • 保持患肢高舉；輕柔敷料；保護水皰（清澈水皰不要打破；血皰可抽吸液體但保留外皮）
  • 破傷風預防；抗生素（若有感染跡象）

嚴重凍傷（III-IV 度）進階治療

  • tPA 溶栓（凍傷後 <24 小時）：血管攝影確認血管血栓 → tPA 動脈內或靜脈溶栓；↓ 截肢率高達 50-75%；禁忌：出血風險（同一般 tPA 禁忌）
  • Iloprost（前列腺素 I₂ 類似物）：靜脈輸注 6 小時 / 天 × 5 天；↓ 血管痙攣 + 血小板凝集；歐洲使用較廣
  • 高壓氧（HBO）：輔助治療，台灣部分醫院可用
  • 核子醫學（Tc-99m 掃描）可在 2-5 天後評估組織存活性，幫助決定截肢時機
  • 截肢決定應延遲至少 3-6 週（組織最終壞死界線需時間才能確定）
溺水 Drowning
現代定義：溺水 = 因浸入 / 沉浸於液體中導致呼吸受損。取消「乾性 / 濕性溺水」「近乎溺水」等舊術語。死亡原因是缺氧，不是水吸入本身——現場 CPR 是最重要的介入，每延遲 1 分鐘開始 CPR 存活率↓ 10%。「續發性溺水（Secondary Drowning）」= 溺水後 4-6 小時出現肺水腫症狀，所有有症狀的溺水患者需觀察至少 6-8 小時。
機轉說明
病生理與臨床表現

病生理

  • 換氣不足 → 低氧血症 → 意識喪失 → 誤吸 → 肺損傷
  • 喉頭痙攣（「乾性溺水」，~10-15%）：聲門痙攣阻止水進入 → 仍死於缺氧
  • 水吸入（~85-90%）：表面張力活性劑（Surfactant）破壞 → 肺泡塌陷 → V/Q 不均 → 肺水腫 → ARDS
  • 淡水 vs 鹽水：臨床差異不大（兩者均導致 Surfactant 損壞和低氧血症）；電解質差異通常無臨床意義（除非吸入量極大）
  • 低體溫（冷水溺水）：保護大腦（↓ 代謝需求）→ 更長復甦視窗；心跳停止 + 冷水溺水 → 積極復甦

緊急處置
現場急救與急診處置

現場急救

  • 將患者移出水中（施救者安全優先）
  • 立刻評估呼吸和脈搏 → 無呼吸 → 立刻開始 CPR
  • 呼吸急救（口對口或 BVM）優先於胸部按壓（溺水以缺氧為主）
  • 不要嘗試「排水」（Abdominal Thrusts）——無效且浪費時間
  • 懷疑脊椎損傷（跳水事故、衝浪）→ 頸椎保護；但不要因此延遲 CPR
  • 低體溫患者 → 保暖；CPR 持續至醫院（「不到溫暖就不算死」）

急診處置

  • 氧氣：高流量 O₂（NRM 面罩）；SpO₂ 目標 ≥95%
  • 輕中度低氧：高流量 O₂ → CPAP / BiPAP（↑ 功能殘氣量，改善肺泡通氣）
  • 嚴重低氧 / 意識改變：插管 + 機械通氣；ARDS 保護性通氣策略（TV 6 mL/kg；PEEP 8-12 cmH₂O；Pplat <30 cmH₂O）
  • 支氣管痙攣：霧化 Albuterol
  • 補液：謹慎（肺水腫 → 避免過度補液）；低血壓 → 小量 IVF 試驗
  • 低體溫：依低體溫程度復溫

觀察和出院標準

  • 所有有症狀患者 → 觀察至少 6-8 小時（「續發性溺水」風險）
  • 可出院標準（同時滿足所有）：清醒 + 定向感完整；SpO₂ >95%（室內空氣）；呼吸功增加無症狀；CXR 正常；6-8 小時觀察期間無惡化
  • 住院適應症：任何低氧血症（需吸氧）；意識改變；CXR 異常；嚴重低體溫；心律不整
  • 預防性抗生素：不建議常規使用；污染水域（下水道、農業廢水）→ 考慮廣效抗生素

預後預測

  • 良好預後因素：冷水（低體溫保護）；目擊者立刻 CPR；急救反應時間短；年輕患者
  • 不良預後因素：水中時間 >25 分鐘；CPR >25 分鐘（除非低體溫）；到院時仍在 CPR；Glasgow <5；瞳孔固定散大（復溫後評估）
  • 不要在急診宣告溺水 + 低體溫心跳停止患者死亡 → 轉至有 ECMO 能力的醫院後才評估
高山症 Altitude Illness
高山症三大類：急性高山症（AMS）→ 高海拔腦水腫（HACE）→ 高海拔肺水腫（HAPE），嚴重度遞增。最重要的治療是下降高度（Descent）——任何嚴重高山症的首要處置，其他治療是輔助。HAPE 是高山症中最常見的死亡原因（非心因性肺水腫）。台灣玉山（3,952 m）和雪山（3,886 m）是常見發生地點。Acetazolamide 預防 AMS 的機轉是加速高海拔適應，不是掩蓋症狀。

高山症三大症候群
AMS（急性高山症）HACE（高海拔腦水腫）HAPE（高海拔肺水腫）
定義 上升至 >2500 m 後出現頭痛 + 以下 ≥1 項：疲勞、頭暈、失眠、噁心嘔吐；Lake Louise AMS 評分 ≥3 AMS 進展 → 腦水腫；共濟失調（步態不穩）或意識改變（混亂 → 昏迷） 通常發生在 >2500 m；上升後 2-4 天；休息時呼吸困難 + 乾咳 → 粉紅泡沫痰；可無 AMS 症狀（50%）
機轉 低氧 → 腦血管擴張 → 輕度腦水腫 嚴重腦水腫（血腦屏障破壞 + 腦血管自調失效） 低氧性肺血管收縮（不均勻）→ 局部高壓 → 非心因性肺水腫（Exudative）；↑ 肺動脈壓是核心機轉
危險因素 快速上升；之前有 AMS 史；個體敏感性（與體能無關） AMS 未處理進展 之前有 HAPE 史（最強危險因子）；快速上升；寒冷；劇烈運動；PFO（卵圓孔未閉）
身體檢查 通常正常 步態共濟失調；意識改變；視乳頭水腫（晚期） 心搏過速；呼吸急促；肺部濕囉音；發紺；SpO₂ 顯著↓（相對同海拔正常人）

預防

  • 緩慢上升：2500 m 以上每天上升 <500 m（睡眠高度）；每上升 1000 m 休息 1 天
  • Acetazolamide（預防用）：125-250 mg bid；上升前 1-2 天開始；磺胺類過敏者禁忌；副作用：多尿、感覺異常（手足）、碳酸飲料味道改變
  • Dexamethasone：2-4 mg q6-12h；預防 AMS（不建議常規，有 rebound 效應）；緊急撤退或無法使用 Acetazolamide 時
  • HAPE 高危患者（有過去病史）：Nifedipine 30 mg qd 或 Tadalafil 10 mg bid 預防性使用（上升前 1 天開始）
  • 體能和年齡無法預測個體對高山症的敏感性；之前的高山症病史是最佳預測因子
