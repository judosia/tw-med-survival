外傷 Trauma

ATLS 框架：先救命再救肢體，先初級評估再次級評估。不要在第一個問題上卡住。

ATLS 初級評估 Primary Survey
外傷的優先順序：出血控制 > 氣道 > 其他。大量出血的病人撐不到你評估完氣道。
緊急處置步驟
ABCDE 初級評估（外傷版）
  • A
    Airway + 頸椎保護：外傷直到排除前都要固定頸椎（頸圈 + 脊椎板）；清除口腔異物；插管前假設有頸椎損傷
  • B
    Breathing：聽診兩側、看胸廓起伏、排除張力性氣胸、血胸、開放性氣胸；SpO₂ 監測
  • C
    Circulation + 止血：大量出血直接加壓止血；建立兩條大口徑 IV（14-16G）；輸血優先於晶體液（1:1:1 策略）
  • D
    Disability：GCS、瞳孔、肢體活動；血糖
  • E
    Exposure：脫光衣物全身檢查（前後都要）；注意保溫（低體溫會加重凝血障礙）
緊急
張力性氣胸 — 不等 CXR
  • 臨床診斷：氣管偏移（偏向健側）+ 頸靜脈怒張 + 單側無呼吸音 + 低血壓
  • 立刻針刺減壓：第 2 肋間鎖骨中線，或第 4-5 肋間腋前線（較安全）
  • 針刺後立刻置放胸管
  • 不要等 CXR 確認！臨床診斷就立刻處理
緊急
開放性氣胸 — 三邊封閉
  • 胸壁傷口 >2/3 氣管直徑 → 空氣優先走傷口不走氣管
  • 立刻：三邊封閉敷料（Asherman seal 或自製）
  • 三邊封住，留一邊讓氣排出（避免張力性氣胸）
  • 之後置放胸管、手術評估
ATLS 次級評估 Secondary Survey
判斷框架
AMPLE 病史
  • A — Allergies（過敏史）
  • M — Medications（目前用藥，特別是抗凝）
  • P — Past medical history（病史）
  • L — Last meal（最後進食時間，影響手術安排）
  • E — Events leading to injury（受傷經過）
  • 受傷機制很重要：車速、安全帶、方向盤撞擊 → 預測可能傷害
處置步驟
頭頸胸腹骨盆四肢完整評估
  • 頭：頭皮撕裂、顱骨凹陷、眼耳鼻出血
  • 頸：頸椎壓痛（固定中不要移除頸圈）
  • 胸：肋骨骨折（連枷胸）、皮下氣腫
  • 腹：壓痛、腹膜刺激徵（FAST 超音波）
  • 骨盆：骨盆環穩定性（一次測，不要重複）
  • 四肢：骨折、神經血管完整性
  • 背部：翻身檢查（Log-roll）
數值判讀
外傷 FAST 超音波
  • F — Focused Assessment with Sonography in Trauma
  • 四個窗口：
  • 心包（心包填塞）
  • Morrison's pouch（右上腹，肝腎之間）
  • Splenorenal space（左上腹，脾腎之間）
  • 骨盆（Douglas pouch，游離液體）
  • eFAST 加：兩側胸膜（氣胸）
  • FAST 陽性（游離液體）+ 血流不穩 → 立刻手術，不等 CT
注意！
外傷致死三角
  • 低體溫（Hypothermia）：<35°C 凝血因子功能下降
  • 酸中毒（Acidosis）：pH <7.2 凝血酶活性下降
  • 凝血障礙（Coagulopathy）：三者互相惡化形成惡性循環
  • 處置：Damage Control Surgery（先止血，不做完整手術）
  • 保溫（毯子、加溫輸液）、早期輸血（1:1:1）、補 Ca²⁺
  • 大量輸液（晶體液）會稀釋凝血因子，加重凝血障礙
出血控制 Hemorrhage Control
緊急處置
外部出血控制
  • 直接加壓：持續加壓至少 5-10 分鐘，不要中途查看
  • 止血帶（Tourniquet）：四肢大量出血首選，記錄使用時間
  • 止血帶位置：傷口近端 5-8 cm，緊到脈搏消失
  • 填塞（Packing）：無法用止血帶的部位（鼠蹊部、腋下）
  • 止血粉/紗布：Combat Gauze（含 Kaolin 或 TXA）
  • 止血帶使用 >2h 有神經血管損傷風險，記錄時間很重要
用藥指引
大量輸血策略
  • 1:1:1 策略：pRBC : FFP : Platelet = 1:1:1（模擬全血）
  • Tranexamic acid（TXA）：1g IV over 10min，傷後 3h 內給（降低死亡率）
  • 目標：收縮壓 80-90 mmHg（Permissive hypotension，在止血前）
  • Calcium：大量輸血後補 Ca²⁺（citrate 結合 Ca²⁺）
  • 避免大量晶體液（NS/LR），會稀釋凝血因子、加重酸中毒
頭部外傷 Head Trauma
數值判讀
頭部外傷嚴重度分類
  • 輕度 TBI：GCS 13-15，可能有短暫意識喪失 <30 分鐘
  • 中度 TBI：GCS 9-12，意識喪失 30 分鐘至 6 小時
  • 重度 TBI：GCS ≤8，意識喪失 >6 小時
  • CT 指征（輕度）：GCS <15、嘔吐 >2 次、年齡 >65、抗凝藥物、頭皮血腫、骨折
注意！
顱內壓升高的處置
  • 床頭抬高 30°
  • 維持 MAP ≥70（保持腦灌流壓 CPP ≥60）
  • Mannitol 0.25-1g/kg IV（滲透壓治療）
  • 3% NaCl：20-40mL bolus（替代 Mannitol）
  • 避免低血壓（收縮壓 <90 會加重腦損傷）
  • 避免低氧（SpO₂ 維持 >95%）
  • 重度 TBI 禁止過度換氣（PaCO₂ 維持 35-45），除非腦疝緊急時短暫用
機轉說明
顱內出血分類
  • 硬膜外血腫（EDH）：顳骨骨折傷中腦膜動脈，CT 雙凸透鏡形；Lucid interval（清醒後再昏迷）
  • 硬膜下血腫（SDH）：橋靜脈撕裂，CT 月牙形；老人跌倒常見
  • 蜘蛛膜下腔出血（SAH）：外傷或動脈瘤破裂，CT 腦溝高密度
  • 腦挫傷：直接撞擊或對衝傷，CT 出血點
  • EDH 有 Lucid interval 是典型表現，但不是所有 EDH 都有
臉部 + 頸部外傷 Face & Neck Trauma
臉部 / 頸部外傷最重要的是氣道優先——顏面骨折、頸部穿刺傷或血腫擴大均可迅速導致氣道阻塞。頸部穿刺傷穿過闊頸肌（Platysma）= 深頸部損傷，需手術探查或 CT 血管攝影。顏面骨折分類：LeFort 骨折（I/II/III）、顴弓骨折、眼眶底部骨折（爆裂骨折）、下顎骨折、鼻骨骨折。眼眶底部爆裂骨折 → 眼外肌嵌頓 → 垂直複視 + 眼球運動受限 = 緊急手術。
機轉說明
顏面骨折分類
骨折類型特徵臨床表現處置
鼻骨骨折 最常見顏面骨折 鼻梁壓痛、腫脹、瘀血；鼻中隔血腫（需立刻引流，避免軟骨壞死） 急診：止血 + 鼻中隔血腫引流；7-10 天後耳鼻喉科門診復位
眼眶底部爆裂骨折（Blowout） 眼眶底部薄弱處骨折 → 眼外肌 / 眶脂肪嵌頓 垂直複視；眼球內陷；眼眶周圍感覺麻木（眶下神經）；眼球運動受限（尤其向上凝視） 緊急眼科會診；眼外肌嵌頓 → 緊急手術；否則 5-7 天後擇期手術
顴弓骨折 顴骨 + 顴弓複合體 臉頰扁平；眼眶周圍瘀血（Raccoon eyes）；下顎開合受限（咬肌嵌頓）；眶下神經麻木 擇期整形外科手術（1-2 週內）
LeFort 骨折（上顎 / 顱底骨折） I 型：上顎水平骨折（牙槽骨）
II 型：錐形骨折（鼻根 + 眼眶底）
III 型：顱面分離（顱底 + 顴弓 + 眼眶） 		臉部腫脹；上顎活動；顱底骨折徵象（CSF 鼻漏、Raccoon eyes、Battle 徵）；常合併頸椎損傷 氣道優先；CT 確認；神經外科 + 整形外科會診
下顎骨折 常為雙側（下顎骨呈環形） 咬合不正；疼痛；牙關緊閉；舌側面腫脹（↑ 氣道阻塞風險） 軟食；口腔外科會診；開放性 → 抗生素；氣道注意

顱底骨折徵象

  • Raccoon eyes（眼眶周圍瘀血）：前顱窩骨折
  • Battle 徵（乳突後瘀血）：顳骨岩部骨折
  • CSF 鼻漏 / 耳漏：β₂-transferrin 試驗（最特異）；暈圈徵（血液在衛生紙上形成環狀）
  • 鼓膜後積血（Hemotympanum）：顳骨骨折
緊急處置
頸部穿刺傷 + 鈍傷
穿過 Platysma 的頸部穿刺傷 = 深頸部損傷，需 CT 血管攝影或手術探查。不要移除嵌入異物。

頸部解剖區域

區域範圍重要結構
Zone I鎖骨到環狀軟骨大血管（鎖骨下、椎動脈）；食道；氣管；肺尖
Zone II環狀軟骨到下顎角頸動脈；頸靜脈；氣管；食道；最常受傷
Zone III下顎角到顱底頸動脈遠端；椎動脈；咽部

穿刺傷評估和處置

  • 硬性適應症（立刻手術）：活動性出血無法控制；血流動力學不穩定；擴大血腫；氣道受損（喘鳴、皮下氣腫）；神經缺損；食道損傷（唾液 / 血液從傷口溢出）
  • 無症狀 / 穩定 → CT 血管攝影（CTA）：評估血管、氣管、食道損傷；Zone I 和 III 尤其依賴影像（手術困難）
  • Zone II 有症狀 → 直接手術探查
  • 暫時止血：直接加壓；不要盲目夾閉血管

頸部鈍傷（Blunt Cerebrovascular Injury，BCVI）

  • 高能量機轉（MVC、跳水、勒頸）→ 頸動脈 / 椎動脈損傷 → 缺血性中風（延遲 12-72 小時）
  • Biffl 分級：I（內膜撕裂）→ V（血管橫斷）
  • 篩查適應症（CTA 頸部）：LeFont II/III；顱底骨折；頸椎骨折（尤其 C1-C3）；Horner 症候群；無解釋的局灶神經缺損
  • 治療：Grade I-II → 抗凝（Heparin）或抗血小板（ASA）；Grade III-V → 手術或血管內介入
胸部外傷 Chest Trauma
緊急
血胸 Hemothorax
  • 大量血胸：>1500mL 或持續出血 >200mL/hr
  • 臨床：呼吸音消失、叩診濁音、低血壓
  • 處置：大口徑胸管（32-36 Fr）引流
  • 大量血胸 → 手術止血（Video-assisted or 開胸）
  • 胸管出來鮮紅血 >1500mL 或持續流 → 立刻通知外科
注意！
連枷胸 Flail Chest
  • 定義：≥3 根連續肋骨各有 ≥2 處骨折 → 矛盾呼吸
  • 吸氣時胸壁內陷，呼氣時外突
  • 伴隨肺挫傷（真正危險所在）
  • 處置：充足給氧、止痛（肋間神經阻斷）
  • 嚴重低氧 → 插管正壓通氣（矯正矛盾呼吸）
注意！
心臟鈍傷 Blunt Cardiac Injury
  • 方向盤撞擊、高速事故後
  • ECG 變化：竇性心跳過速、束枝傳導阻滯、新發心律不整
  • Troponin 升高
  • ECG 正常且 Troponin 正常 → 低風險，不需心監
  • ECG 異常 → 持續心電監測 24-48h
注意！
主動脈損傷 Aortic Injury
  • 高速減速傷（車禍、高處墜落）
  • CXR：縱膈腔增寬 >8cm、主動脈弓模糊、氣管右偏
  • 確診：CT angiography
  • 處置：血壓控制（目標收縮壓 <120）+ 手術或血管內修復
  • 縱膈腔增寬是 CXR 最重要的警示，必須 CT 進一步評估
腹部外傷 Abdominal Trauma
判斷框架
鈍傷 vs 穿刺傷處置方向不同
  • 鈍傷（Blunt）：FAST 超音波 → CT → 選擇性手術（穩定患者）
  • 脾臟（最常受傷）、肝臟、腸道
  • 穿刺傷（Penetrating）：腹腔進入 → 通常需要剖腹探查
  • 槍傷：幾乎都需要手術
  • 刀傷：穩定患者可嘗試保守治療，但腹腔有傷就要手術
緊急
FAST 陽性 + 血流不穩 → 手術
  • FAST 陽性（游離液體）+ 血流動力學不穩定 → 直接手術，不等 CT
  • FAST 陽性 + 血流穩定 → CT 評估損傷程度
  • FAST 陰性不能排除損傷（敏感性約 80%）
  • 腸道損傷：FAST 可能陰性，CT 是標準
  • 時間就是生命，血流不穩定的腹部外傷不要浪費時間做 CT
脊椎外傷 Spinal Trauma
注意！
脊椎固定原則
  • 所有重大外傷 → 假設有頸椎損傷，固定直到排除
  • 頸圈 + 脊椎板（Spinal board）
  • 影像確認：頸椎 CT（比 X-ray 敏感）
  • 清醒且配合的低風險患者可用 NEXUS 或 Canadian C-spine Rule 臨床排除
  • 搬移病人時要 Log-roll（三人以上），保持脊椎中立位
機轉說明
神經性休克 Neurogenic Shock
  • 高位脊髓損傷（T6 以上）→ 交感神經阻斷
  • 臨床：低血壓 + 心跳過慢（和其他休克的心跳過速不同！）
  • 皮膚溫暖乾燥（血管擴張但無汗腺）
  • 處置：輸液 + Norepinephrine（恢復血管張力）
  • 外傷病人低血壓 + 心跳慢 = 先想神經性休克，排除出血後
四肢 + 骨盆外傷 Extremity & Pelvic Trauma
骨盆環破裂可造成大量出血（2-4 L）致死——不穩定骨盆 → 立刻骨盆帶固定（床單或商業骨盆帶）↓ 骨盆容積 → 填塞效應止血。骨筋膜室症候群（ACS）= 外科急症——6P（Pain、Pressure、Paresthesia、Pallor、Paralysis、Pulselessness）是診斷線索，但脈搏消失是晚期表現；筋膜室壓力 >30 mmHg 或 ΔP（舒張壓 − 筋膜室壓力）<30 mmHg → 立刻筋膜切開術。開放性骨折（Gustilo 分類）= 急症 → 抗生素 + 破傷風 + 急診沖洗 + 手術固定。
緊急處置
骨盆環骨折

骨盆骨折分類（Young-Burgess）

類型機轉出血風險合併傷
側面壓縮（LC）側面撞擊（最常見）中等膀胱、尿道、腸道
前後壓縮（APC，Open book）正面撞擊 / 骨盆張開高（靜脈叢撕裂）尿道撕裂、陰道、神經
垂直剪力（VS）高能量（跌落）最高（大血管損傷）腰薦叢神經損傷

出血控制策略

  • 立刻骨盆帶固定（不穩定骨盆）：商業骨盆帶或床單綁在大轉子水平（不是腰部！）→↓ 骨盆容積 → 填塞效應；LC 型骨折骨盆帶可能加重損傷（謹慎）
  • 外固定架（External fixation）：骨盆帶無效 → 骨科手術固定
  • REBOA（Resuscitative Endovascular Balloon Occlusion of Aorta）：Zone III 腹主動脈 → 暫時阻斷出血；橋接至手術 / 介入放射
  • 骨盆填塞（Pelvic Packing）：剖腹後腹膜外填塞；部分創傷中心優先於血管栓塞
  • 血管栓塞（Angioembolization）：動脈出血（CT 血管外滲）；血流動力學穩定 → 考慮；不穩定 → 先手術控制
  • 骨盆骨折出血 80% 來自靜脈叢（非動脈）→ 骨盆帶固定效果良好；動脈出血（高能量 / VS 型）→ 需栓塞或手術

泌尿道損傷評估

  • 骨盆骨折 + 血尿 / 尿道口出血 / 會陰部血腫 → 尿道損傷疑慮 → 先逆行性尿路攝影（RUG）再插導尿管
  • 膀胱損傷：腹腔內（腹膜破裂 → 手術）vs 腹腔外（保守治療 → 持續導尿）
緊急處置
骨筋膜室症候群（ACS）
筋膜室壓力 >30 mmHg 或 ΔP <30 mmHg → 立刻筋膜切開術。不要等脈搏消失。
  • 好發部位：小腿前側筋膜室（最常見）；前臂；大腿；手；足；腹部（腹腔高壓）
  • 病因：骨折（尤其脛骨）；擠壓傷；燒傷；縫合過緊的石膏；血腫；再灌流（缺血後）；橫紋肌溶解

6P 臨床表現（依嚴重度出現）

  • Pain（疼痛）：與外傷不相稱的嚴重疼痛；被動牽拉受累肌肉時疼痛加重（最早、最敏感）
  • Pressure（緊繃）：筋膜室觸診堅硬
  • Paresthesia（感覺異常）：麻木、刺痛（神經缺血早期）
  • Pallor（蒼白）：皮膚顏色改變
  • Paralysis（癱瘓）：肌肉無力（晚期，已有神經損傷）
  • Pulselessness（脈搏消失）：最晚出現，已不可逆損傷

診斷與治療

  • 筋膜室壓力測量：針刺 + 壓力計；正常 <10 mmHg；>30 mmHg → 手術；ΔP（DBP − 筋膜室壓）<30 mmHg → 手術
  • 治療：立刻移除所有環狀包紮（石膏、繃帶）；患肢維持心臟水平（不要抬高過高 → ↓ 灌流壓）；筋膜切開術（全筋膜室減壓）
  • 小腿筋膜切開術：4 個筋膜室均需減壓（前、外側、後深、後淺）
  • 筋膜切開術後：傷口敞開 + 負壓引流 → 3-5 天後延期關閉或植皮
用藥指引
開放性骨折（Gustilo 分類）+ 肢體血管損傷

Gustilo 開放性骨折分類

級別傷口大小特徵抗生素
I<1 cm乾淨；低能量；骨折斷端穿皮第一代 Cephalosporin（Cefazolin 1-2 g IV q8h）× 24 小時
II1-10 cm乾淨；中等污染同 Grade I
IIIA>10 cm廣泛軟組織損傷；可覆蓋骨骼Cefazolin + Gentamicin × 24 小時
IIIB>10 cm廣泛軟組織損傷；骨骼外露；需皮瓣覆蓋同 IIIA（農場傷口 + Penicillin G / Metronidazole）
IIIC>10 cm血管損傷需修復同 IIIA
  • 急診處置：傷口照片（評估污染程度）→ 生理食鹽水沖洗 → 無菌敷料覆蓋 → 骨折固定（夾板）→ 立刻手術
  • 抗生素應在受傷後 1-3 小時內給予（越早越好）
  • 破傷風預防；不要急診縫合（感染風險高）

肢體血管損傷（Hard vs Soft Signs）

  • Hard Signs（立刻手術）：活動性出血；擴大血腫；搏動消失 + 缺血肢體；搏動性血腫；血管雜音 / 震顫（AV 瘻管）
  • Soft Signs（CTA 評估）：鄰近骨折 / 脫臼；神經缺損；小型穩定血腫；ABI <0.9
  • 肢體缺血黃金時間：6 小時——血管修復需在 6 小時內完成以避免不可逆缺血
  • ABI（踝臂指數）：患肢踝部收縮壓 / 上肢收縮壓；<0.9 = 血管損傷；需 CTA 確認
  • 膝關節脫臼 → 高風險膕動脈損傷（15-40%）→ CTA 膝部
特殊族群外傷 Trauma in Special Populations
特殊族群的外傷有其生理特性需要注意：老年人（生理儲備↓、代償↑困難、多重用藥）；孕婦（胎兒評估、子宮壓迫下腔靜脈、Kleihauer-Betke 試驗）；以上均需更低的影像閾值和更積極的早期處置。
機轉說明
老年外傷（依 Tintinalli Ch.255）

生理特性

  • 心臟：心臟儲備↓；對出血的代償性心搏過速受損（β-blocker 或老化竇房結）→ 心率「正常」不代表無出血
  • 腎臟：GFR↓（Cr 可能假性正常）→ 對造影劑和 NSAIDs 更敏感
  • 骨骼：骨質疏鬆 → 低能量即可骨折（脊椎壓迫、股骨頸、肋骨）
  • 腦部：腦萎縮 → 硬膜下血腫拉伸橋接靜脈 → 更容易 SDH；Anticoagulant 使用 → ↑ 顱內出血風險
  • 多重用藥：β-blocker（遮蓋心搏過速）；抗凝血藥（出血風險）；利尿劑（電解質異常）

評估重點

  • 血壓「正常」= 相對低血壓：老年患者基礎收縮壓常 140-160 mmHg；「正常」的 120 mmHg 可能已是休克
  • 更低的 CT 閾值：任何頭部外傷（即使輕微）+ 抗凝血藥 → CT 頭部；跌倒 → 評估是否有內科誘因（心律不整、暈厥、低血糖）
  • 肋骨骨折 → 老年患者死亡率和肺炎風險遠高於年輕人；≥3 根 → 考慮 ICU
  • C-spine 骨折 → 老年患者脊椎退化 → 即使輕微外傷也可能造成不穩定骨折；CT 頸椎比 X 光更敏感

處置要點

  • 早期抗凝血藥逆轉（Warfarin → Vit K + PCC；DOAC → 特異性解毒劑）
  • 積極補液謹慎（心臟儲備↓ → ↑ 肺水腫風險）；目標 MAP 65-70
  • 早期疼痛控制（肋骨骨折 → 硬膜外 / 神經阻斷 → ↓ 肺炎）
  • 跌倒 → 完整跌倒評估；排除內科誘因；社會評估（獨居、失能）
緊急處置
孕婦外傷

孕婦生理變化（影響外傷評估）

  • 血容量↑ 40-50%：失血 30-35% 才出現心搏過速（正常孕婦 HR 已較快）
  • 子宮壓迫下腔靜脈（20 週後）：仰臥 → 靜脈回流↓ → 低血壓；左側臥 15° 或人工徒手推移子宮向左側
  • 橫膈上移 4 cm → 氣胸 / 血胸改變；胸管放置位置需上移一個肋間
  • 胃排空↓ → 誤吸風險↑
  • 凝血活化 → 高凝狀態

胎兒評估

  • 胎心監測（FHR）：≥20 週 → 連續電子胎心監測至少 4 小時（輕度外傷）；重度外傷 → 24 小時
  • 胎盤早剝（Placental Abruption）：即使輕微外傷也可發生；症狀：陰道出血、腹痛、子宮收縮、胎心異常；診斷：超音波（敏感度低 50%）；胎心監測異常；Fibrinogen↓（DIC）
  • Kleihauer-Betke 試驗：母體血液中胎兒紅血球 → Rh 陰性母親 → 需給 Anti-D 免疫球蛋白（Rho-GAM）；劑量依胎兒-母體輸血量決定
  • 子宮破裂（罕見，高能量）：陰道大量出血；胎兒部位可觸及腹壁外；胎心消失 → 緊急剖腹

影像和用藥

  • 放射線：必要影像不要因懷孕而省略（母體安全優先）；胎兒安全劑量 <50 mrad；CT 腹部 / 骨盆 ≈ 3.5 rad（需告知）；可加鉛盾保護
  • 緊急剖腹產（Perimortem C-section）：心跳停止 + ≥20 週 → 4 分鐘內無 ROSC → 立刻剖腹產（↓ 主動脈壓迫 → 可能改善 ROSC）
  • TXA（Tranexamic acid）可在孕婦出血使用；RS 首選 LR（避免大量 NS → 高氯性酸中毒）
  • 母體安全永遠優先——穩定母體 = 最好的胎兒復甦
