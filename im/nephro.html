腎臟科 Nephrology

電解質和酸鹼是整個內科的基礎。搞懂腎臟，其他系統的異常才看得清楚。

酸鹼異常 Acid-Base Disturbances
酸鹼分析的步驟：① 看 pH 決定酸鹼症；② 看主要異常（HCO₃ 或 PaCO₂）；③ 計算代償是否適當；④ 代謝性酸中毒要算 AG；⑤ AG 升高要算 Δ/Δ 找出混合型。代償永遠不會讓 pH 完全正常——正常 pH 代表混合型。

AG 代謝性酸中毒（GOLD MARK）
  • AG = Na − (Cl + HCO₃)；正常 = [Albumin] × 2.5（白蛋白 4 g/dL 時正常 AG ≈ 10）
  • AG 升高 → 未測定陰離子增加（有機酸、磷酸鹽、硫酸鹽）

GOLD MARK 記憶法

  • Glycols（乙二醇、丙二醇）：乙二醇 → 草酸鈣結晶 + AKI；丙二醇（Lorazepam IV）→ 乳酸酸中毒
  • Oxoproline（5-oxoproline/焦麩胺酸）：Acetaminophen + Penicillin 在高風險患者（女性、營養不良、懷孕、腎/肝衰竭）
  • Lactic acidosis：Type A（低氧，如休克、腸缺血、CO 中毒）；Type B（非低氧，如惡性腫瘤、肝功能障礙、Metformin、NRTI、Propofol、INH、Linezolid）
  • D-Lactic acidosis：短腸症候群，標準乳酸試劑無法偵測
  • Methanol（甲醇）：代謝為甲酸 → 視力改變、視乳頭水腫
  • ASA（阿斯匹靈）：早期呼吸性鹼中毒 + 晚期代謝性酸中毒；發燒、耳鳴
  • Renal failure：有機陰離子蓄積
  • Ketoacidosis：DKA、酒精性（AKA）、饑餓、SGLT2i

Δ/Δ 分析（AG 升高時）

  • Δ/Δ = ΔAG / ΔHCO₃（ΔAG = 計算 AG − 預期 AG；ΔHCO₃ = 24 − HCO₃）
  • Δ/Δ = 1-2：純 AG 代謝性酸中毒
  • Δ/Δ <1：AG 代謝性酸中毒 + 非 AG 酸中毒同時存在
  • Δ/Δ >2：AG 代謝性酸中毒 + 代謝性鹼中毒同時存在
  • 懷疑毒物攝取 → 算滲透壓間隙（OG）= 測量滲透壓 − 計算滲透壓（2×Na + glucose/18 + BUN/2.8）；OG >10 → 懷疑攝入（但非特異性）；中毒初期 OG ↑、後期 AG ↑

非 AG 代謝性酸中毒與 RTA

原因

  • GI HCO₃ 流失：腹瀉、腸瘻、胰瘻
  • RTA（腎小管酸中毒）
  • 早期腎衰竭：氨生成障礙
  • 藥物：Acetazolamide、Cholestyramine、Toluene
  • 稀釋性：快速輸注不含 HCO₃ 的液體

尿液 AG（UAG）= UNa + UK − UCl

  • UAG 陰性（↑ NH₄⁺ 排泄）：腎臟對酸血症有適當反應 → 原因在 GI（腹瀉）、NS 輸注或攝入物
  • UAG 陽性（↓ NH₄⁺ 生成）：腎臟無法產生足夠 NH₄⁺ → 遠端 RTA、高醛固酮低下型 RTA、早期腎衰竭

機轉說明
代謝性鹼中毒

分類（用 UCl 區分）

  • 生理食鹽水有效（UCl <25）：GI H⁺ 流失（嘔吐、鼻胃管引流、絨毛腺瘤）；利尿劑（Loop / Thiazide）；術後高碳酸血症後
  • 生理食鹽水無效（UCl >40）：高血壓（礦物皮質素過多）：原發性醛固酮增多症、Cushing's、Liddle's；正常血壓：嚴重低鉀（K <2）、外源性鹼補充（合併 AKI）、Bartter's / Gitelman's 症候群
  • UCl 比 UNa 更可靠（鹼血症時 NaHCO₃ 排泄增加會拉高 UNa，失去容積指標意義）

治療（pH >7.6）

  • 生理食鹽水有效：NS 補液 + 補鉀 + 停利尿劑；心肺功能不允許補液 → KCl 或 Acetazolamide 或 HCl
  • 醛固酮過多：治療原因；保鉀利尿劑；若原發性 → 腎上腺腺瘤切除
機轉說明
呼吸性酸中毒 / 鹼中毒原因

呼吸性酸中毒（CO₂ 蓄積）

  • CO₂ 產生 ↑：發燒、甲狀腺毒症、敗血症、類固醇、過度餵食（碳水化合物）
  • CNS 抑制：鴉片類、苯二氮平類、CNS 創傷、中樞型睡眠呼吸中止症
  • 神經肌肉疾病：GBS、脊髓灰質炎、ALS、重症肌無力、嚴重低磷和低鉀血症
  • 胸壁：氣胸、血胸、連枷胸、脊椎側彎
  • 上呼吸道：異物、喉痙攣、OSA
  • 下呼吸道：氣喘、COPD、肺水腫、肺纖維化

呼吸性鹼中毒（過度換氣）

  • 低氧血症：肺炎、CHF、PE、限制性肺疾病、貧血
  • CNS 刺激：疼痛、焦慮、創傷、中風、CNS 感染
  • 藥物：阿斯匹靈中毒（早期）、β 促效劑、黃嘌呤類、尼古丁
  • 其他：懷孕、敗血症、肝衰竭、甲狀腺機能亢進、發燒
  • 假性呼吸性鹼中毒：低灌流合併正常換氣（如心肺復甦、嚴重低血壓）→ PaCO₂ 低但組織 CO₂ 高
用藥指引
嚴重代謝性酸中毒治療
  • DKA：胰島素 + IVF + 補鉀（NEJM 2015）
  • 酒精性酮酸中毒（AKA）：葡萄糖 + IVF + 補充 K、Mg、PO₄
  • 乳酸酸中毒：治療原因；避免血管收縮劑；停「Type B」藥物
  • 腎衰竭：血液透析
  • 甲醇 / 乙二醇：Fomepizole（>20 mg/dL）；甲醇加 Folate；乙二醇加 Vit B1 + B6；視力改變或 AKI 或 >50 mg/dL → 透析
  • 鹼療法（NaHCO₃）：pH <7.1，或 pH <7.2 合併 AKI（可能 ↓ 死亡率）；估算需求量：（目標 HCO₃ − 現在 HCO₃）× 體重(kg) × 0.4；副作用：容積過多、高鈉、低離子鈣、PaCO₂ ↑（需確保足夠換氣）
急性腎損傷 AKI
數值判讀
KDIGO 定義與分期
  • 定義：48h 內 Cr 上升 ≥0.3 mg/dL，或 7天內上升 ≥1.5 倍，或
    尿量 <0.5mL/kg/hr >6h
  • Stage 1：Cr ×1.5-1.9 或 ↑≥0.3 mg/dL；尿量 <0.5mL/kg/hr >6h
  • Stage 2：Cr ×2-2.9；尿量 <0.5mL/kg/hr >12h
  • Stage 3：Cr ×3 或 ≥4.0 或開始透析；尿量 <0.3mL/kg/hr >24h 或無尿 >12h
  • 尿量標準同樣重要，不要只看 Cr，尿量減少更早反映 AKI
機轉說明
腎前性 / 腎性 / 腎後性
  • 腎前性（最常見）：低血容、低血壓、心衰竭 → BUN/Cr >20、FENa <1%
  • 腎性：ATN（缺血或腎毒性）、腎絲球腎炎、間質性腎炎 → FENa >2%
  • 腎後性：尿路阻塞（攝護腺、結石、腫瘤）→ 超音波看腎盂擴張
  • 腎後性是最容易可逆的，一定要超音波排除阻塞
處置步驟
AKI 的處置原則
  • 1
    超音波排除腎後性阻塞（最快且最可逆）
  • 2
    評估容積狀態：低血容給輸液；過多給利尿劑
  • 3
    停止腎毒性藥物：NSAIDs、Aminoglycosides、造影劑（高風險前預防性輸液）
  • 4
    調整腎排除藥物劑量（依 eGFR）
  • 5
    監測並處理：K⁺（高血鉀）、酸鹼（代謝性酸中毒）、體液平衡
  • 6
    若符合透析指征 → 腎臟科會診
造影劑腎病預防：術前術後各 1L NS，停 Metformin 術前 24-48h
數值判讀
AKI 時尿液分析的意義
  • 棕色顆粒圓柱（Muddy brown cast）：ATN
  • 紅血球圓柱（RBC cast）：腎絲球腎炎（緊急腎臟科會診）
  • 白血球圓柱（WBC cast）：間質性腎炎、腎盂腎炎
  • 玻璃樣圓柱（Hyaline cast）：腎前性、正常人也可見
  • 嗜酸性球尿（Eosinophiluria）：過敏性間質性腎炎（藥物引起）
慢性腎臟病 CKD
數值判讀
CKD 分期（依 eGFR）
  • G1：eGFR ≥90（腎損傷但功能正常）
  • G2：eGFR 60-89（輕度下降）
  • G3a：eGFR 45-59（輕中度下降）
  • G3b：eGFR 30-44（中重度下降）
  • G4：eGFR 15-29（重度下降，開始透析準備）
  • G5：eGFR <15（腎衰竭，透析或移植）
  • 診斷需持續 >3 個月，單次異常不等於 CKD
注意！
CKD 常見併發症監測
  • 高血壓：目標 <130/80，首選 ACEI/ARB（減少蛋白尿）
  • 貧血：EPO 缺乏，Hb <10 考慮 ESA 或鐵劑
  • 腎性骨病：高磷 + 低鈣 + PTH 升高 → 磷結合劑、活性維生素 D
  • 代謝性酸中毒：HCO₃⁻ <22 → 補 NaHCO₃
  • 高血鉀：eGFR <30 風險高，限鉀飲食、停 ACEI/ARB 若嚴重
  • eGFR <30 轉介腎臟科準備透析，不要等到 eGFR <10 才轉
用藥指引
CKD 時需要調整或停用的藥物
  • 停用：NSAIDs（腎毒性）、Metformin（eGFR <30）
  • 減量：Vancomycin、Piperacillin/Tazo、Meropenem、Gabapentin
  • 避免：含鉀補充劑、保鉀利尿劑（eGFR <30）
  • ACEI/ARB：保護腎功能，但 eGFR 急速下降 >30% 或 K⁺ >5.5 時暫停
  • 造影劑：高風險患者（eGFR <30）術前水化
  • 開藥前先查 eGFR，這是住院最常見的用藥錯誤來源之一
腎衰竭 Kidney Disease
AKI 的評估核心是：腎前性 vs 腎性 vs 腎後性，先排除可逆原因再處置。FENa 是主要鑑別工具，但有陷阱（使用利尿劑時改用 FEUN）。CKD 的管理重點是減緩進展（血壓、蛋白尿、SGLT2i）和預防透析前相關併發症。透析的緊急適應症記住縮寫：AEIOU。

AKI 分類與鑑別診斷

AKI 定義（KDIGO）

  • Stage 1：Cr 在 2 天內 ↑ ≥0.3 mg/dL，或 ↑ ≥50%；或 UOP <0.5 mL/kg/h ≥6h
  • Stage 2：Cr ↑ 2-3× 基準值；或 UOP <0.5 mL/kg/h ≥12h
  • Stage 3：Cr ↑ ≥3×；或 UOP <0.3 mL/kg/h ≥24h；或無尿 ≥12h；或 Cr >4 mg/dL
類別代表疾病尿液分析 / 指標
腎前性 低血容（出血、腹瀉）、↓ CO（CHF）、↓ 腎灌流（NSAIDs、ACEI/ARB、SGLT2i、Cyclosporine、Contrast）、大血管（RAS、夾層） 尿液清澈；透明顆粒管型；FENa <1%；BUN/Cr >20；FEUN ≤35%
腎性—ATN 嚴重缺血；敗血症；腎毒性藥物（Vancomycin、Aminoglycosides、Cisplatin、Foscarnet、IVIG）；色素（Hb、Myoglobin）；光鏈（骨髓瘤）；結晶（尿酸、Acyclovir、MTX、Indinavir） 色素性顆粒管型（muddy brown casts）；± RBC 和蛋白（從管道損傷）；FENa >2%；BUN/Cr <20（色素性 AKI 和造影劑例外）；FEUN >50%
腎性—AIN 藥物過敏（β-lactam、磺胺類、NSAIDs、PPI、FQ、別嘌醇）；感染；浸潤（Sarcoid、淋巴瘤）；自體免疫（Sjögren's、TINU、IgG4、SLE、ICI） WBC 管型、WBC（PMN 或嗜酸性球）；± RBC；尿液培養陰性；NSAIDs 引起可見淋巴球
腎性—GN 見腎絲球疾病章節 異形 RBC、RBC 管型
腎性—小血管 膽固醇栓塞（PAN、ANCA 血管炎）；血栓性微血管病變（TTP、HUS、DIC、先兆子癲） ± RBC；膽固醇栓塞：周邊嗜酸性球、↓ C3/C4
腎後性 膀胱頸阻塞（BPH、前列腺癌、神經性膀胱、抗膽鹼藥）；輸尿管阻塞（惡性腫瘤、LAN、後腹腔纖維化、腎結石） 尿液清澈；± 非異形 RBC、WBC、結晶
判斷框架
AKI 評估步驟
  • 病史：藥物（NSAIDs、ACEI/ARB、顯影劑、抗生素）；低攝食；低血壓；感染 / 敗血症；外傷、肌肉痛；BPH / 尿瀦留；最重要：在 Cr 上升前 24-48h 的觸發因素
  • 身體檢查：生命徵象、容積狀態、皮疹
  • 尿液評估：輸出量、尿液分析（試紙 + 尿沉渣）、尿電解質和滲透壓
  • FENa = (UNa/PNa)/(UCr/PCr)；使用利尿劑時改用 FEUN = (UUN/PUN)/(UCr/PCr)
  • 腎臟超音波或 CT：排除阻塞（腎積水）；評估腎臟大小（慢性腎臟萎縮）
  • 血清學（有適應症時）：見腎絲球疾病章節
  • 腎臟切片：原因不明（尤其有蛋白尿 / 血尿）；相對禁忌：SBP >150、ASA / NSAID、抗凝血藥、肝硬化；GFR <45 時先給 DDAVP

特殊 AKI 的處置
  • 腎前性：等張晶體液 ≈ 白蛋白（均可）
  • AIN：立刻停止可疑藥物；若 7 天內確診可考慮早期類固醇；ICI 引起的 AIN 依腫瘤科病程考慮再挑戰
  • 心腎症候群：IV Loop 利尿劑（高劑量 vs 低劑量、持續輸注 vs 間歇注射效果相當）；不建議 Dopamine、Nesiritide、超濾
  • 造影劑 AKI（CIN）：高風險（eGFR <60，尤其合併蛋白尿、DM、MI、低血壓）→ 等張 IVF 預防；停 ACEI/ARB / NSAIDs / 利尿劑；最小造影劑量；使用等滲透壓造影劑；N-acetylcysteine 效益有爭議
  • 橫紋肌溶解：積極 IVF（目標 UOP ~3 mL/kg/h）；尿 pH <6.5 考慮 NaHCO₃；監測 K 和 Ca；監測筋膜室症候群
  • 肝腎症候群（HRS）：Terlipressin 0.85 mg IV q6h × 3 天 → 再評估 Cr（或 Norepinephrine / Vasopressin）；出現低血氧或缺血時停止；橋接至肝臟移植
  • 阻塞性：若尿道梗阻 → Foley；若輸尿管以上 → 腎造瘺；注意梗阻後利尿（補充 ½ UOP 以 ½ NS）
緊急處置
緊急透析適應症（AEIOU）
  • A — Acidosis（酸中毒）：頑固性酸血症（但不明顯優於 NaHCO₃）
  • E — Electrolytes（電解質）：頑固性高鉀血症、高鈣血症、高磷血症、腫瘤溶解症候群
  • I — Intoxications（中毒）：Methanol、乙二醇、Metformin、鋰、Valproic acid、Salicylate、Barbiturate、Theophylline、Thallium；疑問：Carbamazepine、APAP、Digoxin、Dabigatran
  • O — Overload（容積過多）：頑固性高血容導致低血氧（如 CHF）
  • U — Uremia（尿毒症）：心包炎、腦病變、出血
  • Stage 3 AKI：出現以上任何一項；或少尿 >72h；或 BUN >112 mg/dL 的積極考慮
  • AKI 每次發作均 ↑ CKD 進展風險，即使腎功能恢復後也要長期追蹤

慢性腎臟病（CKD）分期與處置

定義與分期

  • GFR <60 ≥3 個月 或 腎臟損傷（蛋白尿、結構異常）
  • 蛋白尿預測全因和 CV 死亡率及 CKD 進展（獨立危險因子）
分期GFR管理目標
G1（正常或 ↑）>90診斷和治療基礎疾病；減緩進展；CV 風險降低
G2（輕度）60-89評估進展速度
G3a（輕中度）45-59評估和治療相關併發症
G3b（中重度）30-44評估和治療相關併發症
G4（重度）15-29準備腎臟替代治療（RRT）；保護手臂靜脈
G5（腎衰竭）<15 或透析透析（若有尿毒症 / 容積過多）；腎臟移植評估

減緩進展的關鍵介入

  • 血壓控制：理想 <120/80；ACEI 或 ARB（不要兩者合用）；啟動後 1-2 週確認 Cr 和 K（停藥若 Cr ↑ >30% 或 K >5.4）
  • SGLT2i：eGFR ≥20 且 DM 或蛋白尿 → CKD 進展和死亡率 ↓ 30-40%（eGFR 初期暫時下降 <5，之後保護作用）
  • 糖尿病：嚴格血糖控制；ACEI/ARB + SGLT2i + GLP-1 RA（eGFR ≥15）+ MRA（eGFR ≥25 且 K 正常）
  • Statin：年齡 >50 均建議（獨立於 ASCVD 風險）
  • 飲食：限鈉（HTN 時）、限鉀（少尿或高鉀時）、限磷、適度蛋白攝取

CKD 相關併發症處置

  • 代謝性酸中毒：NaHCO₃ 或 Sodium citrate 補充（目標 HCO₃ 正常）
  • 貧血：目標 Hb 10-11.5 g/dL；EPO（Epoetin 或 Darbepoetin）；IV 鐵（透析中常用）；HIF-PHI（Daprodustat）可作替代
  • 尿毒性出血：dDAVP 0.3 μg/kg IV 或 3 μg/kg 鼻內
  • 尿毒性搔癢：控制磷、抗組織胺、SSRI、Gabapentin、Difelikefalin（透析中）
  • 次發性副甲狀腺機能亢進：磷結合劑（隨餐服用）；非含鈣磷結合劑（Sevelamer）↓ 死亡率；Vit D 或 Calcitriol；Cinacalcet；副甲狀腺切除術
  • Calciphylaxis：傷口護理 / 清創；↓ PTH；停 Vit D / 鈣 / Warfarin（改 DOAC）；Na thiosulfate；預後差（1年死亡率 60%）

高血鈉 Hypernatremia（>145 mEq/L）
  • 原因：自由水丟失（腹瀉、發燒、尿崩症）或鈉攝入過多
  • 缺水量計算：0.6 × 體重(kg) × [Na/140 – 1]
  • 例：70kg，Na=160 → 0.6×70×(160/140-1) = 6L
  • 矯正方式：D5W 或 0.45% NS（自由水補充）
  • 矯正速率：每小時降 ≤0.5 mEq/L，每日 ≤10 mEq/L
  • 矯正太快 → 腦水腫。每 2-4 小時測 Na⁺，動態調整

鈉水恆定性 Sodium & Water Homeostasis
鈉異常的核心概念：鈉濃度異常反映的是水分失衡，不是鈉本身多寡。低鈉幾乎都是水分相對過多；高鈉幾乎都是水分相對不足。ADH 控制水分，醛固酮控制鈉總量（因而控制容積）。低鈉症的評估從血清滲透壓 → 容積狀態 → 尿液滲透壓和尿鈉三步驟走。
判斷框架
低鈉血症評估框架

第一步：測血清滲透壓

  • 低張性（POsm <280）：最常見，真正水分相對過多
  • 等張性（POsm 280-295）：罕見人工假象——高血脂或高蛋白血症
  • 高張性（POsm >295）：另一有效滲透物質增加（高血糖、甘露醇）將水拉入血管；血糖每升高 100 mg/dL，血鈉下降 ~2 mEq/L

第二步（低張性）：評估容積狀態

容積狀態尿鈉常見原因
低容積（↓ total Na，↓ TBW） 腎臟流失：UNa >20；腎外流失：UNa <10 腎臟：Thiazide（最常見）、鹽耗損腎病、Addison's；腎外：腹瀉、嘔吐、第三腔隙
等容積（↑ TBW relative to Na） UNa >20（SIADH）；UNa <20（原發性多飲、低溶質攝入） SIADH；原發性多飲；甲狀腺低下；糖皮質素不足
高容積（↑ total Na，↑↑ TBW） 通常 UNa <10（晚期腎衰除外） CHF、肝硬化、腎病症候群、晚期腎衰竭

SIADH 原因

低鈉血症治療原則
矯正速度不能超過 6 mEq/L/天（慢性）至 8 mEq/L/天（急性）——過快矯正 → 中央腦橋髓鞘破壞（CPM/ODS），出現截癱、構音障礙。若意外過快矯正 → 緊急用 dDAVP ± D5W 逆轉。
情境治療方式
急性症狀性低鈉（癲癇、意識改變） 初始快速矯正 2 mEq/L/h × 2-3h 直到症狀緩解；3% NaCl ± dDAVP（諮詢腎科）；每 2h 抽血監測
低容積低鈉 0.9% NS 緩慢補液；容積恢復後 ADH 抑制 → 注意過快矯正 → 若 Na 快速上升 → D5W ± dDAVP

  • NS 在 SIADH 中可能加重低鈉：若 UOsm 616，1L NS（308 mOsm）將在 0.5L 尿中排出 → 淨增 0.5L 自由水；相比之下 3% NaCl 在更多尿液中排出 → 淨失去自由水

高鈉血症
  • 幾乎都是低張液體流失 ± 無法獲取水分（高鈉是強烈口渴刺激，通常只在意識改變或無法取水時才發生）

依 UOsm 分類原因

  • UOsm >700-800（腎外水分流失）：嘔吐、腹瀉、鼻胃管引流；不感性流失（發燒、運動、燒傷）
  • UOsm <700-800（腎臟水分流失）：滲透性利尿（高血糖、甘露醇、尿素）；中樞性或腎性尿崩症（DI）
  • UOsm >700-800（鈉過多）：高張 NaHCO₃、礦物皮質素過多

治療

  • 慢性（>48h）：矯正 ≤12 mEq/天（過快 → 腦水腫）
  • 急性（<48h）：可每小時降 2 mEq/L 直到 Na 145
  • 計算自由水缺乏量：1L D5W 給 70kg Na=160 的患者 → 降鈉約 3.7 mEq
  • 液體選擇：D5W（純自由水）；½NS（每升含 500 mL 自由水）；¼NS（每升含 750 mL 自由水）
  • 中樞性 DI：dDAVP（首選）+ 低鈉低蛋白飲食 + HCTZ
  • 腎性 DI：治療原因 + 限鈉 + HCTZ（輕度容積不足 → ↓ 自由水輸送 → 矛盾性效果）；鋰引起 → 考慮 Amiloride；Indomethacin

判斷框架
ABG 四步驟判讀法
  • 1
    pH：<7.35 = 酸中毒；>7.45 = 鹼中毒；7.35-7.45 = 正常
  • 2
    PaCO₂：<35 = 呼吸性鹼（過度換氣）；>45 = 呼吸性酸（低換氣）
  • 3
    HCO₃⁻：<22 = 代謝性酸；>26 = 代謝性鹼；正常 22-26 mEq/L
  • 4
    代償是否適當？ 不適當 = 混合型酸鹼異常
數值判讀
代償公式（判斷是否混合型）
  • 代謝性酸中毒：預期 PaCO₂ = 1.5×HCO₃⁻ + 8 ± 2（Winter's formula）
  • 代謝性鹼中毒：預期 PaCO₂ = 0.7×HCO₃⁻ + 21 ± 2
  • 呼吸性酸中毒（急性）：HCO₃⁻ 每上升 1 mEq，PaCO₂ 上升 10
  • 呼吸性酸中毒（慢性）：HCO₃⁻ 每上升 3.5 mEq，PaCO₂ 上升 10
  • 呼吸性鹼中毒（急性）：HCO₃⁻ 每下降 2 mEq，PaCO₂ 下降 10
  • 實際值偏離預期值 → 混合型酸鹼異常，要找第二個原因
數值判讀
Anion Gap 計算與鑑別
  • AG = Na⁺ – (Cl⁻ + HCO₃⁻)，正常值 8-12 mEq/L
  • AG 升高代謝性酸中毒（MUDPILES）：
  • M — Methanol；U — Uremia；D — DKA
  • P — Paraldehyde；I — Isoniazid/Iron；L — Lactic acidosis；E — Ethylene glycol；S — Salicylates
  • 正常 AG 代謝性酸中毒：腹瀉、RTA（腎小管性酸中毒）、NS 大量輸液
  • 白蛋白每下降 1 g/dL，AG 正常值降低 2.5，低白蛋白患者要校正 AG

常見臨床酸鹼情境
  • Sepsis：代謝性酸（Lactic acidosis）+ 呼吸性鹼（過度換氣）
  • COPD 急性惡化：呼吸性酸中毒（CO₂ 滯留）

緊急處置
緊急透析指征：AEIOU
  • A — Acidosis：pH <7.1，藥物無法矯正的嚴重代謝性酸中毒
  • E — Electrolytes：K⁺ >6.5 或難以控制的高血鉀（ECG 異常）

  • 血液透析（HD）：每週 3 次、每次 4h，清除效率高，需血管通路（瘻管/導管）

操作方法

  • 新鮮尿液（若有 Foley 從近端口取）；離心 3-5 分鐘 1500-3000 rpm；倒掉上清液；搖晃試管底部重懸沉渣；滴於玻片加蓋玻片；高倍（high dry）觀察；相差顯微鏡評估 RBC 型態

細胞

  • RBC：評估數量和型態——異形 RBC（Acanthocytes、棘形）→ 腎絲球源性出血；均一正常形態 → 腎外或非腎絲球源
  • WBC：PMN → UTI；嗜酸性球（需特殊染色）→ AIN；GN 也可見 WBC
  • 上皮細胞：腎小管上皮細胞（ATN）；移行上皮（膀胱或輸尿管）；鱗狀上皮（通常是污染）

  • 排除 UTI 和腎結石
  • 低風險：4-6 週後重複尿液分析；若仍陽性 → 膀胱鏡 + 影像
  • 中風險（男 40-59 歲或女 50-59 歲或 10-30 包年抽菸）：膀胱鏡 + 腎臟超音波
  • 高風險（≥60 歲或 >30 包年）：CT 泌尿系統攝影（Se 96%，Sp 98%）+ 膀胱鏡
  • 大量血尿合併膀胱阻塞 → 膀胱沖洗 + 三腔 Foley 持續膀胱沖洗（CBI）
腎結石 Nephrolithiasis
腎結石的急性處置以止痛和補液為主；≤10 mm 的輸尿管結石給 α-blocker 促進排石。慢性預防靠大量喝水（目標尿量 >2L/天）和針對結石類型的飲食 / 藥物介入。24 小時尿液收集是預防策略的最重要依據。

臨床表現與診斷

臨床表現

  • 血尿（缺乏不能排除）、腰腹絞痛（陣發性、劇烈、放射至腹股溝）、噁心嘔吐、排尿不適或頻尿
  • 輸尿管阻塞：結石 >5 mm 不易自然排出 → 單側腎臟時可引起 AKI
  • UTI 風險：結石近端尿液培養可能正常（遠端尿液可能稀釋感染訊號）

診斷

  • 無顯影劑 CT（首選）：Se 97%、Sp 96%；可見輸尿管擴張（無結石 → 近期排出）
  • 腎臟超音波：Se 57%、Sp 98%；可作為初步篩查（避免輻射，孕婦或兒童優先）
  • X 光（KUB）：草酸鈣和磷酸鈣不透明；尿酸結石透明（不顯影）
  • 尿液過濾結石分析：確認結石類型指導預防
  • 基礎抽血：電解質、BUN/Cr、Ca、PO₄、PTH；尿液分析 + 培養
  • 24 小時尿液（急性期 >6 週後收集 ×2）：Ca、PO₄、草酸鹽、檸檬酸鹽、Na、Cr、pH、K、尿量
緊急處置
急性腎絞痛處置
  • 止痛：NSAIDs + 鴉片類（兩者合用優於單用）；NSAID 對腎絞痛效果良好（抑制前列腺素 → 減少輸尿管痙攣）
  • 補液：確保足夠水分補充
  • 排石藥物（≤10 mm）：α-blocker（Tamsulosin）效果優於 CCB → ↑ 自然排石率和速度
  • 緊急泌尿科評估 / 住院適應症：
  • 輸尿管阻塞（尤其單側腎臟或移植腎）
  • 尿膿毒症（阻塞合併感染）→ 緊急引流（輸尿管支架或腎造瘺）+ 抗生素
  • 頑固性疼痛或嘔吐
  • 明顯 AKI
  • 泌尿科介入：體外震波碎石術（ESWL）；輸尿管鏡取石；腹腔鏡或經皮腎鏡取石術
用藥指引
依結石類型的慢性預防
  • 所有類型（最重要）：大量喝水，目標尿量 >2 L/天

草酸鈣 / 磷酸鈣結石

  • 飲食：↓ 鈉和肉類（↑ 鈣排泄）；↓ 草酸鹽食物（菠菜、堅果、巧克力）和蔗糖 / 果糖；不需限制飲食鈣（反而要適量補充，隨餐）
  • Thiazide（如 Hydrochlorothiazide）：↓ 尿鈣排泄（高鈣尿時）
  • Potassium citrate：尿液檸檬酸鹽低時（檸檬酸鹽抑制結石形成）；也用於鹼化尿液
  • Allopurinol：尿酸排泄量高時（尿酸可作為草酸鈣結石的核心）

尿酸結石

  • 大量液體攝入；尿液鹼化（K-citrate）目標 pH 6.5-7；Allopurinol（尿酸生成過多時）

Struvite（感染性）結石

  • 治療和預防 UTI；泌尿科介入清除結石；Acetohydroxamic acid（尿素酶抑制劑，副作用多，限經驗豐富醫師使用）

胱胺酸結石

  • 大量液體攝入；尿液鹼化（K-citrate）目標 pH 7-8；D-penicillamine 或 Tiopronin（結合胱胺酸 ↓ 結晶）
  • 24 小時尿液草酸鈣飽和度最高的時段是夜間 → 睡前多喝水特別有效
