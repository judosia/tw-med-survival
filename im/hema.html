貧血 Anemia
貧血定義：Hct <41% 或 Hb <13.5 g/dL（男）；Hct <36% 或 Hb <12 g/dL（女）。評估從網狀紅血球指數（RI）開始：RI >2% = 骨髓反應充分（周邊溶血或失血）；RI <2% = 骨髓增生不良。再依 MCV 縮窄診斷。缺鐵性貧血是最常見的原因，先找出血點再補鐵；B12 缺乏不要只補葉酸（掩蓋神經症狀）。
判斷框架
貧血診斷評估
  • 症狀：↓ O₂ 輸送 → 疲勞、勞動性呼吸困難、心絞痛（若有 CAD）
  • 體徵：蒼白（黏膜、掌心皺褶）；心搏過速；直立性低血壓；黃疸（溶血）；脾腫大（地中海貧血、腫瘤、慢性溶血）；瘀點 / 紫斑（出血或血小板異常）；舌炎（缺鐵、葉酸、B12 缺乏）；反甲（缺鐵）；神經異常（B12 缺乏）

診斷步驟

  • CBC + 分類計數：RBC 參數含網狀紅血球、MCV（注意混合病因可 → 正常 MCV）、RDW
  • 網狀紅血球指數（RI）：= [網狀紅血球% × (患者 Hct / 正常 Hct)] / 成熟因子（Hct 45% = 1，35% = 1.5，25% = 2，20% = 2.5）；RI >2% → 骨髓反應充分；RI <2% → 增生不良
  • 周邊血液抹片：選 RBC 剛好互相接觸的區域；看 RBC 大小、形狀、包涵體；WBC 形態；血小板計數
  • 依需要加送：溶血指標（若 RI >2%）；鐵 / TIBC / ferritin；葉酸、B12；LFT；Cr；TFT；Epo；Hb 電泳；酵素 / 基因突變篩查
  • 骨髓穿刺和切片：必要時
用藥指引
缺鐵性貧血（IDA）
  • 機轉：骨髓鐵↓ + 體內鐵儲存耗竭 → ↓ 血基質合成 → 小球性貧血；疲勞等症狀可在貧血出現前就有
  • 其他症狀：情緒改變、落髮、口角炎、萎縮性舌炎、Pica（食冰、黏土等非營養物質）、反甲；Plummer-Vinson 症候群（IDA + 食道蹼 + 萎縮性舌炎）
  • 病因：慢性出血（GI〔含癌症〕、月經、寄生蟲、NSAIDs）；↓ 供應（營養不良；吸收↓：乳糜瀉、Crohn's、↑ 胃 pH、次全胃切除術）；↑ 需求（懷孕）
  • 診斷：↓ Fe、↑ TIBC、↓ 鐵飽和度（Fe/TIBC）、↓ Ferritin（尤其 <30）、↑ 血小板；務必考慮 GIB 評估，含 H. pylori 血清學；考慮乳糜瀉抗體
  • 治療：口服鐵（~6 週矯正貧血；~6 個月補充儲存量）；嚴重貧血、持續出血、手術前 Epo 療法、住院患者 → IV 鐵（注意：Ferumoxytol 可使 MRI T1 信號偏亮影響判讀）
機轉說明
地中海貧血（Thalassemia）

α-地中海貧血（α-globin 基因缺失）

  • 正常 4 個 α 基因；好發於東南亞、地中海、非洲、中東
  • 3 個 α → 靜默帶菌；2 個 α → α-thal 輕型（輕度貧血）；1 個 α → HbH（β4）病（嚴重貧血 + 溶血 + 脾腫大）；0 個 α → Hb Barts（γ4）= 子宮內缺氧和胎兒水腫

β-地中海貧血（β-globin 基因突變）

  • 正常 2 個 β 基因；好發於地中海（希臘 / 義大利）、非洲、亞洲；突變可為缺失（β⁰）或↓產物（β⁺）
  • 1 個突變 β 基因 → thal 輕型（不需輸血）；2 個突變 β 基因 → 依嚴重度分為需輸血（TDT）或非輸血依賴（NTDT）
  • 黃疸（溶血）；多器官損害（鐵過載）；心肌病變；PAH；↓ 骨密度；肝脾腫大（骨髓外造血）；膽結石
  • 診斷：小球性貧血；排除缺鐵；MCV/RBC <13（Mentzer 指數，Se 60%、Sp 98%）；± ↑ 網狀紅血球；嗜鹼性點彩；Hb 電泳（α-thal 輕型正常；β-thal 輕型 ↑ A2）；基因檢測
  • 治療：輸血（需要時）；慢性溶血補充葉酸；鐵螯合治療；考慮 Luspatercept（↓ TGF-β 信號 → 改善紅血球生成）；脾切除術；重症 → HSCT；基因治療（Exa-cel、Beti-cel 已核准）
用藥指引
巨球性貧血（葉酸 / B12 缺乏）
  • 機轉：DNA 合成障礙 → 細胞質成熟快於細胞核 → 無效紅血球生成和大球性貧血；也見於 MDS
  • 抹片：嗜中性球過度分葉；大橢圓形紅血球；大小不等；形態多樣
  • ↑ LDH + ↑ 間接膽紅素（無效紅血球生成）

葉酸缺乏

  • 儲存量僅供 2-3 個月
  • 病因：營養不良（酗酒、厭食、老年人）；↓ 吸收（乳糜瀉）；↑ 需求（慢性溶血、懷孕、惡性腫瘤、透析）；藥物（MTX、Pyrimethamine、TMP）
  • 診斷：↓ 葉酸；↓ RBC 葉酸；↑ 同半胱胺酸但甲基丙二酸（MMA）正常（不同於 B12 缺乏）
  • 治療：葉酸 1-5 mg PO qd 1-4 個月；先排除 B12 缺乏再補充（葉酸可逆轉 B12 缺乏的血液異常但不能逆轉神經異常，且可「竊取」B12 儲存 → 神經合併症惡化）

B12 缺乏

  • B12 只存在於動物性食品；儲存量供 2-3 年；與內在因子（IF，胃壁細胞分泌）結合後在迴腸末端吸收
  • 病因：惡性貧血（PA，抗胃壁細胞自體免疫疾病，↑ 胃癌風險）；吸收↓（胃切除術、乳糜瀉、Crohn's、Metformin）；素食者；腸道細菌過度增殖；魚條蟲
  • 神經表現（亞急性聯合脊髓退化症）：影響周邊神經、脊髓後索和側索 → 麻木、感覺異常、↓ 振動和位置覺、共濟失調、失智、情緒改變
  • 診斷：↓ B12（甚至低正常值）；↑ 同半胱胺酸 + ↑ MMA（關鍵）；anti-IF Ab；惡性貧血時 ↑ Gastrin
  • 治療：B12 1 mg IM qd × 7 天 → qwk × 4-8 週 → 終身 qmo（惡性貧血必須 IM；非惡性貧血可口服）；神經異常在 6 個月內治療可逆轉
機轉說明
慢性發炎性貧血（ACI）+ 再生不良性貧血

慢性發炎性貧血（ACI）

  • 機轉：↑ Hepcidin（IL-6、TNF-α 誘導）→ ↓ 鐵利用 + 功能性缺鐵；↓ Epo 反應性
  • 病因：自體免疫疾病、慢性感染、發炎、HIV、惡性腫瘤
  • 診斷：↓ Fe、↓ TIBC（鐵飽和度通常正常或偏低）；± ↑ Ferritin；通常正血球正色素（~70%）但若持久可呈小球性；同時存在缺鐵性貧血很常見（鑑別：Ferritin <100 或 Fe/TIBC <20% 提示同時有缺鐵）
  • 治療：治療原發疾病；若 Ferritin <100 或 Fe/TIBC <20% → 鐵劑；考慮 ESA（Epo <500 時）；只在症狀性且等待時間不足以等 Epo 反應時才輸血

再生不良性貧血（Aplastic Anemia）

  • 全血球減少 + ↓ 網狀紅血球 + 骨髓切片低細胞性；2-5/百萬/年；雙峰（青少年 + 老年）
  • 病因：特發性（1/2-2/3）；幹細胞破壞（放射線、化療、苯）；藥物反應（Chloramphenicol、NSAIDs、磺胺類、金、Carbamazepine、抗甲狀腺藥）；病毒（HHV-6、HIV、EBV、Parvovirus B19）；免疫疾病（SLE）；PNH；Fanconi 貧血；短縮端粒（TERT、TERC 突變）；體細胞突變（~50% 有 PNH 克隆）
  • 治療（依嚴重度和年齡）：中度 AA → 考慮 HSCT 或「三聯免疫抑制治療」（ATG + Cyclosporine + Eltrombopag）；重度 / 極重度 AA → HSCT（年輕者）或三聯 IST
機轉說明
鐵粒幼細胞性貧血 + 其他正球性貧血

鐵粒幼細胞性貧血（Sideroblastic Anemia）

  • 機轉：RBC 前體細胞內血基質生合成缺陷
  • 病因：遺傳性 / X 連鎖（ALAS2 突變最常見）；特發性；MDS 伴環形鐵粒幼細胞；可逆性（酒精、鉛、INH、Chloramphenicol、銅缺乏、低體溫）
  • 診斷：MCV 可變；↑ 鐵飽和度、↑ Ferritin；嗜鹼性點彩；RBC Pappenheimer 小體（含 Fe 包涵體）；骨髓中的環形鐵粒幼細胞（含鐵粒線體）；基因檢測
  • 治療：治療可逆原因；Pyridoxine / Thiamine（依亞型）；MDS 相關治療；嚴重貧血支持性輸血 + 鐵螯合治療

非巨球性大球性貧血

  • 肝臟疾病（常大球性，可見靶形紅血球或有溶血的棘紅血球）；酒精（骨髓抑制 + 大球性，獨立於葉酸 / B12 缺乏或肝硬化）；網狀紅血球增多；甲狀腺低下；MDS；藥物（Zidovudine、5-FU、Hydroxyurea、Ara-C）

PNH（陣發性夜間血色素尿症）

  • 獲得性 PIG-A 基因突變 → GPI 錨缺乏（↓ CD55、CD59）→ 補體介導 RBC 溶解、血小板聚集、高凝狀態；常與再生不良性貧血和 MDS 相關
  • 臨床：血管內溶血性貧血；高凝狀態（靜脈 > 動脈，尤其腹腔內、腦靜脈）；平滑肌張力障礙；造血不良
  • 診斷：流式細胞術（↓ CD55 和 CD59）；尿液含鐵血黃素
  • 治療：支持性（鐵劑、葉酸、輸血）；考慮抗凝；C5 抑制劑（Ravulizumab > Eculizumab）；補體旁路抑制劑（Iptacopan、Danicopan、Pegcetacoplan）；嚴重骨髓衰竭 → 異體 HSCT；接種莢膜細菌疫苗
RBC 異常 Red Cell Disorders
溶血性貧血的診斷靠：↑ 網狀紅血球指數（RI >2%）+ ↑ LDH + ↓ 觸珠蛋白（Haptoglobin，Se 83%、Sp 96%）+ ↑ 間接膽紅素。Coombs 試驗（DAT）陽性 = 免疫介導；陰性 = 非免疫介導（MAHA、機械性、酵素缺乏、遺傳性球形紅血球）。鐮刀型紅血球病：血管阻塞發作、急性胸部症候群、感染是最重要的住院原因；Hydroxyurea 是疾病改善藥物的一線選擇。

溶血性貧血評估

溶血指標

  • ↑ 網狀紅血球計數（RI >2%）；↑ LDH；↓ Haptoglobin（Se 83%、Sp 96%）；↑ 間接膽紅素

用藥指引
自體免疫溶血性貧血（AIHA）

溫型 AIHA（最常見）

  • 機轉：IgG 抗體在體溫下與 RBC 結合 → 脾臟 MΦ 清除
  • 病因：特發性；淋巴增生性（CLL、NHL）；自體免疫（SLE）；藥物；HIV；巴貝蟲病
  • 診斷：DAT IgG 陽性（pan-agglutinin，不同於同種抗體）；抹片見球形紅血球
  • 治療：類固醇 ± 脾切除術、IVIG、細胞毒性藥物、RTX

冷凝集素病（Cold Agglutinin Disease）

  • 機轉：IgM 在 <37°C 與 RBC 結合 → 補體固定 → 血管內溶血 + 肢端發紺（暴露於寒冷時）
  • 病因：特發性；淋巴增生性（Waldenström，單株型）；Mycoplasma pneumoniae 感染；傳染性單核球增多症（多株型）
  • 診斷：DAT C3 陽性；冷凝集素效價；熱幅度；抹片見 RBC 凝集
  • 治療：嚴重 → Sutimlimab 或血漿置換；RTX 和 / 或 Bendamustine；治療原發淋巴增生性疾病

藥物誘發溶血性貧血

  • Coombs 通常陰性；↑ LDH；停藥後改善
  • 常見藥物：抗生素（Cephalosporins、磺胺類、Rifampin、Ribavirin）；心血管（Methyldopa、Procainamide、Quinidine、Thiazide）；TCA、Phenothiazine、NSAIDs、磺醯脲類、MTX、5-FU；Rasburicase（G6PD 缺乏時）
機轉說明
G6PD 缺乏 + 遺傳性球形紅血球症

G6PD 缺乏

  • X 染色體連鎖；最常見於非洲裔或地中海裔；慢性溶血 DAT 陰性、RI >2%、↑ LDH、↓ Haptoglobin；抹片可變；Heinz 小體（氧化 Hb）→ 被脾臟去除後形成「咬痕細胞」
  • 誘發溶血的藥物：磺胺類、Dapsone、Nitrofurantoin、Rasburicase、Primaquine、Doxorubicin、Methylene blue
  • 也由感染、DKA、食物（蠶豆病，NEJM 2018；378:60）誘發
  • 治療：避免誘發因素；支持性治療

遺傳性球形紅血球症（HS）

  • RBC 膜骨架缺陷（Ankyrin、Spectrin、Band 3、Pallidin）；北歐族群最常見（1/5000）；75% 有家族史
  • 臨床：貧血、黃疸（主要在新生兒）、脾腫大、色素性膽結石
  • 診斷：抹片見球形紅血球；滲透脆性試驗（Se ~80%）；↓ EMA 結合（Se 93%、Sp 99%）；酸化甘油溶解試驗（Se 95%）
  • 治療：葉酸；輸血；中重度 HS 考慮脾切除術（需衡量未來血栓和感染風險增加）
緊急處置
鐮刀型紅血球病（Sickle Cell Disease）
  • 遺傳：β-globin 隱性突變 → HbS 在低氧時聚合 → 膜損傷、RBC 脫水、↓ 變形性 → 溶血 + 微血管阻塞；~8% 非裔美國人為異合子（「鐮刀型特徵」，通常無症狀）；~1/400 為同合子（鐮刀型紅血球病）
  • 診斷：抹片見鐮刀形 RBC 和 Howell-Jolly 小體；Hb 電泳

抗磷脂症候群（APS）

Sapporo 修訂標準（≥1 臨床 + ≥1 實驗室）

  • 臨床：血栓性（大血管靜脈或動脈血栓，含中風）；產科（3 次 <10 週自發性流產；或 ≥1 次 >10 週胎兒死亡；或 <34 週因子癲前症或胎盤功能不足早產）
  • 實驗室（同一抗體在相隔 ≥12 週 ≥2 次陽性）：Lupus anticoagulant（LA）；中至高效價抗心磷脂抗體（ACL）；或 β2-GP-I 抗體；IgG 比 IgM 更有意義；效價 >40U 比低效價或單獨 IgM 更重要

其他表現

  • 微血管 APS：DAH、網狀青斑、aPL 腎病、心臟損傷；心臟瓣膜增厚 / 疣狀贅生物；↓ 血小板
  • 病因：原發性（特發性）或繼發性（SLE 最常見；惡性腫瘤；感染；藥物反應）

治療

  • 初次靜脈血栓：Warfarin INR 2-3；若非三重陽性可考慮 DOAC
  • 初次動脈血栓：Warfarin INR 2-3 ± ASA 81 mg/天
  • 三重陽性（ACL + LA + β2-GP-I 均陽性）：優先 Warfarin（DOAC 血栓復發率更高）
  • 復發性血栓：DOAC → Warfarin；↑ INR 目標；加 ASA；LMWH / Fondaparinux
  • 微血管 APS：免疫調節治療

  • ≥3 個器官系統在 <1 週內受累 + APLA 陽性 + 組織微血栓；死亡率 44%

  • 最常見遺傳性出血疾病；vWF 功能 = 血小板黏附 + 因子 VIII 血漿載體
  • Type 1（~85%）：部分定量 vWF 缺乏；Type 2（~15%）：質性缺乏；Type 3（<1%）：完全 / 幾乎完全缺乏
  • 獲得性 vWD：惡性腫瘤、MPN（↑ 血小板數）；自體免疫；甲狀腺低下；藥物；Heyde 症候群（嚴重 AS 誘發 vWF 破壞，合併 GI AVM 出血）
  • 診斷：↓ vWF 抗原（vWF:Ag）；↓ vWF 活性；正常 Propeptide；↓ 因子 VIII；± ↑ PTT；± ↓ 血小板；多聚體分析；基因檢測
  • 治療：輕度出血 / 預防：Desmopressin（DDAVP，先做療效試驗確認 Type 1 有效）；血小板輸注；Antifibrinolytics（TXA）；嚴重出血 → vWF 濃縮製劑

  • TRALI：低血壓；無容積過多跡象；BNP 正常；胸部 X 光雙側浸潤但無心臟擴大；可能有發燒

輸血指引與血液製品

RBC 輸血閾值

  • 限制性策略（Hb <7 g/dL）：穩定非出血住院患者；ICU 患者
  • 較自由策略（Hb <8 g/dL）：急性 MI / 不穩定心絞痛；術後患者；心血管手術後
  • Hb <10 g/dL：急性 STEMI 或中風患者（目標維持 Hb 10-12）
  • 預期每單位 pRBC → Hb ↑ 1 g/dL 或 Hct ↑ 3%
  • 限制性策略在大多數住院患者（心因性休克除外）死亡率和心臟事件不劣於自由策略（NEJM MINT trial 2023）

血小板輸血閾值

  • 預防性：血小板 <10k（穩定）；<20k（敗血症或其他出血危險因子）；<50k（侵入性操作或外科手術）；<100k（神經外科手術或眼科手術）
  • 治療性（活動性出血）：血小板 <50k
  • 預期每單位血小板 → ↑ 10-50k（依體重）
  • ITP 和 TTP 不建議預防性血小板輸血（TTP 禁忌）

骨髓增生不良症候群（MDS）

真性紅血球增多症（PV）
  • 克隆性 JAK2 突變（V617F，95%）→ 不依賴細胞激素的 JAK-STAT 信號 → 紅血球系增殖
  • 臨床：Hct ↑；頭痛、視覺改變、耳鳴（高黏稠度）；大血管血栓（DVT、MI、中風）；微血管症狀（肢端感覺異常、紅斑性肢痛症）；水性搔癢（洗澡後）；脾腫大；出血（血小板功能障礙）
  • WHO 診斷標準（3 項主要 或 2 主要 + 1 次要）：主要：Hb >16.5（男）或 >16（女）或紅血球容積增加；JAK2 V617F 或 JAK2 exon 12 突變；骨髓切片：三系增生 + 多形性大型成熟巨核細胞；次要：血清 Epo <正常值

  • 搔癢：SSRIs、Ruxolitinib、IFNα
  • 約 15-20% 最終轉化為骨髓纖維化；1-3% 轉化為 AML

原發性血小板增多症（ET）

原發性骨髓纖維化（PMF）
  • 造血幹細胞突變（JAK2 V617F 50%；CALR 25%；MPL 5%；三重陰性 15%）→ ↑ 細胞激素 → 骨髓纖維化 + 骨髓外造血

  • 低危無症狀：觀察
  • 有症狀 / 中高危：JAK 抑制劑（Ruxolitinib 一線：↓ 脾腫大 + 改善憲法性症狀；Fedratinib、Pacritinib、Momelotinib 二線；Momelotinib 對貧血較有利）

  • 異體 HSCT：唯一治癒性治療；適合年輕且中高危患者

白血病 Leukemia
白血病分為急性（AML/ALL，快速進展，需立刻治療）和慢性（CML/CLL，較緩慢，部分可觀察等待）。AML 中 APL（M3，t(15;17)）是緊急情況——DIC 死亡率高，ATRA 必須立刻給（不要等基因確認）。CML：BCR-ABL 抑制劑（TKI）是標準治療，大多數患者預後接近正常人。CLL：低危可觀察等待；需要治療時首選 BTK 抑制劑（Ibrutinib、Acalabrutinib）或 BCL-2 抑制劑（Venetoclax）。

急性骨髓性白血病（AML）
  • AML 定義：骨髓或血液芽細胞 ≥20%（或特定分子標記如 t(8;21)、inv(16)、t(15;17) 不論芽細胞比例）
  • 流行病學：最常見急性白血病（成人）；中位年齡 67 歲；5 年存活率 ~30%（年輕患者可達 50-70%）
  • 病因：特發性；繼發性（t-AML）：化放療後，MDS 轉化；易感因子：Down 症、先天性骨髓衰竭症候群、苯暴露
  • 臨床：全血球減少症狀（貧血、感染、出血）；高白血球血症（WBC >100k → 白血球鬱積症〔呼吸困難、神智改變〕）；骨痛；臟器浸潤（肝脾、淋巴結、牙齦〔急性單核球白血病〕、皮膚〔綠色瘤〕）；DIC（APL 尤其嚴重）
  • 診斷：CBC + 抹片；骨髓切片（芽細胞 ≥20%）；細胞遺傳學（核型）；分子標記（FLT3-ITD、NPM1、CEBPA、IDH1/2、TP53 等）；流式細胞術（免疫表型）

APL（M3，t(15;17)）——緊急！

高度懷疑 APL → 立刻給 ATRA，不要等基因確認！DIC 是早期死亡最常見原因。
  • PML-RARA 融合基因（t(15;17)）；抹片見 Auer rods、法戈細胞（Faggot cells）；DIC（促凝顆粒釋放）→ 積極補充 Fibrinogen（目標 >150 mg/dL）、血小板（目標 >50k）、FFP
  • 治療：低危（WBC <10k）→ ATRA + 三氧化二砷（ATO）；高危（WBC ≥10k）→ ATRA + ATO + Idarubicin；分化症候群（ATRA/ATO 誘導發燒、肺浸潤、水腫、低血壓）→ 立刻 Dexamethasone 10 mg IV bid + 考慮暫停 ATRA/ATO

非 APL AML 治療

  • 適合強化治療者（younger / fit）：誘導（7+3：Cytarabine × 7 天 + Anthracycline × 3 天）；± 靶向藥物（FLT3 抑制劑〔Midostaurin、Quizartinib〕加在誘導和維持中；IDH1 抑制劑〔Ivosidenib〕；IDH2 抑制劑〔Enasidenib〕）；完全緩解後鞏固（高強度化療 or 異體 HSCT，依危險度）
  • 年長 / 不適合強化治療者（older / unfit）：Venetoclax + HMA（Azacitidine 或 Decitabine）；反應率 ~70%，中位 OS >14 個月；CPX-351（脂質體 Daunorubicin + Cytarabine，用於繼發性 AML）
  • 危險分層：良好（t(8;21)、inv(16)、NPM1 無 FLT3-ITD、CEBPA 雙突變）→ 化療後高劑量 Ara-C 鞏固；中危 → 化療後鞏固或 HSCT；不良（TP53、複雜核型、t(6;9)、inv(3)）→ HSCT（若可行）

淋巴瘤 Lymphoma
淋巴瘤分為霍奇金淋巴瘤（HL）和非霍奇金淋巴瘤（NHL）。HL 的特徵是 Reed-Sternberg 細胞（雙核「貓頭鷹眼」），治療後 5 年存活率 >85%。NHL 依侵犯性分類：惰性（慢性，不可治癒但長期存活）vs 侵犯性（需立刻治療，部分可治癒）。DLBCL（最常見 NHL，~30%）一線 R-CHOP；濾泡型淋巴瘤（最常見惰性 NHL）低危可觀察等待。PET-CT 是治療反應評估的標準工具。

  • CXCR4 突變 → BTK 抑制劑反應較差；血漿置換：用於高黏稠度急症（不會降低腫瘤負荷）
造血幹細胞移植 Hematopoietic Stem Cell Transplantation
HSCT 分為自體（Auto-HSCT）和異體（Allo-HSCT）。Auto-HSCT = 自己的幹細胞，無 GvHD，主要用於 MM 和淋巴瘤。Allo-HSCT = 他人的幹細胞，有移植物對腫瘤效應（GvL），可治癒血液惡性腫瘤，但有 GvHD 風險。急性 GvHD（≤100 天，皮膚 + 肝臟 + 腸道）和慢性 GvHD（>100 天，多系統纖維化）是主要併發症。感染預防是 HSCT 後的關鍵——依造血恢復時間序列安排各類預防藥物。

前處置與早期併發症

前處置方案

  • 骨髓清除性（MA）：高劑量化療 ± 全身放射線（TBI）→ 徹底清除腫瘤和免疫系統；適合年輕和體能好的患者
  • 減強度（RIC）或非骨髓清除性（NMA）：低劑量化療 ± TBI → 依靠 GvL 效應而非高劑量化療殺瘤；適合年長 / 體弱患者；較低毒性但復發率 ↑

早期（移植後 0-30 天）

  • 骨髓空白期（第 0-14 天）：全血球減少 → 感染（細菌 + 黴菌）；出血
  • 造血恢復：嗜中性球 ANC >500 通常在第 10-14 天（PBSC）或第 14-21 天（骨髓）
  • 口腔炎：前處置毒性；預防：冰晶 cryotherapy（Melphalan 方案）；Palifermin
  • 肝竇阻塞症候群（SOS / VOD）：肝竇內皮損傷 → 疼痛性肝腫大 + 腹水 + 黃疸（移植後 <21 天）；危險因子：Bu/CY 方案、前次肝臟損傷；重症 → Defibrotide
  • 移植失敗（植入失敗）：原發性（未植入）或繼發性（植入後喪失）；需再移植

免疫治療 Cancer Immunotherapy
癌症免疫治療革命了腫瘤科實踐。免疫檢查點抑制劑（ICI）恢復 T 細胞對腫瘤的免疫監控，但可引起免疫相關不良事件（irAE）影響幾乎任何器官——辨識和處置 irAE 是住院醫師的必備技能。CAR-T 細胞治療在 B 細胞惡性腫瘤和 MM 有突破性療效，但細胞激素釋放症候群（CRS）和免疫效應細胞相關神經毒性症候群（ICANS）是兩大急性毒性。雙特異性抗體（BiTE）同時結合腫瘤抗原和 T 細胞，機轉類似 CAR-T 但無需製造個人化細胞。
