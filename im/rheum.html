風濕病總論 Approach to Rheumatic Disease
風濕病評估從關節痛定性開始：關節內（活動和被動 ROM 都痛）vs 關節外（主動 ROM 比被動 ROM 更痛）；發炎性（腫、熱、紅、晨僵 >30 分鐘）vs 非發炎性。自體抗體只在有臨床懷疑時才檢查——ANA 在一般人群中陽性率 5-30%，單獨陽性無診斷意義。住院患者常見的風濕科急症：DAH（ANCA 血管炎、Goodpasture）、RPGN（SLE、ANCA）、SRC（硬皮病腎危象）。
機轉說明
風濕病主要分類
  • 發炎性關節炎：晶體性（痛風、CPPD）；RA；脊椎關節炎；成人斯蒂爾病（AOSD）
  • 結締組織疾病：SLE；修格連症候群；硬皮病；發炎性肌病（DM/PM）；MCTD
  • 血管炎：大血管（GCA、Takayasu）；中血管（PAN）；小血管（ANCA 相關：GPA、EGPA、MPA；免疫複合體：IgA 血管炎、冷凝球蛋白）；Behçet
  • 其他：IgG4 相關疾病；自體發炎疾病（FMF、TRAPS、VEXAS）；Sarcoidosis；HLH / MAS；復發性多軟骨炎

自體抗體判讀

發炎指標

  • ESR：間接測量（RBC 聚集 ← 急性期蛋白如 fibrinogen、Ig）；上升和下降均慢；年齡、懷孕、貧血、肥胖、ESRD 可使其 ↑；ESR >100 的鑑別：惡性腫瘤（尤其 MM、淋巴瘤）、GCA 或其他血管炎、心內膜炎、TB、骨髓炎
  • CRP：直接測量（肝臟產生的蛋白）；通常比 ESR 更早隨治療上升和下降

ANA（抗核抗體）

  • 篩查針對細胞核蛋白的抗體；只在臨床懷疑 CTD 時才測
  • 健康人群陽性率：1:40（25-30%）；1:80（10-15%）；≥1:160（5%）
  • ANA 陽性 + 高度懷疑 CTD → 進一步測 anti-dsDNA、Smith、Ro/La、RNP、Scl-70 和肌炎特異性抗體
  • ANA 也陽性於：AIH、PBC、甲狀腺疾病、特定感染和惡性腫瘤、IBD、IPF
  • ANA 與疾病活動性不相關 → 不要連續追蹤 ANA

感染性關節炎 + 滑囊炎 Infectious Arthritis & Bursitis
急性單關節炎 = 感染性關節炎除外之前不要輕易放棄。關節穿刺是診斷金標準——盡可能在給抗生素前先抽液。WBC >50k + PMN 為主 → 高度懷疑細菌感染，但低 WBC 不能排除。晶體和感染可以並存 → 永遠同時送晶體分析和培養。播散性淋球菌感染（DGI）是性活躍年輕成人最常見的感染性關節炎，關節液培養陽性率只有 50%，要同時送生殖道 / 口咽 / 直腸 PCR。
緊急處置
細菌性（非淋球菌性）關節炎

危險因子

  • 免疫抑制宿主：DM、酗酒、HIV、年齡 >80 歲、SLE、癌症、類固醇使用
  • 受損關節：RA、OA、痛風、外傷、前次手術 / 人工關節、前次關節穿刺（少見）
  • 菌血症來源：IVDU、心內膜炎；或來自鄰近感染（蜂窩組織炎、骨髓炎）

臨床表現

  • 急性單關節炎（>80%）+ 疼痛（Se 85%）+ 腫脹（Se 78%）+ 熱感
  • 最常見：膝關節；也見於髖、腕、肩、踝；IVDU 常累及軸向關節（骶髂關節、胸鎖關節）
  • 全身症狀：發燒（Se 57%）、寒顫（19%）、出汗（27%）
  • 感染可擴散形成瘻管、膿瘍或骨髓炎

診斷

  • 關節穿刺（金標準）：WBC 通常 >50k（Se 62%、Sp 92%）但可 <10k；>90% PMN；Gram stain 陽性率：Staph ~75%、GNR ~50%；培養陽性 >90%；關節滑膜切片最敏感
  • 血培養 >50% 陽性（多關節受累時 ~80%）
  • 白血球增多（Se 90%、Sp 36%）；↑ ESR/CRP（Se >90%）
  • X 光早期正常；2 週後可見骨侵蝕、關節間隙狹窄、骨髓炎；CT / MRI 用於髖關節或疑似硬膜外膿瘍

治療

Gram stain / 族群初始抗生素
GPC 叢狀（S. aureus）；正常關節 / 受損關節 / 人工關節Vancomycin（後依藥敏改 Antistaphylococcal PCN 或 Cefazolin）
GPC 鏈狀（Streptococci）；健康成人 / 無脾症PCN-G 或 Ampicillin
雙球菌（N. gonorrhoeae）；性活躍年輕成人第三代 Cephalosporin（Ceftriaxone、Cefotaxime）
GNR（IVDU、GI 感染、免疫抑制、外傷、老年）第三代 Cephalosporin；有假單胞菌風險時加抗假單胞菌覆蓋
  • IV 抗生素 ≥2 週 → 換口服；通常需要多次引流（關節鏡或穿刺）；預期 WBC ↓ 和培養轉陰
  • 死亡率 10-15%（多關節受累高達 50%）
用藥指引
播散性淋球菌感染（DGI）
  • 流行病學：性活躍年輕成人最常見感染性關節炎；補體末端成分缺乏者風險 ↑；女：男 = 4:1（歷史上）；MSM 發生率 ↑；<3% N. gonorrhoeae 感染播散
  • 黏膜感染前驅（常無症狀）：宮頸、尿道、肛門、口咽

兩種臨床症候群（可同時存在）

  • 關節定位型（40%）：化膿性關節炎 1-2 個關節（膝 > 腕 > 踝）
  • 關節炎-皮膚炎症候群：三聯症：多關節痛（游走性，可累及小或大關節）+ 腱鞘炎（腕、手指、踝、趾的腱鞘疼痛 / 發炎）+ 皮膚病灶（肢端和軀幹的槍金屬色膿疱，伴紅斑基底）
  • 少見併發症：Fitz-Hugh-Curtis 症候群（肝周炎）；心包炎；腦膜炎；骨髓炎

診斷與治療

  • 關節液 WBC >50k（但可 <10k）；Gram stain 陽性 ~25%；培養陽性 ~50%（Thayer-Martin 培養基）
  • 血培養：關節炎-皮膚炎症候群較陽性；定位型少見
  • 關鍵：宮頸 / 尿道 / 口咽 / 直腸 PCR 或 Thayer-Martin 培養；同時篩查 Chlamydia
  • 治療：Ceftriaxone × 7-14 天 + 經驗性 Doxycycline × 7 天（排除 Chlamydia 合併感染前）
  • 化膿性關節炎可能需關節鏡 / 沖洗（多次少見）
用藥指引
人工關節感染（PJI）
  • 流行病學：術後前 2 年風險最高；發生率低（0.5-2.4%）；危險因子：肥胖、RA、免疫抑制、類固醇、淺層手術部位感染
  • 病原體：Staphylococci（凝固酶陰性和 S. aureus）>50%；多菌性 10-20%
  • 時序分類：早期（<3 個月，手術時植入）→ 毒性菌（MRSA）；延遲性（3-24 個月）→ 低毒性菌（P. acnes、凝固酶陰性 Staph）；晚期（>24 個月）→ 血行性播種
  • 診斷：關節穿刺；ESR 和 CRP（CRP Se 73-91%、Sp 81-86%）；影像上鬆脫的硬體是線索
  • 治療：長期抗生素（初始經驗：Vancomycin + 第三/四代 Cephalosporin）+ 硬體更換（保留與 ~40% 失敗率相關）；或終身壓制性抗生素；找骨科和感染科會診
用藥指引
感染性滑囊炎（鷹嘴突 + 髕前）
  • 最常感染的是鷹嘴突滑囊和髕前滑囊；S. aureus（80%）最常見，其次 Streptococci
  • 危險因子：直接外傷、經皮接種或相鄰感染（蜂窩組織炎）；非感染性發炎（痛風、RA）也可引起滑囊炎
  • 診斷：滑囊腫脹、紅斑、以滑囊中心最痛、關節 ROM 保留；若懷疑感染 → 穿刺送細胞計數 + Gram stain + 細菌培養 + 晶體分析；WBC >20k + PMN 為主 → 懷疑細菌感染（低 WBC 很常見；晶體不能排除感染！）
  • 注意：不要穿過感染皮膚穿刺，避免將感染帶入滑囊；評估鄰近關節是否也有積液（也可能是感染的）
  • 治療：輕症 → 口服抗生素（Staphylococci + Streptococci 覆蓋）；外觀不佳 → Vancomycin；依 Gram stain / 培養調整；療程 1-3 週；每 1-3 天系列穿刺直到無積液
  • 若穿刺無法引流、有異物或壞死、或反覆 / 頑固性感染 → 手術
痛風 + 偽痛風 Crystal Deposition Arthritides
晶體性關節炎的診斷靠關節穿刺偏光顯微鏡——痛風：針狀、負雙折射（偏光軸平行時呈黃色）；偽痛風（CPPD）：菱形、弱正雙折射（偏光軸平行時呈藍色）。急性痛風發作時血清尿酸 25% 正常，不能用來診斷或排除。慢性痛風治療目標：尿酸 <6 mg/dL（痛風石患者 <5），Allopurinol 從低劑量開始（CKD 患者尤其注意），亞裔族群給藥前先篩查 HLA-B5801。

相關疾病代謝症候群；CKD；CHF3H：副甲狀腺功能亢進（Hyperparathyroidism）、低鎂血症（Hypomagnesemia）、血色素沉著症（Hemochromatosis）
X 光早期：非特異性腫脹；晚期：痛風石、「懸伸邊緣」關節侵蝕；超音波：雙輪廓徵；DECT：尿酸鹽沉積軟骨鈣化（chondrocalcinosis）：關節軟骨內線狀緻密影；常見於膝半月板、腕纖維軟骨；CT：冠狀突（CPPD 在 C1-C2）
  • 細胞內晶體特異性更高；感染可與急性發作並存 → 永遠同時送 Gram stain + 培養 + 晶體分析
用藥指引
急性痛風治療
  • Colchicine、NSAIDs 和類固醇均為一線；依副作用和共病選擇；越早給越好；持續至急性發作緩解；嚴重時考慮聯合治療
  • 已在使用降尿酸治療的患者：急性發作期間繼續使用（不要停）

降尿酸治療（慢性痛風）

適應症

  • 每年 ≥2 次發作；痛風石；痛風相關關節損傷；尿酸腎結石
  • 目標尿酸：<6 mg/dL（一般）；<5 mg/dL（痛風石患者）
  • 開始降尿酸藥的同時給予預防性 Colchicine 0.6 mg qd-bid（或 NSAIDs / 類固醇）至少 3-6 個月（避免藥物引發急性發作）

CPPD 治療
  • 無症狀軟骨鈣化：不需治療

類風濕性關節炎 Rheumatoid Arthritis
RA 是慢性、對稱性、破壞性炎症性多關節炎，以滑膜增生（pannus）為特徵。診斷靠臨床（小關節對稱性關節炎 + 晨僵 >30 分鐘 + 持續 ≥6 週）+ 血清學（RF 和 / 或 anti-CCP）。治療目標是早期達到臨床緩解或低疾病活動度——MTX 是中重度 RA 的一線 DMARD。生物製劑前必須篩查 TB 和 HBV。C1-C2 不穩定（30%）→ 擇期插管前先照頸椎 flex/ext X 光。
機轉說明
臨床表現
  • 盛行率 1/100 成人；女：男 = 3:1；高峰發病 40-75 歲
  • 關節炎：通常隱匿性發作；疼痛、腫脹、↓ 功能 + 晨僵 >30 分鐘 ≥6 週；PIP、MCP、腕、膝、踝、MTP（60% 小關節，30% 大關節，10% 兩者）；早期可為單關節（膝、肩、腕）
  • 關節變形：尺側偏移、天鵝頸（MCP 屈曲 + PIP 過伸 + DIP 屈曲）、鈕扣花（PIP 屈曲 + DIP 過伸）、cock-up 變形（趾）
  • C1-C2 不穩定（30%）：→ 脊髓病變；擇期插管前先照頸椎 flex/ext X 光

成人斯蒂爾病（AOSD）
  • 罕見自體發炎疾病（<4/百萬/年）；男 = 女；雙峰發病（15-25 歲或 36-46 歲）；與全身性幼年特發性關節炎（sJIA）可能為同一疾病

山口標準（需 5 項且 ≥2 項主要標準；排除感染、惡性腫瘤、其他風濕病、藥物反應）

  • 主要標準：發燒 ≥39°C 持續 ≥1 週（通常每天一次或兩次高燒）；關節痛 / 關節炎 ≥2 週；Still 皮疹（常與發燒同起伏）；↑ WBC（>80% PMN）
  • 次要標準：喉嚨痛；淋巴結腫大；肝脾腫大；↑ AST/ALT/LDH；ANA 陰性 + RF 陰性
  • Still 皮疹（>85%）：非搔癢性黃橙色斑疹或斑丘疹；通常在軀幹或肢體；可被外傷誘發（Koebner 現象）或熱水誘發；常與發燒同時出現

復發性多軟骨炎（Relapsing Polychondritis）
  • 罕見（1/百萬/年）；40-60 歲發病；男 = 女；軟骨結構的炎症破壞 → 萎縮和變形
  • 多部位軟骨發炎：耳廓軟骨炎（spares lobule，痛紅腫後萎縮）；非侵蝕性炎症性關節炎；肋軟骨炎；鼻軟骨炎（鞍鼻）；喉 / 氣管軟骨炎（→ 氣道阻塞）；瓣膜病變；眼（鞏膜炎 / 表層鞏膜炎）；耳蝸 / 前庭功能障礙

脊椎關節炎 Spondyloarthritis
脊椎關節炎（SpA）是一組共享臨床特徵的炎症性疾病：軸向骨骼發炎（骶髂關節炎）、周邊關節炎、肌腱端炎（enthesitis）和關節外表現（眼、皮膚、腸道）。血清陰性（無 RF / anti-CCP）。炎症性背痛口訣：IPAIN（隱匿發作、夜間疼痛、年齡 <40 歲、運動後改善、休息不改善）。HLA-B27 是風險因子但非診斷標準（AS 中 90% 陽性，但一般族群也有 6-8%）。
機轉說明
SpA 亞型與主要特徵
亞型流行病學主要特徵
僵直性脊椎炎（AS） 男：女 = 3:1；青少年至 20 多歲發病（40 歲後罕見） 進行性脊椎活動受限；晨僵；臀部疼痛；「竹節脊椎」；Schober 試驗陽性；HLA-B27 ⊕ 90%
乾癬性關節炎（PsA） 男 = 女；高峰 45-54 歲；乾癬患者中 20-30% ~15% 關節炎早於皮膚表現數年；與乾癬活動度不相關；↑ 與 HIV 相關；見下方 PsA 亞型
反應性關節炎（ReA） 男 >> 女；20-40 歲；GI 或 GU 感染後 10-30 天 關節炎 + 尿道炎 + 結膜炎（三聯症）；多數 12 個月內緩解；觸發菌：GU（Chlamydia）；GI（Shigella、Salmonella、Yersinia、Campylobacter、C. diff）

主要臨床表現

  • 炎症性背痛（IPAIN）：骶髂關節（骶髂關節炎）和脊椎關節發炎；隱匿發作、夜間疼痛、<40 歲、運動 / 熱水改善、休息不改善；對 NSAIDs 有反應；晨僵
  • 周邊關節炎：通常不對稱、寡關節、大關節、下肢 > 上肢；但 PsA 可為對稱多關節（類似 RA）
  • 肌腱端炎（Enthesitis）：肌腱 / 韌帶附著於骨的部位發炎；尤其 Achilles 腱、足底筋膜（足跟附著點）、髕骨前、外上髁
  • 脊椎強直：進行性骨刺連接椎間盤（「竹節脊椎」X 光）
  • 指炎（Dactylitis）：「臘腸指」= 整個手指發炎（關節 + 腱鞘發炎）
  • 葡萄膜炎：最常見關節外表現；前葡萄膜炎、通常單側、疼痛 + 眼紅 + 畏光 + 視力模糊
判斷框架
診斷評估
  • 血清陰性：RF 和其他自體抗體通常陰性；± ↑ ESR/CRP
  • HLA-B27：非特異性（一般族群 6-8%）；臨床高度懷疑但影像正常時最有用；AS 中 90% 陽性；其他 SpA 亞型 20-80%

影像學

  • MRI：偏好早期骶髂關節炎偵測（骨髓水腫）；也用於肌腱端炎
  • X 光：偵測晚期結構變化（骶髂關節侵蝕 / 硬化 / 強直）；橋接對稱性骨贅（「竹節脊椎」）；椎體「方形化」和去礦化（「亮角」）
  • 超音波：偵測肌腱端炎，準確性與 MRI 相當

PsA 亞型

  • 單 / 寡關節型（大關節或 DIP、指炎）：最常見的初始表現
  • 多關節型（手足小關節、腕、踝、膝、肘）：與 RA 難區分，但常不對稱
  • 殘毀性關節炎（Arthritis mutilans）：嚴重破壞性關節炎伴骨溶解，尤其手
  • 軸向型：常為單側 / 不對稱骶髂關節炎
  • DIP 型：與甲板點蝕和甲板溶離相關（nail pitting + onycholysis）
  • 皮膚表現：乾癬（邊界清晰的紅斑，覆有厚銀白色鱗屑）；環狀龜頭炎；Keratoderma blennorrhagica（掌蹠 / 陰囊 / 頭皮過度角化）；結節性紅斑；壞疽性膿皮病

反應性關節炎評估

  • GU：尿液分析、Chlamydia PCR；GI：糞便培養、C. diff 毒素；考慮 HIV
