風濕病總論 Approach to Rheumatic Disease
風濕病評估從關節痛定性開始：關節內（活動和被動 ROM 都痛）vs 關節外（主動 ROM 比被動 ROM 更痛）；發炎性（腫、熱、紅、晨僵 >30 分鐘）vs 非發炎性。自體抗體只在有臨床懷疑時才檢查——ANA 在一般人群中陽性率 5-30%，單獨陽性無診斷意義。住院患者常見的風濕科急症：DAH（ANCA 血管炎、Goodpasture）、RPGN（SLE、ANCA）、SRC（硬皮病腎危象）。

風濕病主要分類
  • 發炎性關節炎：晶體性（痛風、CPPD）；RA；脊椎關節炎；成人斯蒂爾病（AOSD）

自體抗體判讀

發炎指標

  • ESR：間接測量（RBC 聚集 ← 急性期蛋白如 fibrinogen、Ig）；上升和下降均慢；年齡、懷孕、貧血、肥胖、ESRD 可使其 ↑；ESR >100 的鑑別：惡性腫瘤（尤其 MM、淋巴瘤）、GCA 或其他血管炎、心內膜炎、TB、骨髓炎
  • CRP：直接測量（肝臟產生的蛋白）；通常比 ESR 更早隨治療上升和下降

ANA（抗核抗體）

  • 篩查針對細胞核蛋白的抗體；只在臨床懷疑 CTD 時才測
  • 健康人群陽性率：1:40（25-30%）；1:80（10-15%）；≥1:160（5%）
  • ANA 陽性 + 高度懷疑 CTD → 進一步測 anti-dsDNA、Smith、Ro/La、RNP、Scl-70 和肌炎特異性抗體
  • ANA 也陽性於：AIH、PBC、甲狀腺疾病、特定感染和惡性腫瘤、IBD、IPF
  • ANA 與疾病活動性不相關 → 不要連續追蹤 ANA

緊急處置
細菌性（非淋球菌性）關節炎

危險因子

  • 免疫抑制宿主：DM、酗酒、HIV、年齡 >80 歲、SLE、癌症、類固醇使用
  • 受損關節：RA、OA、痛風、外傷、前次手術 / 人工關節、前次關節穿刺（少見）
  • 菌血症來源：IVDU、心內膜炎；或來自鄰近感染（蜂窩組織炎、骨髓炎）

臨床表現

  • 急性單關節炎（>80%）+ 疼痛（Se 85%）+ 腫脹（Se 78%）+ 熱感
  • 最常見：膝關節；也見於髖、腕、肩、踝；IVDU 常累及軸向關節（骶髂關節、胸鎖關節）
  • 全身症狀：發燒（Se 57%）、寒顫（19%）、出汗（27%）
  • 感染可擴散形成瘻管、膿瘍或骨髓炎

診斷

  • 關節穿刺（金標準）：WBC 通常 >50k（Se 62%、Sp 92%）但可 <10k；>90% PMN；Gram stain 陽性率：Staph ~75%、GNR ~50%；培養陽性 >90%；關節滑膜切片最敏感
  • 血培養 >50% 陽性（多關節受累時 ~80%）
  • 白血球增多（Se 90%、Sp 36%）；↑ ESR/CRP（Se >90%）
  • X 光早期正常；2 週後可見骨侵蝕、關節間隙狹窄、骨髓炎；CT / MRI 用於髖關節或疑似硬膜外膿瘍

用藥指引
播散性淋球菌感染（DGI）
  • 流行病學：性活躍年輕成人最常見感染性關節炎；補體末端成分缺乏者風險 ↑；女：男 = 4:1（歷史上）；MSM 發生率 ↑；<3% N. gonorrhoeae 感染播散
  • 黏膜感染前驅（常無症狀）：宮頸、尿道、肛門、口咽

兩種臨床症候群（可同時存在）

  • 關節定位型（40%）：化膿性關節炎 1-2 個關節（膝 > 腕 > 踝）
  • 關節炎-皮膚炎症候群：三聯症：多關節痛（游走性，可累及小或大關節）+ 腱鞘炎（腕、手指、踝、趾的腱鞘疼痛 / 發炎）+ 皮膚病灶（肢端和軀幹的槍金屬色膿疱，伴紅斑基底）
  • 少見併發症：Fitz-Hugh-Curtis 症候群（肝周炎）；心包炎；腦膜炎；骨髓炎

診斷與治療

  • 關節液 WBC >50k（但可 <10k）；Gram stain 陽性 ~25%；培養陽性 ~50%（Thayer-Martin 培養基）
  • 血培養：關節炎-皮膚炎症候群較陽性；定位型少見
  • 關鍵：宮頸 / 尿道 / 口咽 / 直腸 PCR 或 Thayer-Martin 培養；同時篩查 Chlamydia
  • 治療：Ceftriaxone × 7-14 天 + 經驗性 Doxycycline × 7 天（排除 Chlamydia 合併感染前）
  • 化膿性關節炎可能需關節鏡 / 沖洗（多次少見）
用藥指引
人工關節感染（PJI）
  • 流行病學：術後前 2 年風險最高；發生率低（0.5-2.4%）；危險因子：肥胖、RA、免疫抑制、類固醇、淺層手術部位感染
  • 病原體：Staphylococci（凝固酶陰性和 S. aureus）>50%；多菌性 10-20%
  • 時序分類：早期（<3 個月，手術時植入）→ 毒性菌（MRSA）；延遲性（3-24 個月）→ 低毒性菌（P. acnes、凝固酶陰性 Staph）；晚期（>24 個月）→ 血行性播種
  • 診斷：關節穿刺；ESR 和 CRP（CRP Se 73-91%、Sp 81-86%）；影像上鬆脫的硬體是線索
  • 治療：長期抗生素（初始經驗：Vancomycin + 第三/四代 Cephalosporin）+ 硬體更換（保留與 ~40% 失敗率相關）；或終身壓制性抗生素；找骨科和感染科會診
用藥指引
感染性滑囊炎（鷹嘴突 + 髕前）
  • 最常感染的是鷹嘴突滑囊和髕前滑囊；S. aureus（80%）最常見，其次 Streptococci
  • 危險因子：直接外傷、經皮接種或相鄰感染（蜂窩組織炎）；非感染性發炎（痛風、RA）也可引起滑囊炎
  • 診斷：滑囊腫脹、紅斑、以滑囊中心最痛、關節 ROM 保留；若懷疑感染 → 穿刺送細胞計數 + Gram stain + 細菌培養 + 晶體分析；WBC >20k + PMN 為主 → 懷疑細菌感染（低 WBC 很常見；晶體不能排除感染！）
  • 注意：不要穿過感染皮膚穿刺，避免將感染帶入滑囊；評估鄰近關節是否也有積液（也可能是感染的）
  • 治療：輕症 → 口服抗生素（Staphylococci + Streptococci 覆蓋）；外觀不佳 → Vancomycin；依 Gram stain / 培養調整；療程 1-3 週；每 1-3 天系列穿刺直到無積液
  • 若穿刺無法引流、有異物或壞死、或反覆 / 頑固性感染 → 手術
痛風 + 偽痛風 Crystal Deposition Arthritides
晶體性關節炎的診斷靠關節穿刺偏光顯微鏡——痛風：針狀、負雙折射（偏光軸平行時呈黃色）；偽痛風（CPPD）：菱形、弱正雙折射（偏光軸平行時呈藍色）。急性痛風發作時血清尿酸 25% 正常，不能用來診斷或排除。慢性痛風治療目標：尿酸 <6 mg/dL（痛風石患者 <5），Allopurinol 從低劑量開始（CKD 患者尤其注意），亞裔族群給藥前先篩查 HLA-B5801。

適應症

  • 1. 複發性口腔潰瘍（≥3 次 / 年，通常第一個表現）
  • 2. 複發性生殖器潰瘍（陰唇 / 陰囊，常留疤）
  • 3. 眼部：葡萄膜炎（尤其全葡萄膜炎）、鞏膜炎、視網膜血管炎、視神經炎
  • 4. 皮膚：膿疱、丘疹、假性毛囊炎、結節性紅斑
  • 5. 針刺試驗陽性（無菌針刺前臂 → 膿疱；白人中敏感性低）
  • 其他表現：關節炎 / 關節痛（非變形性，單 / 寡關節，膝、踝、腕）± 肌腱端炎；CNS（中腦實質受累）；血管（淺或深靜脈血栓 25%；動脈狹窄、閉塞和動脈瘤）；GI（潰瘍）

治療

  • 黏膜皮膚：輕度 → 外用類固醇、Colchicine、Dapsone、Apremilast（PDE-4i）；嚴重 → 口服類固醇 + 減量藥
  • 關節炎：NSAIDs、Colchicine、類固醇、減量藥
  • 眼部：外用和 / 或全身類固醇 ± 減量藥
  • 減量藥：Anti-TNF；CYC、AZA、CsA、MTX、IFNα-2A
  • 靜脈血栓：類固醇 ± 抗凝（若有動脈瘤需謹慎）
自體發炎疾病 + 澱粉樣變 Autoinflammatory Diseases & Amyloidosis
自體發炎疾病是先天免疫過度活化所致（不同於自體免疫疾病的適應性免疫失調）。FMF 最常見，Colchicine 是有效預防藥；VEXAS（UBA1 突變）是近年新認識的疾病，幾乎只發生於男性，合併 MDS 和多系統發炎。HLH 是危及生命的過度炎症症候群——診斷靠 H-score，Ferritin 常 >10000，觸發原因（感染、自體免疫、惡性腫瘤）的治療同樣重要。澱粉樣變的核心是正確分型（AL vs ATTR vs AA），治療完全不同。
用藥指引
家族性地中海熱（FMF）
  • 遺傳：MEFV 基因隱性突變（活化 pyrin，inflammasome 上游）
  • 流行病學：好發於亞美尼亞人、塞法迪猶太人、北非人、土耳其人、阿拉伯人；10 歲前發病
  • 臨床：發燒發作 1-3 天 + 腹痛（漿膜炎，可類似急腹症）+ 胸膜炎 + 大關節單關節炎 / 關節痛 + 丹毒樣皮膚病灶；發作間隔不規律
  • 未治療的長期風險：ESRD（繼發性 AA 澱粉樣變）；腹膜黏連；不孕
  • 治療：Colchicine 有效預防並 ↓ 澱粉樣變；頑固性 → IL-1 抑制劑（Anakinra、Canakinumab）
機轉說明
TRAPS + VEXAS

TRAPS（TNF 受體相關週期性症候群）

  • TNF 受體突變 → 每 5-6 週一次的發燒發作，每次持續 >7 天 + 肌痛 + 腹痛 / 噁心 + 游走性皮疹 + 眶周水腫；AA 澱粉樣變風險
  • 流行病學：常染色體顯性，可變外顯率；盛行率 1/百萬；通常兒童期發病
  • 治療：輕度發作 → NSAIDs；典型發作 → 7-10 天類固醇療程；頻繁或嚴重 → IL-1 抑制劑；頑固性 → TNFi（Etanercept）或加 Colchicine

VEXAS（空泡、E1 酶、X 連鎖、自體發炎、體細胞）症候群

  • 機轉：造血幹細胞中 UBA1（X 染色體泛素化酶）體細胞突變 → 骨髓 / 紅血球系空泡 → 血球減少、MDS、軟骨炎、血管炎、血栓、肺泡炎、嗜中性球性皮膚病、發燒
  • 流行病學：幾乎只發生於男性（X 連鎖）；平均發病年齡 >60 歲
  • 診斷：↑ MCV；± ↓ 血小板；周邊血液體細胞 UBA1 突變（複製性造血）
  • 治療：類固醇 ↓ 發炎症狀但難以減量；考慮 JAKi 或 IL-6i；高風險 MDS → Azacitidine；異體移植
緊急處置
嗜血細胞性淋巴組織球增多症（HLH）/ 巨噬細胞活化症候群（MAS）
  • 機轉：↑↑ 免疫活性；NK 細胞 / CTL 無法↓活化的巨噬細胞 → ↑ 細胞激素（IFNγ、IL-18）→ 巨噬細胞吞噬其他血球細胞、細胞激素風暴、器官衰竭
  • 觸發原因（破壞免疫穩態）：免疫活化（感染、自體免疫發作）或免疫缺陷；~25% 家族性（穿孔素介導細胞毒性基因突變）
  • 風濕科疾病引起的 HLH 稱為 MAS（巨噬細胞活化症候群），最常見於 AOSD 或全身性 JIA

診斷標準

  • 發燒；脾腫大；血球減少（≥2 系列）；↑ TG；↓ 纖維蛋白原；血球吞噬現象（骨髓、脾臟、淋巴結）；↓ NK 細胞活性；↑ Ferritin（常 >10000，甚至 >100000）；↑ 可溶性 IL-2R（sCD25）；H-score 計算 HLH 可能性

治療

  • 原發性 HLH：Etoposide + 類固醇 ± CsA ± 鞘內 MTX ± BMT
  • 繼發性（2°）HLH / MAS：治療原發觸發原因（風濕發作、感染、癌症）；類固醇；IL-1 抑制劑；Emapalumab（Anti-IFNγ）；JAKi；頑固性 → Etoposide
  • 未治療死亡率極高；及早識別觸發原因是最重要的治療步驟

定義與分型

  • 錯誤折疊蛋白沉積為不可溶纖維（β-摺板結構）在正常組織中；關鍵是正確分型——不同類型治療完全不同
