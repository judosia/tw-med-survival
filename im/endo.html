腦下垂體疾病 Pituitary Disorders
腦下垂體疾病分為功能低下（Hypopituitarism）和功能亢進（Hyperpituitarism）。住院患者最重要的是辨識腎上腺功能不足和甲狀腺低下——兩者並存時先給類固醇再補甲狀腺素，否則可能誘發腎上腺危象。高泌乳素血症（Hyperprolactinemia）是最常見的功能性腦下垂體腺瘤，多巴胺促效劑（Cabergoline）是一線治療。
機轉說明
腦下垂體功能低下（Hypopituitarism）

病因

  • 原發性（垂體本身）：手術、放射線（約 4-5 年後出現）、腫瘤（原發或轉移）、感染、浸潤（Sarcoidosis、血色素沉著症）、自體免疫、缺血（含 Sheehan 症候群——產後垂體梗塞）、頸動脈瘤、海綿竇血栓、外傷、CTLA4/PD-L1/PD-1 抑制劑、垂體卒中、空蝶鞍
  • 繼發性（下視丘或垂體柄損傷）：顱咽管瘤、感染、浸潤、放射線、手術、外傷

各軸缺乏的臨床表現

  • 中樞性腎上腺功能不足（↓ ACTH）：類似原發性腎上腺功能不足，但無鹽分渴望和高鉀血症（醛固酮保留）；無色素沉著（ACTH/MSH 不升高）
  • 中樞性甲狀腺低下（↓ TSH）：症狀同原發性甲狀腺低下，但無甲狀腺腫；診斷靠 free T4（低）+ TSH（低或「不恰當地」正常）
  • 中樞性性腺功能低下（↓ FSH/LH）：↓ 性欲、陽萎、月經不規律或停止、不孕、肌肉流失、骨質疏鬆；睾丸縮小、腋毛/陰毛脫落
  • 低泌乳素血症（↓ Prolactin）：最常見為 Sheehan 症候群；產後無法哺乳或月經恢復失敗
  • 生長激素缺乏（↓ GH）：慢性骨質疏鬆風險 ↑、疲勞、肌肉量減少
  • 中樞性尿崩症（↓ ADH）：多見於蝶鞍外腫瘤；嚴重多尿 + 口渴 + 輕度高鈉血症；見腎內科鈉水恆定性 block

診斷與治療

  • 慢性：↓ 目標腺體激素 + ↓ 或「不恰當正常」的垂體促激素；垂體 MRI（增強）
  • 急性：目標腺體激素尚未下降，但腎上腺功能不足皮質醇缺乏可快速發展
  • 治療：補充缺乏的目標腺體激素；住院患者最重要的是辨識腎上腺功能不足和甲狀腺低下；若兩者並存 → 先類固醇再補甲狀腺素
用藥指引
腦下垂體功能亢進（Hyperpituitarism）

腦下垂體腺瘤

  • 腺瘤 → 促激素過量（若有功能，30-40% 無功能）± 壓迫其他促激素軸導致缺乏
  • 非特異性症狀：頭痛、視野缺損、複視、顱神經病變
  • <10 mm、無佔位效應、無激素過量 → 12 個月後追蹤 MRI

高泌乳素血症（最常見的功能性腺瘤）

  • 病因：泌乳素瘤（PRL 常 >100）；垂體柄壓迫 → ↓ 多巴胺抑制 → ↑ PRL（通常 <100）
  • 臨床：閉經、溢乳、不孕、↓ 性欲、陽萎；排除懷孕 / 外源性雌激素 / 甲狀腺低下 / 多巴胺拮抗劑（精神科用藥、止吐劑）/ 腎衰竭 / 肝硬化
  • 治療：無症狀 + 微腺瘤 → 追蹤；有症狀或大腺瘤 → 多巴胺促效劑：Cabergoline（70-100% 成功率，耐受性優於 Bromocriptine）；手術（藥物失敗 / 無法耐受 / GH 共分泌 / 神經症狀）；放射線（藥物手術失敗時）
  • Hook effect：極高 PRL（>1000）時免疫分析可給出假性低值 → 稀釋後重測

肢端肥大症（↑ GH，10% 腺瘤）

  • 機轉：GH → ↑ IGF-1（Somatomedin C）
  • 臨床：↑ 軟組織、關節痛、下頜增大、頭痛、腕隧道症候群、巨舌、聲音嘶啞、睡眠呼吸中止、DM、HTN/心肌病變、大腸息肉
  • 診斷：↑ IGF-1；OGTT 後 GH 不被抑制至 <1 ng/mL；垂體 MRI
  • 治療：手術；Octreotide（長短效製劑）；多巴胺促效劑（合併 PRL 分泌）；Pegvisomant（GH 受體拮抗劑）；放射線；↑ 死亡率即使治療後
緊急處置
垂體卒中（Pituitary Apoplexy）
  • 機轉：垂體腺瘤（通常為腺瘤）出血或梗塞 → 快速擴張
  • 症狀：劇烈頭痛（突發「霹靂頭痛」）、複視（CN III、IV、VI）、視野缺損、腦下垂體功能低下
  • 治療：立刻高劑量類固醇（Hydrocortisone 100 mg IV）；若有嚴重神經功能障礙或意識改變 → 緊急手術減壓；輕症可保守治療

甲狀腺低下 Hypothyroidism
甲狀腺低下最常見的原因是橋本氏甲狀腺炎（自體免疫）。診斷靠 TSH（最敏感）+ free T4。治療用 Levothyroxine（1.5-1.7 μg/kg/天），每 4-6 週追蹤 TSH 調整。黏液水腫昏迷（Myxedema Coma）是緊急情況——先給類固醇再給甲狀腺素（合併未治療腎上腺功能不足時可能誘發腎上腺危象）。

TBG 影響 Total T4 的因素

  • ↑ TBG（→ ↑ T4）：雌激素（OCP、懷孕）、肝炎、鴉片類、遺傳性
  • ↓ TBG（→ ↓ T4）：雄激素、類固醇、腎病症候群、肝硬化、肢端肥大症、抗癲癇藥、遺傳性

甲狀腺低下病因、診斷與治療

病因

  • 原發性（>90%，↓ free T4 + ↑ TSH）：
  • 甲狀腺腫型：橋本氏甲狀腺炎（Anti-TPO >90% 陽性）；碘缺乏；Lithium；Amiodarone；PD-1/CTLA4 抑制劑誘發（甲亢 → 甲低）
  • 非甲狀腺腫型：手術切除；放射碘（RAI）或放射線治療後；Amiodarone
  • 繼發性（中樞性，↓ free T4 + TSH 低或不恰當正常）：下視丘或垂體功能異常

臨床表現

  • 早期：疲勞、關節痛、肌痛、頭痛、憂鬱、怕冷、體重增加、便秘、月經過多、皮膚乾燥、頭髮粗脆、指甲脆裂、腕隧道症候群、DTR 遲緩（「懸掛式」反射）、舒張壓高血壓、高脂血症
  • 晚期：語速緩慢、聲音嘶啞、眉毛外三分之一脫落、黏液水腫（非凹陷性皮膚增厚）、眶周浮腫、心搏過緩；肋膜 / 心包 / 腹腔積液；動脈粥樣硬化

治療

  • Levothyroxine：1.5-1.7 μg/kg/天；每 4-6 週追蹤 TSH，每 8-12 週調整劑量至 TSH 正常
  • 低起始劑量（0.3-0.5 μg/kg/天）：缺血性心臟病或老年患者
  • 需增加劑量：吸收不良（鐵劑、鈣劑、Cholestyramine、Sucralfate、PPI 干擾吸收）；加速代謝藥物（Phenytoin、Phenobarbital、Rifampin）；懷孕（第 8 週起需增加 ~30%；TSH 目標依孕期調整）
  • 亞臨床甲狀腺低下：TSH <7 或 anti-TPO 陰性 → ~50% 2 年內自行正常化；不一定需要治療；若 TSH >10、有甲狀腺腫、懷孕或不孕症 → 多數建議治療
  • 中樞性甲狀腺低下：監測 free T4（非 TSH）調整劑量
緊急處置
黏液水腫昏迷（Myxedema Coma）
先給 Hydrocortisone，再給甲狀腺素——合併腎上腺功能不足時，若先給甲狀腺素可能誘發腎上腺危象。
  • 表現：低體溫、低血壓、換氣不足、意識改變（少數真正昏迷）、低鈉血症、低血糖；常由感染或重大心肺 / 神經疾病誘發
  • 治療：
  • 支持性治療最重要（藥物代謝減慢可能導致昏迷，需謹慎用藥）
  • 第一步：Hydrocortisone 100 mg IV（先給，保護腎上腺）
  • 第二步：T4 負荷劑量 200-400 μg IV → 之後 50-100 μg IV qd
  • 若血流動力學不穩定 + 心搏過緩 + 低體溫（罕見情況）：也可加 T3 5-20 μg IV q8h（T3 更易誘發心律不整，謹慎使用）
數值判讀
病態甲狀腺正常症候群（Sick Euthyroid）
  • 嚴重非甲狀腺疾病引起的甲狀腺功能異常 → 急性住院期間只在高度懷疑甲狀腺疾病時才抽 TFT
  • 輕度疾病：↓ T4 → T3 轉換 + ↑ rT3 → ↓ T3
  • 重度疾病：↓ TBG 和白蛋白 + ↑↑ rT3 → ↓↓ T3；↑ T4 降解；中樞性 ↓ TSH → ↓↓ T3 + ↓↓ T4 + ↓ free T4 + ↓ TSH
  • 恢復期：TSH 先升高 → 之後 T4 → T3 恢復（通常 2-4 週）
  • 危重症患者：不建議補充甲狀腺素（無益）；若有甲狀腺低下症狀才考慮；TSH 單項不可靠，需同時測 total T4、free T4 和 T3
甲狀腺亢進 Hyperthyroidism
甲狀腺亢進最常見原因是Graves 病（60-80%）。RAIU 掃描是鑑別病因的關鍵工具（TSH 必須已受抑制）。治療選擇：抗甲狀腺藥物（Methimazole 一線）、放射碘（RAI）、手術。甲狀腺風暴（Thyroid Storm）死亡率 20-30%，β-blocker 是第一步。Amiodarone 相關甲狀腺亢進要區分 Type 1（多結節甲狀腺腫）vs Type 2（破壞性甲狀腺炎），治療不同。
數值判讀
臨床表現與診斷
  • 症狀：坐立不安、震顫、皮膚濕溫、頭髮細軟、心搏過速、AF、體重減輕、大便次數增加、月經不規律、反射亢進、骨質疏鬆、凝視和眼瞼遲滯（交感神經過度活化）
  • 冷漠型甲狀腺亢進（老年）：以嗜睡為主要表現，缺乏典型症狀
  • 血液：↑ free T4 + ↑ total T3；↓ TSH（除 TSH 分泌型腫瘤）；高鈣尿症 ± 高鈣血症；↑ Aϕ；貧血
  • RAIU 掃描（TSH 必須已受抑制時使用）：
  • ↑ 攝取 → Graves 病；毒性多結節性甲狀腺腫；熱結節
  • 無攝取 → 亞急性甲狀腺炎（痛性或無痛性）；外源性甲狀腺素；近期碘負荷；Struma ovarii；Amiodarone
  • 近期 IV 顯影劑或 Amiodarone → 碘阻斷 RAIU；改查自體抗體（TSI / TBII）
用藥指引
Graves 病
  • 女：男 = 5-10:1；多在 40-60 歲診斷
  • 診斷：TSI 或 TBII 陽性（80%）；anti-TPO；甲狀腺 Doppler 超音波呈過度血管化；T3/T4 比值通常 >20
  • 特有表現：瀰漫性無壓痛甲狀腺腫 + 甲狀腺血管雜音；眼病變（50%，眶周水腫、眼瞼退縮、突眼、結膜炎、複視）；脛前黏液水腫（3%）

治療選擇

  • β-blocker：所有患者症狀控制（Propranolol 也 ↓ T4 → T3 轉換）；先評估 CHF 病史
  • 抗甲狀腺藥物：Methimazole 一線（60% 停藥後復發）；副作用：搔癢、皮疹、關節痛、發燒、噁心、粒細胞缺乏症（0.5%）；PTU 二線（肝毒性 / TID 給藥 / 較慢起效）；懷孕第一孕期優先用 PTU（Methimazole 致畸胎風險）；用藥前 + 追蹤期間：LFT、WBC、TSH
  • 放射碘（RAI）：門診執行；有心血管疾病或老年 → RAI 前先用抗甲狀腺藥物預處理（停藥 3 天後再做 RAI）；>75% 治療後成甲狀腺低下
  • 手術：阻塞性甲狀腺腫或眼病變時優先考慮；副作用：甲狀旁腺功能低下、喉返神經損傷
  • 眼病變：RAI 後可能惡化 → 高危患者預防性 Prednisone；治療：Selenium、類固醇、Teprotumumab（IGF-1R 抑制劑）、放射線、手術減壓

甲狀腺風暴（Thyroid Storm）
死亡率 20-30%；治療順序：β-blocker → PTU / Methimazole → Iodide（PTU 後 >1h）→ 類固醇。
  • 診斷：無統一標準；甲狀腺毒症症狀（可能含收縮壓高、脈壓差大、心搏過速、發燒、譫妄）；Burch-Wartofsky 量表評估可能性；非常罕見，出現前排除其他診斷
  • 治療（順序很重要）：
  • 1. β-blocker（先給）：Propranolol（給藥前先評估 CHF 病史）；↓ 心搏過速 + ↓ T4 → T3 轉換
  • 2. PTU 或 Methimazole：抑制新的甲狀腺素合成
  • 3. Iodide（PTU 後至少 1 小時再給）：Iopanoic acid 或 Iodide → Wolff-Chaikoff 效應（抑制甲狀腺素釋放）；必須先給 PTU 避免 iodide 被用來合成更多 T4
  • 4. 類固醇：↓ T4 → T3 轉換

庫欣症候群 Cushing's Syndrome
庫欣症候群（Cushing's Syndrome）= 皮質醇過量；庫欣病（Cushing's Disease）= 垂體 ACTH 腺瘤引起的庫欣症候群。最常見原因是醫源性（外源性類固醇）。診斷分兩步：先確認皮質醇過量（篩查）→ 再找病因定位（ACTH 依賴 vs 非依賴）。住院患者急性疾病本身可引起高皮質醇血症，診斷應在門診進行。

病因與臨床表現

病因

  • 醫源性（最常見）：外源性類固醇或 Megestrol
  • ACTH 依賴性：
  • 庫欣病（非醫源性中 60-70%）：垂體 ACTH 分泌腺瘤（微腺瘤 > 大腺瘤）
  • 異位 ACTH 分泌（10-15%）：SCLC、類癌腫瘤、胰島細胞瘤、甲狀腺髓樣癌、嗜鉻細胞瘤
  • ACTH 非依賴性：
  • 腎上腺腫瘤（10-15%）：腎上腺腺瘤或（罕見）腎上腺癌

臨床表現（依特異性排序）

  • 非特異性（也見於其他病因）：葡萄糖不耐受 / DM、高血壓、肥胖、月經不規律、骨質疏鬆
  • 較特異性：向心性肥胖 + 四肢消瘦、頸背部脂肪墊（水牛背）、自發性瘀青
  • 高度特異性：近端肌無力、月亮臉、面部潮紅、寬紫色妊娠紋（>1 cm）
  • 其他：憂鬱 / 失眠 / 精神症狀 / 認知障礙；低鉀血症；痤瘡；多毛症；↑ ACTH → 色素沉著；真菌性皮膚感染；腎結石；多尿
判斷框架
診斷流程

第一步：確認皮質醇過量（門診）

  • 隔夜 1 mg DST（低劑量地塞米松抑制試驗）：晚上 11 點給 1 mg → 早上 8 點測血清皮質醇；正常 = <1.8 μg/dL（若無法抑制 → 疑似庫欣）
  • 24 小時尿游離皮質醇（UFC）× 2：正常範圍以上異常；>4× ULN 幾乎確診
  • 晚上 11 點唾液皮質醇 × 2：異常 = 數值升高（正常時皮質醇應在夜間最低）
  • 住院患者：急性疾病本身可引起高皮質醇血症，結果無法解讀 → 診斷應在門診進行

第二步：定位病因

  • 血清 ACTH：↑ 或正常 = ACTH 依賴性（垂體或異位）；↓ = ACTH 非依賴性（腎上腺腫瘤）
  • ACTH 依賴性 → 高劑量 DST（8 mg 隔夜）：早上 9 點皮質醇 <50% 基礎值 → 庫欣病（垂體腺瘤可被高劑量抑制）；無抑制 → 異位 ACTH
  • CRH 刺激試驗：1 μg/kg IV → ACTH ↑ >35% 或皮質醇 ↑ >20% → 庫欣病
  • 影像：垂體 MRI（垂體腺瘤）；腎上腺 CT（腎上腺腫瘤）；胸腹 CT（異位 ACTH）
  • 雙側下岩竇靜脈採樣（BIPSS）：中樞 vs 周邊 ACTH 比值；基礎 ≥2 或 CRH 後 ≥3 → 垂體來源；陰性 → 繼續尋找異位 ACTH
用藥指引
治療
  • 手術（首選）：垂體腺瘤切除（經蝶竇手術，TSS）；腎上腺腫瘤切除；異位 ACTH 分泌腫瘤切除；雙側腎上腺切除（無法控制 ACTH 來源時）
  • 藥物（橋接至手術或手術禁忌時）：Ketoconazole、Metyrapone、Osilodrostat（↓ 皮質醇合成）；Cabergoline（垂體腺瘤）；Pasireotide（SSA，適用庫欣病）；Mifepristone（阻斷皮質醇在 GR 的作用）；Mitotane（腎上腺癌）
  • 放射線：垂體放射線治療，但起效慢（6 個月至 2 年），且有全垂體功能低下風險
  • 術後皮質醇替代：TSS 後需皮質醇替代 6-36 個月（HPA 軸恢復）；醫療或手術腎上腺切除後 → 終生補充類固醇 + 礦皮質醇
  • 快速恢復 ACTH 刺激試驗（12 個月內通過）是復發的預測因子
原發性醛固酮增多症 Hyperaldosteronism
原發性醛固酮增多症（PA）是頑固性高血壓最常見的可矯正原因——佔所有高血壓的 5-10%，頑固性高血壓超過 10%。經典三聯症：高血壓 + 低鉀血症 + 代謝性鹼中毒，但低鉀血症常常不出現。篩查：血漿醛固酮 / 腎素比值（PAC/PRA >20）。腺瘤（Conn 症候群）→ 手術；雙側增生 → Spironolactone 或 Eplerenone。
機轉說明
病因分類

原發性（腎上腺疾病，腎素非依賴性）

  • 雙側腎上腺增生（60-70%）
  • 腎上腺腺瘤（Conn 症候群，30-40%）
  • 腎上腺皮質癌（罕見）
  • 糖皮質醇可矯正性醛固酮增多症（GRA）：ACTH 依賴性啟動子重排

繼發性（腎外因素，腎素依賴性）

  • 原發性腎素分泌腫瘤：極罕見
  • 繼發性腎素升高：腎血管疾病（RAS、惡性高血壓）；有效動脈容量降低的水腫狀態（CHF、肝硬化、腎病症候群）；低血容量、利尿劑；T2DM；Bartter 症候群（↑ 腎素，類似 Loop 利尿劑效果）；Gitelman 症候群（類似 Thiazide 效果）

非醛固酮性礦皮質醇過量（模擬原發性醛固酮增多症）

  • 11β-HSD2 缺乏 → 皮質醇無法失活 → 與礦皮質醇受體結合
  • 甘草（甘草酸抑制 11β-HSD2）；極度高皮質醇血症（壓倒 11β-HSD2）；外源性礦皮質醇
  • Liddle 症候群（遠端腎小管 Na 通道持續活化 / 過度表達）
判斷框架
臨床表現與診斷流程

臨床表現

  • 輕至中度高血壓（佔所有高血壓 5-10%，頑固性 >10%）
  • 頭痛、肌肉無力、多尿、多渴；無周邊水腫（因為 Na 重吸收後出現「逃脫」現象）
  • 低鉀血症（經典但常缺如）；代謝性鹼中毒；惡性高血壓少見

篩查適應症

  • 高血壓 + 低鉀血症；腎上腺結節；頑固性 / 早發性高血壓

診斷步驟

  • 篩查：早上 8 點血漿醛固酮（PAC）和腎素（PRA）；PAC >15-20 ng/dL + PAC/PRA >20 → 高度懷疑；停 Spironolactone / Eplerenone 6 週；停藥順序注意：ACEI/ARB、利尿劑、CCB 會 ↑ 腎素 → ↓ PAC/PRA 比值；β-blocker 可能 ↑ PAC/PRA；診斷期間改用 α-blocker 控制血壓
  • 確認試驗：鈉負荷抑制試驗（口服鹽 + KCl × 3 天 → 24h 尿醛固酮 >12 μg/天 = 陽性）；或 IV NS 2L/4h → 血漿醛固酮 >5 ng/dL = 陽性
  • 影像：腎上腺 CT（排除大型腫瘤和癌症）
  • 腎上腺靜脈採血（AVS）：即使 CT 見到結節也需做，排除另一側腎上腺增生；比較兩側醛固酮 / 皮質醇比值（>4 倍偏側化 = 腺瘤）
用藥指引
治療
  • 腺瘤（Conn 症候群）→ 腎上腺切除術 vs Spironolactone / Eplerenone 藥物治療
  • 雙側增生 → Spironolactone 或 Eplerenone
  • GRA（糖皮質醇可矯正性）→ 低劑量類固醇（抑制 ACTH）± Spironolactone
  • 腎上腺癌 → 手術

藥物選擇

  • Spironolactone：礦皮質醇受體拮抗劑；副作用：男性女乳、性欲下降、月經不規律（抗雄激素效應）
  • Eplerenone：選擇性礦皮質醇受體拮抗劑；副作用較少（無抗雄激素效應）；較貴

腎上腺功能不足 Adrenal Insufficiency
腎上腺功能不足分為原發性（Addison 病，腎上腺本身病變）和繼發性（ACTH 分泌不足）。最常見的繼發性原因是長期類固醇使用後突然停藥。原發性特有：高鉀血症、色素沉著、鹽分渴望。診斷靠 Cosyntropin 刺激試驗。腎上腺危象是緊急情況——不要因為等診斷而延誤給 Hydrocortisone。

病因

原發性（腎上腺皮質病變 = Addison 病）

  • 自體免疫：孤立性或合併 APS（最常見）
  • 感染：TB、CMV、組織漿菌病、副球孢子菌病
  • 血管性：出血（多在敗血症背景下）、腎上腺靜脈血栓、HIT、外傷
  • 轉移性癌症：需破壞 >90% 腎上腺才引起功能不足
  • 沉積性疾病：血色素沉著症、澱粉樣變、Sarcoidosis
  • 藥物：Azole 抗真菌藥；Etomidate（即使單次使用）；Rifampin；抗癲癇藥

繼發性（ACTH 分泌不足，醛固酮正常）

  • 任何原因的原發性或繼發性腦下垂體功能低下
  • 最常見：長期類固醇治療（使用 >3-4 週後即可，即使劑量 <10 mg/天；避免方法：緩慢減量數週）；Megestrol

僅原發性（醛固酮缺乏 + ↑ ACTH）

  • 高鉀血症（醛固酮缺乏）
  • 明顯直立性低血壓（容積缺乏）
  • 鹽分渴望
  • 色素沉著（ACTH / MSH 升高 → 皮膚皺褶、黏膜、壓力點、乳頭色素增加）

僅繼發性

  • 其他腦下垂體功能低下的表現（見腦下垂體疾病 block）

其他實驗室異常

  • 低血糖；嗜酸性球增多；淋巴球增多；± 嗜中性球減少；ACTH：原發性 ↑、繼發性 ↓ 或正常偏低
判斷框架
診斷
  • Cosyntropin（ACTH）刺激試驗：250 μg ACTH IV → 正常 = 60 分鐘後皮質醇 ≥15 μg/dL
  • 原發性：腎上腺病變 → 無法給出足夠皮質醇 → 異常
  • 慢性繼發性：腎上腺萎縮 → 無法反應 → 異常

腎上腺危象（Adrenal Crisis）
不要因為等待診斷試驗而延誤給 Hydrocortisone——臨床高度懷疑即立刻治療。
  • 誘發因素：既有腎上腺功能不足 + 嚴重感染或 GI 疾病；雙側腎上腺出血或梗塞；垂體梗塞；未先給類固醇就補充甲狀腺素（合併未治療的腎上腺功能不足）
  • 表現：休克 + 食欲不振、噁心嘔吐、腹痛、虛弱、疲勞、意識改變、發燒
  • 實驗室：低鈉血症；高鉀血症（原發性）
  • 治療：Hydrocortisone 100 mg IV → 之後 50 mg IV q6h + 積極 IVF（NS）；不要等待診斷試驗

危重症患者的腎上腺功能不足

  • 低皮質醇結合蛋白 → 診斷困難；大多數休克患者腎上腺功能不足少見（除非腎上腺梗塞 / 出血、Waterhouse-Friderichsen、CNS 或垂體出血）
  • 高度懷疑時可測（但結果需結合臨床解讀，有假陰性和假陽性）
  • 若高度懷疑 → 早給類固醇：Hydrocortisone 100 mg IV → 50 mg IV q6h
嗜鉻細胞瘤 / 副神經節瘤 Pheochromocytoma & Paraganglioma
嗜鉻細胞瘤（腎上腺髓質）和副神經節瘤（腎上腺外）是神經內分泌腫瘤，過量分泌腎上腺素和正腎上腺素。典型三聯症：頭痛、心悸、出汗（3Ps），但只有 50% 有陣發性高血壓。高達 40% 有遺傳性基因突變——所有患者建議基因檢測。治療前必須先做 α-blockade 再加 β-blockade（順序不能顛倒）。

臨床表現與診斷

臨床表現

  • 神經內分泌腫瘤 → 不適當、陣發性釋放腎上腺素、正腎上腺素，罕見多巴胺

  • 陣發誘因：β-blocker 使用（未先 α-blockade）；腹部操作（按壓腫瘤）

緊急處置
術前準備與手術
必須先 α-blockade，再加 β-blockade——若先給 β-blocker，可能因 α 受體未阻斷 → 嚴重高血壓危象。
  • 第一步：α-blockade（術前 10-14 天）：Phenoxybenzamine（非選擇性、不可逆）或 Doxazosin（選擇性 α1）
  • 第二步：加 β-blockade（α 充分後才加）：Propranolol 控制心搏過速
  • 術前擴充血容量：極重要——腫瘤切除後 → 兒茶酚胺驟降 → 嚴重低血壓；術前給予充分 IVF / 高鈉飲食
  • 手術：腹腔鏡腎上腺切除術（首選）；術中血壓波動需麻醉科密切配合
  • 術後：監測低血糖（腫瘤切除 → 胰島素分泌反彈）；追蹤生化指標排除復發或轉移
腎上腺結節 Adrenal Incidentaloma
腎上腺結節在腹部 CT 的盛行率約 4%，隨年齡增加。評估重點：功能性評估（排除激素過量）+ 惡性評估（影像學特徵）。所有患者都必須排除嗜鉻細胞瘤（未處置的 Pheo 手術風險極高）。有癌症病史者約 50% 的腎上腺結節是惡性轉移。

門診追蹤目標
  • HbA1c：每 3-6 個月追蹤；目標 <7%（多數患者）；<6.5%（低低血糖風險者）；≤8%（嚴重低血糖史、老年、多重共病）；懷孕目標 <6.0-6.5%

住院血糖管理

  • 目標：避免低血糖和極端高血糖（>180 mg/dL）；ICU 目標 140-180 mg/dL
  • T1DM：絕對不停基礎胰島素（可引發 DKA）；若使用胰島素幫浦 → 找內分泌科協助
  • T2DM：住院期間通常停口服降糖藥（避免低血糖或藥物交互作用，除非短暫住院、控制良好、無 IV 顯影劑計畫、正常飲食）

DKA / HHS
DKA 多見於 T1DM（胰島素缺乏 → 高血糖 + 酮症 + 陰離子間隙代謝性酸中毒）；HHS 多見於 T2DM 老年患者（極端高血糖 + 高滲透壓 + 意識改變，無酮症）。DKA 治療三大柱：積極補液（LR 優於 NS）+ 胰島素點滴 + 電解質補充（K 尤其重要）。K <3.3 時先補鉀再給胰島素。HHS 補液量需求更大（平均失水 8-10L）。
機轉說明
DKA 誘因與病生理

誘因（The I's）

  • 胰島素缺乏（Insulin deficiency）：忘記注射或劑量不足
  • 醫源性（Iatrogenesis）：類固醇；SGLT2i（可在血糖正常時引發 DKA）
  • 感染（Infection）：肺炎、UTI
  • 發炎（Inflammation）：胰臟炎、膽囊炎
  • 缺血 / 梗塞（Ischemia / Infarction）：心肌梗塞、腦梗塞、腸道缺血
  • 中毒（Intoxication）：酒精、藥物

病生理

  • 多見 T1DM（也見於酮症傾向 T2DM）；↑ 升糖素 + ↓ 胰島素
  • 高血糖：↑ 糖質新生 + ↑ 肝醣分解 + ↓ 細胞葡萄糖攝取
  • 酮症：胰島素缺乏 → 脂肪酸動員和氧化 + ↑ 肝臟酮體生成 + ↓ 酮體清除
  • SGLT2i 相關 DKA：血糖可正常（Euglycemic DKA）→ 容易漏診
數值判讀
DKA 臨床表現與診斷

臨床表現

  • 多尿、多渴、脫水 → ↑ 心率、低血壓、黏膜乾燥、皮膚彈性差
  • 噁心嘔吐、腹痛、腸阻塞（DKA 本身或腹腔內病程）
  • Kussmaul 呼吸（深大）補償代謝性酸中毒；丙酮氣味
  • 意識改變 → 嗜睡、昏迷；死亡率 ~1%（即使在三級醫療中心）

診斷

  • ↑ 陰離子間隙代謝性酸中毒：pH <7.3 + HCO₃ <18；之後可能發展非 AG 酸中毒（酮體隨尿液流失 = HCO₃ 等效物 + NS 補液含 Cl⁻）
  • 酮症：尿液和血清酮體陽性；主要酮體為 β-OH-butyrate（但分析測 acetoacetate）；正常禁食也可能尿酮陽性
  • 血糖：通常 >250 mg/dL（SGLT2i 相關可正常）；↑ BUN 和 Cr
  • 低鈉血症：校正 Na = 測量 Na + [2.4 × (血糖 − 100)/100]
  • 鉀：血清 K 可 ↓ 或 ↑（但即使血清 K 升高，總體 K 通常耗竭）；↓ 總體磷
  • 白血球增多 + ↑ 澱粉酶（即使無胰臟炎）

高滲透壓高血糖狀態（HHS）
  • 定義：極端高血糖（無酮症酸中毒）+ 高滲透壓 + 意識改變；多見 T2DM 老年患者
  • 誘因：同 DKA，但也包括脫水和腎衰竭
  • 病生理：高血糖 → 滲透性利尿 → 容積缺乏 → 腎前性氮血症 → ↑ 血糖惡化循環
  • 診斷：血糖通常 >600 mg/dL；↑ 測量血清滲透壓 >320 mOsm/L；有效滲透壓 = 2 × Na + 血糖/18；pH >7.3（無酮症）；通常 ↑ BUN 和 Cr；[Na] 依血糖和脫水程度而定
  • 預後：~15% 死亡率（死於誘發因素）

治療

  • 排除誘因
  • 積極補液：先 NS → 之後 ½NS；平均失水 8-10L；優先補液再給胰島素（補液本身可大幅降低血糖）
  • 胰島素：10U IV 推注 → 0.05-0.1 U/kg/h；目標血糖 ~250 mg/dL 直到患者清醒
  • HHS 與 DKA 可同時存在（尤其酮症傾向 T2DM）；處置以 HHS 為主，加上胰島素並監測 AG
低血糖 Hypoglycemia
低血糖定義為血糖 <~55 mg/dL。糖尿病患者最常見原因是胰島素或磺醯脲類過量。非糖尿病患者低血糖要排除胰島素瘤——72 小時禁食試驗是金標準診斷工具。β-blocker 可掩蓋腎上腺素症狀（心悸、顫抖）但不掩蓋出汗。

臨床表現與病因

臨床表現（血糖 <~55 mg/dL）

  • 神經糖質缺乏（CNS）：頭痛、視覺改變、意識改變、無力、癲癇發作、意識喪失
  • 腎上腺素症狀：出汗、心悸、顫抖（β-blocker 可掩蓋心悸和顫抖，但不掩蓋出汗）

病因

  • 糖尿病患者：胰島素或口服降糖藥過量；跳餐；腎衰竭（↓ 胰島素和磺醯脲類清除）；β-blocker 掩蓋腎上腺素症狀

  • ↑ 胰島素：外源性胰島素；磺醯脲類；胰島素瘤；抗胰島素抗體
  • ↓ 葡萄糖生成：垂體功能低下；腎上腺功能不足；升糖素缺乏；肝衰竭；腎衰竭；CHF；酒精性；敗血症；嚴重營養不良

非糖尿病患者的評估

  • 臨床病況不佳者：↓ 反覆低血糖措施；BUN、Cr、LFT、TFT、前白蛋白；IGF-I/IGF-II 比值（必要時）

測量注意事項

  • 脂蛋白 = 脂質核心（膽固醇酯 + 三酸甘油酯）+ 磷脂表面 + 蛋白質；包含：Chylomicrons、VLDL、IDL、LDL、HDL、Lp(a)

  • LDL-C 在 ACS 後 <24 小時穩定，之後下降，約 6 週後恢復正常 → ACS 後盡早測
  • 身體檢查線索：肌腱黃色瘤（跟腱）→ LDL >300；皮下黃色瘤（伸肌表面）→ TG >1500；眼瞼黃色素瘤（眼瞼黃色條紋）

肥胖 Obesity & Weight Management
肥胖（BMI ≥30）是慢性疾病，全球盛行率持續上升（美國 >70% 過重）。治療核心是生活型態介入為基礎，再依 BMI 和共病加上藥物或代謝手術。GLP-1 RA 和 GLP-1/GIP 雙重促效劑（Tirzepatide）是目前最有效的藥物，體重減輕 12-20%，且有 CV 和腎臟保護效益。術前使用 GLP-1 RA 需注意麻醉前停藥（↑ 胃排空延遲 → 誤吸風險）。

臨床評估——篩查繼發性原因和共病

  • 促進體重增加的藥物：部分精神科用藥、抗癲癇藥、胰島素、磺醯脲類、類固醇
  • 內分泌疾病：庫欣症候群、甲狀腺低下、多囊性卵巢症候群（PCOS）
  • 精神疾病：憂鬱症、暴食症
  • 辨識體重相關併發症：HTN、血脂異常、T2DM、CKD、OSA
判斷框架
生活型態與行為介入
  • 飲食：目標每天 ↓ 500-750 kcal 熱量赤字；考慮轉介營養師
  • 運動：每週 ≥150 分鐘中等強度運動（維持體重需 ≥200-300 分鐘 / 週）
  • 行為介入：自我監測飲食和體重；目標設定；刺激控制；壓力管理
  • 生活型態介入是所有藥物和手術治療的基礎，必須同步進行
用藥指引
藥物治療

適應症

  • BMI ≥30；或 BMI ≥27-<30 + ≥1 個體重相關共病；必須同時進行生活型態介入

Incretin 類藥物（最有效）

藥物機轉體重減輕額外效益
Semaglutide（GLP-1 RA） GLP-1 促效劑 → ↑ 飽足感、↓ 食慾、改善胰島素敏感性 ~15% ↓ MACE 和 HF 事件（有 CVD 無 DM 者）；↓ CKD 進展（T2DM）；↓ MASH
Tirzepatide（GLP-1/GIP 雙重 RA） GLP-1 + GIP 雙重促效 → 更強的體重減輕和血糖控制 ~20% ↓ HbA1c ~2%（DM）；↓ SBP ~5 mmHg；↓ TG ~20%；↓ hsCRP ~35%；↓ HF 事件（HFpEF）；改善 PAD 步行距離
  • GLP-1 RA 副作用：噁心、嘔吐、便秘、腹脹（通常輕微，在起始或加量期間）；小餐量和劑量滴定可減輕
  • GLP-1 RA 禁忌：懷孕；個人或家族甲狀腺髓樣癌或 MEN2a 病史；胰臟炎病史
  • 術前停藥：每週注射劑型 → 術前停 7 天；每日注射劑型 → 術前停 24 小時（↑ 胃排空延遲 → 麻醉誤吸風險）

非 Incretin 類藥物（效果較弱，<5% 體重減輕）

  • Orlistat：胰脂肪酶抑制劑 → ↓ TG 水解和游離脂肪酸吸收；副作用：脂肪瀉
  • Naltrexone / Bupropion：改善中樞飲食行為調控；副作用：噁心、頭痛；禁忌：懷孕、未控制 HTN、癲癇病史、長期鴉片類使用
  • Phentermine / Topiramate：中樞 ↓ 食慾；副作用：口乾、感覺異常、失眠；禁忌：懷孕、已確立 CVD、青光眼、未治療甲狀腺亢進
  • 非 Incretin 類藥物僅作為輔助工具，無法影響臨床結果（死亡率、CV 事件）
用藥指引
代謝手術（Metabolic Surgery）

適應症

  • BMI ≥35；或 BMI 30-<35 + 代謝疾病（DM 或 MASH）；亞洲族群：BMI ≥27.5 + 其他適應症可考慮

術式與效果

術式1 年體重減輕15 年體重減輕
腹腔鏡袖狀胃切除術（LSG）~25%~20%
Roux-en-Y 胃繞道術（RYGB）~35%~30%
  • LSG / RYGB：↑ HTN、DM、血脂異常、MASH 的控制 / 緩解 / 預防率；優於一般治療
  • 圍術期死亡率：低（~0.2%）
  • 併發症：吻合口漏 / 狹窄；邊緣潰瘍 / 內疝（RYGB 2-5%）；微量營養素缺乏（鐵）；骨質流失和骨折風險 ↑；長期體重回升（約 20-40% 患者在長期追蹤中回升 ≥25% 已減輕體重）
