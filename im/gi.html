食道與胃疾病 Esophageal & Gastric Disorders
食道疾病的評估從症狀型態開始：固體和液體都難以吞嚥 → 神經肌肉問題；只有固體困難 → 結構性阻塞。GERD 的診斷靠臨床，PPI 試驗有效即可確診，有警示症狀才需內視鏡。消化性潰瘍先確認 H. pylori，有即根除。
機轉說明
吞嚥困難（Dysphagia）

結構性（固體 > 液體）

  • 口咽：Zenker's 憩室（老年、吸入、影像診斷）；惡性腫瘤；近端狹窄 / 蹼 / 環；感染；甲狀腺腫大；骨刺；放射線損傷
  • 食道：Schatzki 環（近 GE 接口，間歇性吞嚥困難 + 食物嵌頓）；Plummer-Vinson 症候群（蹼 + 缺鐵性貧血）；消化性或放射線狹窄；腫瘤；外在壓迫
  • 感染性食道炎（痛嚥 > 難嚥）：免疫抑制者 → Candida、HSV、CMV
  • 藥物性食道炎：NSAIDs、KCl、Bisphosphonates、Doxycycline
  • 嗜酸性球性食道炎（EoE）：年輕至中年男性；症狀 = 食道功能障礙（吞嚥困難、食物嵌頓）+ 切片 >15 eos/HPF；一線：PPI → 3D 策略（飲食去除、藥物〔吞嚥吸入型類固醇或 Dupilumab〕、擴張）

神經肌肉性（固體 + 液體）

  • 口咽：中風、ALS、肌病變、CNS 腫瘤
  • 食道運動障礙（高解析度測壓診斷）：
  • 賁門失弛緩症（Achalasia）：同步收縮 ↓ + LES 鬆弛 ↓；鋇劑 = 擴張食道 + 遠端「鳥喙狀」；多為特發性（或 Chagas）；治療：氣球擴張 = 腹腔鏡 Heller 肌切開術；POEM；CCB / 硝酸鹽 / PDE5 抑制劑；Botox（不能手術者）
  • 遠端食道痙攣；過度收縮食道；無效食道動力（硬皮病、DM、甲狀腺低下）
用藥指引
胃食道逆流（GERD）
  • 機轉：食道酸暴露增加，源於 LES 短暫鬆弛 ↑；肥胖、孕婦、食道裂孔疝氣加重；罕見由高分泌狀態引起（ZE 症候群）
  • 誘發因素：仰臥、高脂食物、咖啡因、酒精、菸；CCB、懷孕、肥胖
  • 食道症狀：火燒心、非典型胸痛、逆流、酸味、吞嚥困難
  • 食道外症狀：乾咳（常見難治氣喘的原因）、喉炎、齒齦侵蝕

診斷與治療

  • 臨床診斷（PPI 試驗）：典型症狀 + 低劑量 PPI 有效即可確診
  • 需做內視鏡（EGD）：PPI 無效；或有警示症狀（吞嚥困難、嘔吐、體重減輕、貧血、年齡 >60 歲）
  • 治療：生活型態（避免誘發因素、減重、睡前 2 小時不進食）；PPI（低劑量開始，最高 40 mg bid）；難治性 → pH 測試確認 → 考慮 Vonoprazan
  • Barrett 食道：鱗狀上皮被化生柱狀上皮取代；每年 0.1-0.3% 腺癌風險（異生程度越高風險越高）；篩查對象：男性 + 慢性 GERD + ≥3 個危險因子（>50 歲、白人、肥胖、抽菸、家族史）；PPI + 每 3-5 年內視鏡追蹤（無異生）；高度異生 → 內視鏡消融
用藥指引
消化性潰瘍（PUD）

病因

  • H. pylori：約 80% 十二指腸潰瘍（DU）和 30-40% 胃潰瘍（GU）；全球約 50% 人口帶菌，但只有 10-15% 發展為 PUD
  • ASA / NSAID：↓ 前列腺素合成 → 黏膜損傷；5-6 倍 ↑ GIB 風險；造成大多數非 H. pylori 相關潰瘍
  • 其他：抽菸、酒精、胃癌 / 淋巴瘤、Crohn's、病毒感染（CMV / HSV）；壓力性潰瘍（ICU + 凝血障礙、機械通氣）；Zollinger-Ellison 症候群（罕見）
  • 症狀：上腹部鈍痛（DU 進食後緩解；GU 進食後加重）；併發症：UGIB、穿孔、胃出口阻塞

H. pylori 檢測

  • 糞便抗原（Stool Ag）、尿素呼氣試驗（UBT）或 EGD + 快速尿素酶試驗（RUT）
  • 注意：抗生素、鉍劑、PPI 會造成假陰性 → 測試前盡量停用
  • 血清學特異性低，僅用於排除低流行區感染

治療

  • H. pylori 陽性 → 根除：四聯療法 14 天：[MNZ + 四環素 + 鉍劑 + PPI] 或複方片（Pylera / Helidac）；根除後 4 週做「根除確認」（停 PPI 1-2 週再測試）
  • H. pylori 陰性 → 胃酸抑制：高劑量 PPI
  • 除 PUD 外，也需「測試並根除」H. pylori 的情況：胃 MALT 淋巴瘤、胃癌切除後、無法解釋的缺鐵性貧血、ITP、未確診的消化不良（<60 歲）、長期 NSAID 使用前
  • ASA / NSAID 使用者的 GI 預防：有 PUD / UGIB 病史 + （P2Y12 或抗凝）或（≥2 危險因子：>60 歲、類固醇、消化不良）→ 加 PPI
消化道出血 Gastrointestinal Bleeding
GIB 分為上消化道（Treitz 韌帶以上）和下消化道。UGIB 最常見原因是 PUD；LGIB 最常見是憩室出血。評估核心：先穩定生命徵象 → 評估嚴重度 → 輸血目標 Hb >7（有 CAD 則 >8）→ 緊急內視鏡。靜脈曲張出血先用 Octreotide + 預防性抗生素再做內視鏡。
緊急處置
初始評估與復甦
  • 嚴重度評估：心跳過速 → 失血 ~10%；直立性低血壓 → ~20%；休克 → >30%
  • 「嚴重」GIB 定義：休克、直立性低血壓、Hct ↓ 6% 或 Hb ↓ 2 g/dL、需要輸血 ≥2 U
  • 復甦：兩條大口徑（≥18G）IV；NS 或 LR 補液達到正常生命徵象 + 尿量 + 意識
  • 輸血：限制性 Hb 目標 >7（門脈高壓尤其重要）；CAD 目標 >8；先定血型，緊急用 O 型陰性
  • 逆轉凝血障礙：生命威脅性出血或 INR 明顯升高 → 考慮 PCC 或 FFP；血小板 >50k；尿毒症 → dDAVP；抗凝藥 → 各自的拮抗劑
  • 抽血：BUN/Cr（UGIB 比 >36 因腸道血液再吸收）；Hct（急性出血早期可能正常）；PT/PTT；LFT
  • 門診治療（低風險）標準：SBP ≥110、HR <100、Hb ≥13（男）或 ≥12（女）、BUN <18、無黑便 / 暈厥 / 心衰竭 / 肝病

靜脈曲張出血（門脈高壓）
  • 藥物（懷疑靜脈曲張 → 先給，不等 EGD）：
  • Octreotide（Somatostatin 類似物）2-5 天；20% 肝硬化 + GIB 入院時有感染，~50% 住院期間發展感染

  • 內視鏡（EGD 在 12 小時內）：
  • 食道靜脈曲張 → 內視鏡靜脈曲張結紮術（EVL）>90% 成功率

  • 胃靜脈曲張 → 血管攝影 + 線圈栓塞；或氰丙烯酸酯注射（若有）；頑固性 → TIPS 或 BRTO

腹瀉 Diarrhea
腹瀉評估先分急性（<4 週）vs 慢性（>4 週），急性以感染為主；慢性依滲透壓間隙、脂肪含量、發炎指標分類。C. diff 是住院患者最常見的院內感染——有症狀才治療，不要常規「test of cure」。慢性腹瀉要找代謝、藥物、功能性原因。

治療原則

  • 無警示症狀（輕度）：口服補液（Gatorade 或口服電解質溶液）+ Loperamide + Bismuth；不用抗生素
  • 旅遊者腹瀉：Azithromycin 1g × 1 天（亞洲 FQ 抗藥性高）；Rifaximin；疑似原蟲 → MNZ 或 Nitazoxanide
  • 非 C. diff 發炎性腹瀉（重度 / 住院 / 高危）：Azithromycin 1g × 1 天（發燒 / 血便首選）或 FQ × 3-5 天
  • 疑似 E. coli O157:H7（接觸史、血便）：避免抗生素 → 可能 ↑ HUS 風險
注意！
困難梭狀芽孢桿菌感染（CDI）
  • 機轉：攝入孢子 → 抗生素改變腸道菌叢 → 定殖 → 釋放毒素 A/B → 結腸黏膜壞死和發炎 → 偽膜性結腸炎
  • 危險因子：任何抗生素（尤其 β-lactam、Clindamycin、FQ）在使用中或停藥後 10 週內；老年；免疫抑制；IBD
  • 約 1/3 為社區感染（無近期抗生素使用）

臨床表現與診斷

  • 非嚴重：WBC <15k、Cr <1.5
  • 嚴重：>10 次/天、WBC >15k、Cr >1.5
  • 爆發性（2-3%）：低血壓 / ICU、腸阻塞、毒性巨結腸（結腸擴張 ≥6 cm + 全身毒性）
  • 診斷：只在有症狀時檢測（腹瀉、結腸炎）；送液態糞便；毒素免疫分析（高特異性）+ GDH（高敏感性）；PCR（敏感性高但可能測到無症狀定殖，若毒素陰性 PCR 陽性需謹慎解讀）
  • 若懷疑爆發性 → CT 腹腔；若診斷不確定 → 軟式乙狀結腸鏡

治療

  • 盡快停用誘因抗生素；停止止瀉藥和 Cholestyramine（會結合 Vancomycin）
  • 非嚴重或嚴重：Fidaxomicin 200 mg bid × 10 天（↓ 復發率，首選但較貴）或 Vancomycin 125 mg PO qid × 10 天
  • 爆發性：Vancomycin 500 mg PO qid + MNZ 500 mg IV q8h ± Vancomycin 灌腸；考慮 FMT；手術會診（次全結腸切除術 / 分流迴腸造口術）
  • 惡化跡象（腸阻塞、↑ WBC、↑ Lactate、休克、毒性巨結腸）→ 緊急外科會診

復發（停藥後 15-30% 在 2 週內復發）

  • 第一次復發：Fidaxomicin 或 Vancomycin × 10-14 天；考慮加 Bezlotoxumab 10 mg/kg IV × 1（抗 B 毒素單抗）↓ 復發（CHF 患者慎用）
  • 後續復發：Fidaxomicin 或 Vancomycin 遞減療程；ID 會診；≥3 次 CDI → 糞菌移植（FMT）
  • 預防：Vancomycin 125 mg PO qd 在需要其他抗生素且有嚴重或復發 CDI 病史時考慮

消化不良：慢性胰腺炎或胰臟癌（↓ 胰酶）；膽汁酸 ↓（肝硬化、膽汁淤積、迴腸切除）
發炎性 ↑ 糞便 WBC / Calprotectin；血便；發燒；腹痛 慢性 C. diff；E. histolytica；IBD；放射性腸炎；缺血性結腸炎；腸道腫瘤

顯微鏡性結腸炎

  • 中年女性 + 自體免疫疾病；NSAIDs、SSRIs、PPIs 為常見誘發藥物
  • 大腸鏡正常但切片顯示淋巴球 / 漿細胞浸潤 ± 黏膜下膠原增厚
  • 治療：Budesonide（一線）、止瀉藥、Cholestyramine；頑固性 → 生物製劑
  • 慢性腹瀉評估：大腸鏡（未知原因）± 腹部 CT/MRI；糞便 Calprotectin 有助鑑別 IBD vs IBS
用藥指引
腸道動力障礙與 IBS

腸躁症（IBS）

  • ≥6 個月腹痛 + ≥2 項：排便後緩解、排便頻率改變、糞便形態改變
  • IBS-C（便秘型）/ IBS-D（腹瀉型）/ IBS-M（混合型）
  • IBS-D 治療：Loperamide；Rifaximin；Eluxadoline；膽汁酸螯合劑；Alosetron；Amitriptyline
  • IBS-C 治療：↑ 纖維 + 補充液體；Lubiprostone；Linaclotide；Tenapanor；Tegaserod
  • 認知行為治療；低 FODMAP 飲食；益生菌；腸道解痙藥

胃輕癱（Gastroparesis）

  • 胃排空延遲（無機械性阻塞）；噁心 >90%、嘔吐 >80%
  • 原因：DM（最常見）、術後、病毒後、Parkinson's、鴉片類、CCB、抗膽鹼藥
  • 診斷：核子醫學胃排空閃爍攝影（4 小時後殘存食物 >10%）
  • 治療：促動力藥（Metoclopramide 或 Erythromycin）；止吐藥；必要時鼻空腸管灌食；胃幽門成形術或 Botox

便秘

  • 次發性原因（4 Ms）：機械阻塞（腫瘤、狹窄）；藥物（鴉片類、TCA、抗膽鹼藥、CCB、NSAIDs）；代謝（DM、甲狀腺低下、高鈣、低鉀）；神經肌肉（Parkinson's、Hirschsprung、脊椎損傷）
  • 治療：一線 → ↑ 液體 + 纖維 + 運動；軟便劑（Docusate）；二線 → 纖維素、刺激性瀉藥（Senna、Bisacodyl）、滲透性瀉藥（PEG、Mg）；三線 → Linaclotide、Lubiprostone、Prucalopride（鴉片類 → Methylnaltrexone）
炎症性腸病 Inflammatory Bowel Disease
IBD 分為潰瘍性結腸炎（UC）和克羅恩病（CD）：UC 是從直腸往近端延伸的結腸黏膜層發炎；CD 是跳躍性透壁性發炎，可累及口腔到肛門任何部位。治療目標是誘導緩解 → 維持緩解 → 黏膜癒合。生物製劑前必須排除 TB 和 HBV。
機轉說明
潰瘍性結腸炎（UC）
  • 定義：結腸黏膜層連續性、從直腸向近端延伸的發炎；95% 累及直腸
  • 範圍：直腸炎（30-60%）；左側結腸炎（15-45%）；廣泛性（Pancolitis，15-35%）
  • 症狀：血便（最典型）、腹部絞痛、裡急後重、頻便少量
  • 腸外表現（>25%）：結節性紅斑、壞疽性膿皮病、口腔潰瘍、葡萄膜炎、鞏膜炎、血栓栓塞事件（尤其急性發作時）、AIHA、血清陰性關節炎（最常見）、PSC
  • 診斷：大腸鏡 + 切片（表淺炎症、隱窩膿瘍和架構扭曲）；排除感染（尤其 TB、HBV、CMV、C. diff）再啟動免疫抑制
  • 預後：50% 任何時間點在緩解中；10 年結腸切除率 24%；嚴重急性 UC 死亡率 <2%

併發症

  • 毒性巨結腸（5%）：結腸擴張 ≥6 cm（KUB）+ 結腸蠕動消失 + 全身毒性 + 穿孔風險 → IV 類固醇 + 廣效抗生素；手術若需要
  • CRC 風險：活動性疾病 8 年後每 1-2 年大腸鏡監測；合併 PSC 風險更高
機轉說明
克羅恩病（CD）
  • 定義：跳躍性透壁性發炎，可累及口腔到肛門任何部位
  • 分布：小腸 / 迴腸炎（~25%）；迴結腸（~50%）；結腸（~25%）；孤立性上段 GI 罕見
  • 症狀：腹痛、慢性腹瀉（常無血）、倦怠、體重減輕；N/V 和腹脹（若有阻塞）
  • 腸外表現：同 UC；肛門周圍疾病（肛裂、瘻管、肛周膿瘍）在 24% 患者，且可先於腸道症狀出現
  • 診斷：迴結腸鏡 + 切片（透壁炎症、非乾酪性肉芽腫 <25%、跳躍病灶）+ MR 小腸造影評估小腸範圍

併發症

  • 腸道狹窄：術後腹痛；可引起完全性腸阻塞 → 手術
  • 瘻管：腸腸、直腸陰道、腸膀胱、腸皮膚
  • 膿瘍：發燒、腹部壓痛腫塊、↑ WBC；類固醇可掩蓋症狀 → 高度懷疑；CT 引導引流 + 抗生素
  • 吸收不良：迴腸病變 → 膽汁酸吸收不良 → 膽結石；脂肪酸吸收障礙 → 草酸鈣腎結石；脂溶性維生素（D）缺乏 → 骨質疏鬆

用藥指引
憩室炎（Diverticulitis）
  • 危險因子：↓ 纖維、慢性便秘、肥胖、抽菸、久坐、酒精、NSAIDs、年齡（<40 歲佔 10%；>80 歲佔 50-66%）
  • 分布：90% 在左側（以乙狀結腸為主）；亞洲族群 75-85% 在右側
  • 非複雜性（75%）：微穿孔 → 局部感染；LLQ 腹痛 + 發燒 + ↑ WBC
  • 複雜性（25%）：大穿孔 → 膿瘍、腹膜炎、瘻管（65% 為腸膀胱瘻管）、阻塞

診斷

  • 腹部 CT（含靜脈顯影劑）：憩室 + 腸壁增厚 + 結腸周圍脂肪浸潤 ± 膿瘍 / 瘻管
  • 急性期禁止大腸鏡（穿孔風險）；無近期大腸鏡者 → 症狀緩解後 6-8 週複查排除腫瘤

治療

  • 輕度（門診）：合併症少且可口服 → 口服抗生素（MNZ + FQ 或 Amox/Clav）× 7 天；流質飲食直到臨床改善
  • 抗生素 vs 觀察：輕度憩室炎中抗生素和觀察等待效果相當（但台灣多數仍用抗生素）
  • 重度（住院）：無法口服、需嗎啡止痛或有併發症 → NPO + IVF + 鼻胃管（若腸阻塞）；IV 抗生素（GNR + 厭氧菌覆蓋：Ceftriaxone + MNZ 或 Pip-tazo）
  • 膿瘍 >4 cm：CT 引導或手術引流；引流後 4 天抗生素可能足夠
  • 手術：藥物治療無效 / 無法引流的膿瘍 / 游離穿孔 → 手術
  • 復發風險 10-30%（10 年內第一次發作後）；堅果和種子不增加復發風險（飲食禁忌的迷思）；免疫抑制者手術閾值要更低

大腸息肉與大腸癌篩查

息肉病理分類

  • 腫瘤性（有惡性潛能）：腺瘤（管狀腺瘤、絨毛腺瘤、管狀絨毛腺瘤；異生程度↑惡性潛能↑）；鋸齒狀腺瘤 / 息肉（SSA/SP，有大腸癌間隔性風險）；癌變
  • 非腫瘤性：增生性息肉；幼年型；Peutz-Jeghers（可惡變）；發炎性

大腸癌篩查

  • 平均風險：從 45 歲開始（以往 50 歲）
  • 篩查方式（擇一）：
  • 大腸鏡（金標準）：每 10 年（若正常）
  • 糞便潛血試驗（FOBT/FIT）：每年
  • 糞便 DNA（Cologuard）：每 3 年
  • CT 大腸影像（虛擬大腸鏡）：每 5 年
  • 軟式乙狀結腸鏡：每 5 年 ± 每年 FIT
  • 有一等親 CRC 病史：從 40 歲開始或比最年輕診斷親屬早 10 年；每 5 年
  • 保護因子：體力活動、ASA / NSAIDs、鈣質攝取、HRT、↓ BMI；可能：纖維、維生素 D、魚油、Statin、硒
腸道缺血 Intestinal Ischemia
腸道缺血分為急性腸繫膜缺血（AMI）（小腸、死亡率高）和缺血性結腸炎（大腸、通常較輕）。AMI 的關鍵是高度懷疑——疼痛程度與腹部觸診表現不成比例是典型線索。腸壞死前診斷是唯一能改善存活率的因素，確診就要緊急手術或血管介入。

  • 動脈阻塞：突發劇烈腹痛，疼痛程度與腹部觸診不成比例（早期腹部柔軟）
  • 靜脈阻塞：較隱匿，間歇性疼痛起伏

  • 高度懷疑才能及早診斷 → 腸壞死前診斷是最強的存活預測因子

  • CT 血管攝影（動脈期）：非侵入性首選；靜脈期可診斷 MVT
  • 血管攝影：金標準，且有治療功能（適用於懷疑血管阻塞）

治療

  • IVF + NPO + 優化血流動力學（最小化升壓藥）+ 廣效抗生素 + 抗凝（Heparin ± tPA 用於阻塞性病變）；IV Papaverine（血管擴張劑，用於 NOMI）
  • 有腹膜炎 → 手術：腸切除 ± 血管重建
  • SMA 血栓：經皮血管成形術 + 支架或手術血管重建
  • SMA 栓塞：導管取栓（抽吸 vs 手術）
  • NOMI：治療原發原因（尤其心臟）

缺血性結腸炎（Ischemic Colitis）
  • 機轉：非阻塞性、繼發於全身循環改變或腸繫膜血管局部改變；老年多見；常無明確病因
  • 好發部位：「分水嶺」區域（脾曲和直腸乙狀結腸交界）最易受累；25% 右側

臨床表現

  • 痙攣性 LLQ 腹痛 + 明顯血便；發燒和腹膜刺激徵象 → 懷疑腸梗塞
  • 疾病譜：可逆性結腸病（35%）→ 一過性結腸炎（15%）→ 慢性潰瘍性結腸炎（20%）→ 狹窄（10%）→ 壞疽（15%）→ 爆發性結腸炎（<5%）

診斷與治療

  • 診斷：軟式乙狀結腸鏡或大腸鏡 / CT 腹腔確認診斷；排除 IBD、感染性結腸炎
  • 治療：腸道休息 + IVF + 廣效抗生素 + 系列腹部檢查
  • 手術適應症：腸梗塞 / 爆發性結腸炎 / 大出血 / 藥物治療失敗 / 復發性敗血症 / 狹窄
  • >50% 在 48 小時內接受保守治療後緩解
  • 與 AMI 的鑑別重點：缺血性結腸炎疼痛較輕，通常有血便；AMI 疼痛極劇烈但早期腹部柔軟（不成比例）

慢性腸繫膜缺血
  • 機轉：腸繫膜動脈粥樣硬化 → 腸道血流不足
  • 症狀：「腸道心絞痛」= 餐後 30 分鐘腹痛（腸道需氧增加）+ 早飽 + 體重減輕（因懼怕進食）；疼痛若變為持續性 → 懷疑急性血栓形成
  • 診斷：雙功能超音波或 CTA；血管攝影金標準；胃張力計（Gastric tonometry）運動測試

急性胰腺炎的治療核心是積極補液（LR 優於 NS）+ 早期口服進食 + 鎮痛。膽石性胰腺炎合併膽管炎或膽汁阻塞 → 24 小時內 ERCP。預防性抗生素無益。重症（持續器官衰竭 ≥48h）死亡率高，感染性壞死是最危險的局部併發症。

  • 酒精（30%）：男性 > 女性；每天 4-5 杯 >5 年；通常慢性 + 急性發作
  • 代謝性：高三酸甘油酯（TG >1000；2-5%）；高鈣血症
  • 藥物（<5%）：5-ASA、6-MP / AZA、ACEI、類固醇、Furosemide、GLP-1 RA、INH、MNZ、磺胺類、Thiazide、Valproate、Tetracycline；Cisplatin
  • 解剖性：胰腺分裂、環狀胰腺、Sphincter of Oddi 功能障礙
  • 感染：Coxsackie、CMV、EBV、HIV、腮腺炎、Mycoplasma、TB
  • ERCP 後（5%）：預防：直腸 Indomethacin ↓ 發生率；高風險 → 暫時性胰管支架
  • 腫瘤：胰腺 / 壺腹部腫瘤；轉移（RCC 最常見、乳癌、肺癌、黑色素瘤）

診斷標準（符合 2/3）

  • 特徵性腹痛（上腹 / LUQ，半數放射至背部）
  • Lipase 或 Amylase >3× ULN（Lipase 更靈敏且 t1/2 更長；>10k → 80% PPV 膽石性）
  • 影像確認（CT 或 MRI）
  • Amylase / Lipase 不與嚴重度相關；ALT >3× ULN → 95% PPV 膽石性胰腺炎
數值判讀
嚴重度評估
  • 輕度（80%）：無器官衰竭、無局部 / 全身併發症；低死亡率
  • 中度：一過性（<48h）器官衰竭 ± 局部 / 全身併發症；高發病率
  • 重度：持續（≥48h）器官衰竭（AKI、呼吸衰竭、GIB、休克）；極高死亡率

預後工具

  • BISAP（第一個 24 小時使用）：5 分（BUN >25、意識改變、SIRS、年齡 >60 歲、肋膜積液）；≥3 分 → ↑ 器官衰竭 / 死亡率風險
  • Ranson / APACHE II：48 小時評估；PPV 不佳
  • CTSI（CT 嚴重度指數）：48-72 小時後 CT 資料（液體積聚 + 壞死程度）→ 預測死亡率；但落後於臨床症狀

影像

  • 腹部超音波（所有患者）：排除膽石 / 膽管擴張；氣體常遮擋胰腺視野
  • CT（含顯影劑）：前 3 天不建議（局部併發症尚未顯現 + AKI 風險）；48-72h 後若持續疼痛或臨床惡化 → CT 排除壞死 / 液體積聚
  • MRI / MRCP：較早顯示壞死；可評估結石和胰管破裂
  • EUS：用於隱匿性膽道疾病（膽泥）
用藥指引
急性胰腺炎治療

補液

  • 中度積極補液：低血容量 → 10 mL/kg IVB → 1.5 mL/kg/h；過度積極補液不改善預後且增加液體過多風險
  • LR 優於 NS（↓ SIRS；有高鈣血症時避免）
  • 目標：↓ BUN 和 Hct（12-24 小時）；UOP 0.5-1 mL/kg/h

營養

  • 早期口服進食（24 小時內）：可耐受時立刻開始；不需等待腹痛完全緩解或 Lipase 正常化
  • 輕度：疼痛緩解 + 無噁心嘔吐 + 無腸阻塞 → 開始低脂低殘渣飲食（和流質飲食一樣安全）
  • 重症（48-72 小時內）：早期腸道灌食優於 TPN（↓ 感染併發症）；鼻胃管 = 鼻空腸管

其他

  • 止痛：IV 鴉片類藥物（監測呼吸；腎功能不全時調整劑量）
  • 膽石性胰腺炎：合併膽管炎、敗血症或 Total bili ≥5 → 24 小時內緊急 ERCP；輕度 → 初次住院期間膽囊切除術（↓ 復發）；壞死性胰腺炎先延後手術至發炎消退
  • 高三酸甘油酯：胰島素輸注（活化脂蛋白脂酶）；Fibrate；± 血漿置換
  • 預防性抗生素：無效（即使壞死性胰腺炎）

局部併發症

  • 急性液體積聚：早期發生；無包膜；通常無症狀；1-2 週內自然消退
  • 假性囊腫：發作後約 4 週；有包膜；無症狀不需治療（無論大小）；有症狀 → 內視鏡引流（優先）vs 經皮 / 手術
  • 胰腺壞死（無菌性）：無症狀可觀察；不需預防性抗生素
  • 感染性壞死（高死亡率）：多為 GNR 腸道菌；Carbapenem 或 Pip/tazo 或（第三代 Cephalosporin + MNZ）；CT 引導 FNA 若惡化；若穩定 → 延後引流 >4 週讓壞死液化（WOPN）→ 內視鏡壞死清除術或經皮引流（優於開放手術）
  • WOPN（纖維壁包裹的胰腺壞死）：感染或有症狀 → 內視鏡或經皮引流
  • 全身性：ARDS、腹腔間室症候群、AKI、GIB（假性動脈瘤）、DIC；代謝性：低鈣血症、高血糖；血管性：假性動脈瘤、脾靜脈血栓（最常見的腹腔靜脈血栓）

慢性胰腺炎
  • 機轉：反覆急性發作 → 發炎浸潤 → 纖維化 → 外分泌和內分泌功能喪失（90% 胰腺功能損失後才出現）

代謝功能相關脂肪肝（MASLD / MASH）
  • 定義：肝臟脂肪浸潤 + 排除酒精或其他原因（HCV 等）；MASLD = 脂肪變性無發炎；MASH = 脂肪變性 + 發炎 ± 纖維化

病毒性肝炎 Viral Hepatitis
病毒性肝炎的核心：HAV 和 HEV 是急性自限性（無慢性化）；HBV 和 HCV 可慢性化導致肝硬化和 HCC。HBV 需依免疫活躍期決定治療；HCV 現在有高效 DAA（直接作用抗病毒藥），SVR >90%，且所有慢性感染者均建議治療。

C 型肝炎（HCV）
  • 傳播：血液（IVDU、1992 年前輸血）> 性行為；20-30% 無明確誘因；潛伏期 1-5 個月
  • 急性感染：80% 無症狀；10-20% 有症狀肝炎；急性肝衰竭罕見；自發清除取決於 IL28B 和 HLA class II 基因型
  • 慢性化：高達 85% → 慢性肝炎；20-30% 發展為肝硬化（約 20 年）；肝硬化後每年 1-4% HCC 風險
  • 肝外表現：混合型冷凝球蛋白血症、紫質症、扁平苔蘚、MPGN、NHL、單株球蛋白病

診斷

  • 篩查（所有成年人至少一次）：anti-HCV（ELISA）→ 6 週轉陽；陽性不代表痊癒或免疫
  • 確認活動性感染：HCV RNA（2 週內陽性）；基因型（指引療程和預測療效）
  • 急性 = HCV RNA ⊕、anti-HCV ±；痊癒 = HCV RNA ⊖、anti-HCV ±；慢性 = HCV RNA ⊕、anti-HCV ⊕

D 型肝炎（HDV）與 E 型肝炎（HEV）

D 型肝炎（HDV）

  • 需要 HBV 才能感染（需要 HBsAg 作為外膜）
  • 傳播：血液或性行為；非洲 / 東歐流行
  • 共同感染（HBV + HDV 同時）：>80% 痊癒

  • 診斷：所有 HBV 患者篩查 anti-HDV 一次；陽性 → HDV RNA 確認
  • Bulevirtide（NTCP 受體阻斷劑）是首個核准 HDV 靶向藥物

E 型肝炎（HEV）

  • 流行地區最常見的急性病毒性肝炎原因
  • 傳播：糞口途徑；中亞 / 東南亞 / 非洲 / 墨西哥旅遊；接觸豬（歐洲）
  • 自然史：多無症狀；懷孕死亡率高（10-20%）；移植患者少數慢性化
  • 診斷：IgM anti-HEV（透過 CDC）；HEV RNA；治療支持性為主
  • 肝外症狀：關節炎、胰臟炎、貧血、神經（GBS、腦膜腦炎）
其他肝炎 Other Causes of Hepatitis
非病毒性肝炎的三大類：自體免疫肝炎（AIH）靠切片診斷，類固醇有效；酒精性肝炎預後工具（Maddrey、Lille、MELD）決定是否給類固醇；APAP 中毒的 NAC 應盡快給，不要等到 ALT 升高才開始。
用藥指引
自體免疫肝炎（AIH）

分類

  • Type 1：anti-smooth muscle Ab（ASMA）、ANA；anti-SLA（10-30%，與重症復發相關）；IgG 常 ↑
  • Type 2：anti-LKM1；anti-LC-1
  • 重疊症候群：AIH + PBC（懷疑若 AMA 陽性 + 特徵性組織學 → 「自體免疫膽管炎」）
  • 藥物誘發：Minocycline、Nitrofurantoin、Infliximab、Hydralazine、Methyldopa、Statin、ICI

臨床與診斷

  • 70% 女性；雙峰分布（青春期和第五個十年）
  • 40% 以重症 AIH 表現（ALT >10× ULN，3% 為急性肝衰竭）；34-45% 無症狀
  • 肝外表現：甲狀腺炎、關節炎、UC、Sjögren's、Coombs 陽性貧血、乳糜瀉
  • 診斷：血清學（ANA、ASMA、anti-LKM1）+ IgG ↑ + 無病毒性肝炎 + 肝臟切片（界面性肝炎 + 淋巴漿細胞浸潤）
  • 治療適應症：ALT/AST >10× ULN；IgG >2× ULN + ALT >5×；橋式 / 多腺泡壞死；肝硬化 + 發炎；AST/ALT >2× + 症狀

治療

  • 誘導方案（選一）：Prednisone 單藥；Prednisone + AZA；Budesonide（非肝硬化）+ AZA → 65-80% 緩解率
  • 類固醇緩慢減量；復發率 50-80%
  • 無反應或 AZA 不耐受：Cyclosporine、Tacrolimus、MMF、Rituximab、Infliximab
注意！
酒精性肝炎（Alcohol-associated Hepatitis）
  • 症狀：漸進性黃疸、壓痛性肝腫大、發燒、腹水、食欲不振
  • 血液：AST 通常 <400；AST:ALT ≥ 2:1；↑ Tbili >~3；↑ INR；↑ WBC

預後工具

  • Maddrey 判別函數（MDF）：4.6 × (PT − control) + Tbili；≥32 → 未治療 1 個月死亡率 30-50%
  • Lille 模型：類固醇治療 4-7 天後評估；>0.45 → 類固醇反應差 + 6 個月存活率 ↓
  • MELD：>18 合理考慮類固醇；MELD + Lille 組合是死亡率最佳預測因子

治療

  • 適應症（滿足任一）：MDF ≥32；MELD >18；有肝性腦病
  • Glucocorticoid：Methylprednisolone 32 mg/天 或 Prednisolone 40 mg/天 × 4 週 → 4-6 週緩慢減量；↓ 1 個月死亡率（但不影響 3 個月死亡率）
  • 類固醇禁忌：活動性 GIB、未控制的細菌 / 真菌 / TB 感染、AKI、未治療的 HBV
  • 早期營養：目標 35-40 kcal/kg/天 + 1.5 g 蛋白質/kg/天（腸道灌食若不能口服）
  • 早期肝臟移植：精心篩選的患者（不需等待傳統的 6 個月戒酒期）
  • Lille 分數 >0.45 在類固醇治療 1 週後 → 考慮停止類固醇（持續使用增加感染風險但無額外存活獲益）
緊急處置
乙醯胺酚（APAP）肝毒性
  • 機轉：90% APAP → 無毒代謝物；~5% 經 CYP2E1 → NAPQI（肝毒性中間代謝物）→ 被穀胱甘肽解毒；過量 (>10g) → 耗盡穀胱甘肽 → 肝損傷
  • CYP2E1 誘導（低劑量 2-6g 即可引起毒性）：禁食、酒精、抗驚厥藥、抗 TB 藥（INH）
  • 症狀出現時間：肝功能異常 2-6 天後才明顯；早期症狀為噁心嘔吐和腹痛

治療

  • 洗胃 + 活性碳：攝入後 4 小時內
  • 早期轉移植中心評估
  • N-Acetylcysteine（NAC）：攝入後 72 小時內均可給予；攝入時間不明或慢性超量（>4g/天）→ 低閾值給予 NAC
  • 口服 NAC（首選）：140 mg/kg 負荷 → 70 mg/kg q4h × 17 次
  • NAC 也改善非 APAP 急性肝衰竭的存活率
數值判讀
缺血性肝炎（「休克肝」）
  • 定義：ALT / AST >1000 + ↑↑ LDH（ALT:LDH <1.5，因 LDH 同步大幅升高）；延遲出現 ↑↑ Tbili
  • 原因：通常需要低血壓 / 休克 + ↑ 靜脈壓 + 低氧同時存在（如 CHF、敗血性休克）；Budd-Chiari
  • 診斷：臨床病史 + 腹部超音波 + Doppler；ALT:LDH <1.5（毒素或病毒性 >1.5）
  • 一旦血流動力學改善 → LFT 迅速下降（數天內）
肝硬化 Cirrhosis
肝硬化的處置核心是管理門脈高壓的三大併發症：腹水、靜脈曲張出血、肝性腦病。一旦出現失代償事件（腹水、靜脈曲張出血、腦病）→ 2 年死亡率 >50%。MELD-Na 決定移植優先順序；Child-Pugh 分類預測 1 年存活率。預防 SBP 和 HCC 是每次門診的必要事項。
機轉說明
病因與臨床表現

常見病因

  • 酒精、毒素（砷）
  • MASLD（大多數「隱源性」肝硬化的原因）
  • 病毒性肝炎（慢性 HBV、HCV、HDV）
  • 自體免疫肝炎；代謝疾病（血色素沉著症、Wilson 病、α1-AT 缺乏、乳糜瀉）
  • 膽道疾病（PBC、PSC、繼發性膽汁性肝硬化）
  • 血管性（Budd-Chiari、右心衰竭、縮窄性心包炎、SOS）
  • 藥物（Amiodarone、MTX、Vit A、Valproate、INH）

身體檢查

  • 肝臟：早期腫大（左葉為主）→ 晚期萎縮、結節狀
  • 肝功能衰竭跡象：黃疸、蜘蛛血管瘤 + 手掌紅斑（↑ 雌二醇）；Dupuytren 攣縮；Muehrcke 線和 Terry 甲；腮腺 / 淚腺 ↑；男性女乳 / 睾丸萎縮；撲翼樣震顫；腦病；杵狀指
  • 門脈高壓跡象：脾腫大；腹水；腹部靜脈擴張（caput medusae）；Cruveilhier-Baumgarten 靜脈雜音
數值判讀
預後評分與追蹤

Child-Pugh 分類（腹水、腦病、Bili、Alb、INR）

  • A 級（5-6 分）：1 年存活 100%
  • B 級（7-9 分）：1 年存活 80%
  • C 級（10-15 分）：1 年存活 45%

MELD-Na（3 個月存活率）

  • 基於 Sex、Alb、Cr、INR、Total Bili、Na；器官分配的關鍵指標
  • MELD <21：額外預測因子包括頑固性腹水、↑ HVPG、↓ QoL
  • 評估器官移植優先順序：MELD ≥15 開始評估

常規追蹤

  • HCC 篩查：所有肝硬化患者每 6 個月腹部超音波 ± AFP
  • 食道靜脈曲張：初診時 EGD；有靜脈曲張 → 非選擇性 β-blocker 或 EVL
  • 疫苗：HAV、HBV、PCV13、PPSV23、COVID-19；每年流感
  • 藥物調整：APAP ≤2g/天；避免 NSAIDs；禁止 Aminoglycosides；限制 ACEI/ARB（影響利尿劑效果）
  • 營養：補充脂溶性維生素 + 鋅；蛋白質攝入 1.2-1.5 g/kg/天
緊急處置
食道胃靜脈曲張

一級預防（預防首次出血）

  • 初診 EGD 篩查；中到大型靜脈曲張 → 非選擇性 β-blocker（Carvedilol 優於 Nadolol 或 Propranolol → ↓ MAP 和 HVPG 更多；↓ 靜脈曲張進展）
  • EVL（內視鏡靜脈曲張結紮）：↓ 首次出血風險 > β-blocker，但死亡率無差異；有食道穿孔、潰瘍等嚴重併發症；每 1-4 週重複直到靜脈曲張消失
  • β-blocker vs EVL：依患者和醫師偏好；小型靜脈曲張 β-blocker 優先；大型可選 EVL；不建議合用於一級預防
  • SBP 需維持 ≥90 mmHg；若低血壓 / MAP ≤82、Na <120、AKI、HRS、SBP、敗血症、嚴重酒精性肝炎 → 考慮停 β-blocker

急性靜脈曲張出血（見 GIB 章節）

  • Octreotide + 預防性抗生素（CTX）→ 12 小時內 EGD + EVL
  • 早期 TIPS（Child B/C，72 小時內）→ ↓ 再出血，↑ 腦病，死亡率無差異

二級預防（預防再出血）

  • 所有患者在首次出血後（再出血風險 ~50%，死亡率 ~30%）→ 非選擇性 β-blocker + EVL 優於單一治療
  • TIPS：頑固性出血或考慮 Child B/C 72 小時內入院（↓ 再出血 + ↑ 腦病 + 死亡率無改變）
用藥指引
肝性腦病（Hepatic Encephalopathy）
  • 機轉：肝臟解毒 NH₃ 失敗 + 其他物質（ADMA）→ 腦水腫、↓ O₂ 消耗、↑ ROS → 腦功能障礙
  • 誘發因素：出血、感染、藥物不遵從、↓ K、↓ Na、脫水、低氧、門體分流（TIPS）、鎮靜藥、急性肝損傷（PVT）
  • 分級：1 = 注意力 ↓、睡眠障礙；2 = 撲翼樣震顫、意識改變；3 = 嗜睡、僵硬；4 = 昏迷
  • 血清 NH₃：敏感性差，不可用於診斷或監測治療；仍為臨床診斷

治療

  • 找出並矯正誘因
  • Lactulose：結腸酸化 → NH₃ → NH₄⁺（無法吸收）；目標每天 2-4 次軟便；PEG 效果相當
  • Rifaximin 550 mg bid：↓ 腸道細菌 → ↓ NH₃ 產生；加在 Lactulose 上（第二次復發後、TIPS 後、頑固性）↓ 復發
  • 維持 K >4；避免鹼血症；嚴重腦病（3-4 級）→ 插管
  • Lactulose 在 ALF 的腦病中效益有限且可能有害（不建議）
注意！
肝腎症候群（HRS）
  • 機轉：腎臟血管收縮（交感神經、RAAS、加壓素）→ ↓ 腎血流
  • 診斷標準（所有項目）：肝硬化 + 腹水；AKI（Cr 48h 內 ↑ ≥0.3 或 50%）；停利尿劑 + 白蛋白補充（1 g/kg/天 × 2 天）後無改善；無休克 / 腎毒性藥物 / 固有腎臟疾病
  • HRS-AKI：<2 週發展；通常於嚴重肝衰竭後；中位存活 2 週
  • HRS-CKD：較隱匿；中位存活 6 個月
  • 誘因：GIB、過度利尿、感染、系列大量抽腹水、Aminoglycosides、NSAIDs

治療

  • 危重症：Terlipressin（或 Norepinephrine / Vasopressin）+ Albumin → ↑ MAP 10 mmHg
  • 非危重症：Octreotide（100-200 μg SC tid）+ Midodrine（最高 15 mg PO tid）+ Albumin 1 g/kg（第一天，最高 100g）→ 之後 20-60g qd ↑ MAP
  • 可能需透析或 TIPS 橋接至肝臟移植
  • 低血氧或缺血時停止血管收縮劑；唯一根治方法是肝臟移植
緊急處置
自發性細菌性腹膜炎（SBP）
  • 危險因子：腹水蛋白 <1 g/dL；SBP 病史；當前 GIB
  • 表現：腦病、腹痛、發燒，但 25% 無症狀 → 所有住院肝硬化 + 腹水患者常規診斷性穿刺
  • 病原體：單菌性 GNR（E. coli、Klebsiella）> GPC（S. pneumo、腸球菌）
  • 診斷：腹水 PMN ≥250/mm³（修正血性穿刺：每 250 RBC 減 1 PMN）
  • 治療：第三代 Cephalosporin 一線；危重症考慮 Carbapenem；若先前 MRSA 陽性 → Vancomycin；IV 白蛋白 1.5 g/kg（診斷當天）+ 1 g/kg（第 3 天）→ 有腎功能障礙或肝臟失代償者 ↑ 存活率
  • 治療 48 小時無改善 → 重複穿刺；預期 PMN 下降 25%

預防

  • 無限期預防適應症（符合之一）：有 SBP 病史；或腹水蛋白 <1.5 + (Na ≤130 或 Cr ≥1.2 或 BUN ≥25 或 [CPS ≥9 + Bili ≥3])
  • Ciprofloxacin 500 mg qd 或 TMP-SMX DS qd
  • 短期預防（GIB）：Ceftriaxone 1g IV × 5 天（進食後改 Cipro bid 或 TMP-SMX DS bid）
判斷框架
肝臟移植
  • 評估時機：MELD ≥15
  • 適應症：反覆 / 嚴重腦病；頑固性腹水；反覆靜脈曲張出血；HRS；HPS；POPH（肺動脈高壓）；HCC（單顆 ≤5 cm 或 ≤3 顆最大 ≤3 cm）；ALF
  • 絕對禁忌：社會支持不足；活躍物質濫用（部分例外）；敗血症；嚴重心肺疾病；肝外惡性腫瘤；膽管癌 / 血管肉瘤；持續不遵從；AIDS；ALF 合併持續 ICP >50 或 CPP <40
  • 存活率：1 年高達 90%；5 年高達 80%
  • 自體免疫肝病（AIH / PBC / PSC）移植後 10-30% 復發，但不影響長期存活
腹水 Ascites
腹水評估從SAAG（血清-腹水白蛋白梯度）開始：≥1.1 = 門脈高壓；<1.1 = 非門脈高壓。治療的核心是限鈉（<2g/天）+ Spironolactone + Furosemide（5:2 比例）。頑固性腹水定義為最大劑量利尿劑仍難以控制，選項包括 LVP + 白蛋白或 TIPS。

肝硬化腹水治療
一般腹水
  • 限鈉：<2g（90 mEq）/天；Na <125 時限制自由水攝取
  • 利尿劑：Spironolactone + Furosemide 以 5:2 比例（如 100:40 mg）；目標每天 ↓ 體重 ~1L（有周邊水腫）或 ~0.5 kg/天（無水腫）
  • 尿液 Na/K >1 → 有效利尿（若患者遵從低鈉飲食）
  • 避免 NSAIDs / ACEI / ARB（↓ 利尿劑效果）
  • 長期白蛋白輸注 ↓ 死亡率（但尚未廣泛採用）
頑固性腹水（5-10% 患者；2 年存活率 25%）
  • 定義：最大劑量利尿劑（Spironolactone 400 mg + Furosemide 160 mg）下腹水仍難以控制，或因利尿劑副作用（AKI、Na <125、↑K、腦病）而無法使用最大劑量
  • 大量腹水穿刺（LVP，>5L）：每移除超過 5L 給予 6-8 g 白蛋白/L（↓ 穿刺後循環功能障礙風險）；若有 SBP 避免 LVP
  • TIPS：↓ 腹水 75%；↑ Cr；↑ 腦病（20-30%）；存活獲益 vs LVP 仍有爭議；禁忌：二級以上腦病、CHF 或肺高壓、活躍感染、膽道阻塞
  • 若 β-blocker 受限（低血壓）：可加 Midodrine
  • 考慮肝臟移植（頑固性腹水失敗選項）

緊急處置
ICU 處置與特定治療
確診 ALF → 立刻轉移植中心 ICU；同時評估肝臟移植適應症。

一般 ICU 支持

  • 腦水腫 / ↑ ICP：考慮 ICP 監測（3-4 級腦病時）；↑ ICP → Mannitol 0.5-1.0 g/kg；動脈 NH₃ >150 + 3-4 級腦病 + AKI 或需升壓藥 → 預防性 3% NS（目標 Na 145-155 mEq/L）；頑固性 ↑ ICP → Barbiturates 或低體溫療法
  • 腦病：3-4 級 → 插管；Lactulose 效益有限且可能有害（不建議）
  • 凝血障礙：Vit K；FFP / 血小板 / Cryo 僅用於活動性出血（↑ 容積過多風險）；PPI 預防 GIB
  • 感染：發燒或感染懷疑 → 低閾值廣效抗生素（Vancomycin + 第三代 Cephalosporin）；高風險患者（TPN、CVVH）考慮抗真菌
  • 腎臟：AKI → CVVH；嚴重低磷血症 / 低鈉血症 / 低鉀血症需積極補充
  • 血糖：積極補充葡萄糖（↓ 葡萄糖合成）

特定病因治療

  • APAP：NAC（至攝入後 72h 均可給）
  • HBV：抗病毒藥（Entecavir 或 TDF）
  • HSV：IV Acyclovir
  • Wilson 病：血漿置換（暫時）→ 移植
  • A. phalloides 中毒：PCN-G + Silymarin；移植
  • 妊娠相關：分娩
  • Budd-Chiari：TIPS + 抗凝
  • 自體免疫肝炎：類固醇（證據有限但常使用）

用藥指引
血色素沉著症（Hemochromatosis）
  • 遺傳性（HFE 突變，85%）：C282Y 同合子（北歐裔 0.5%）；C282Y/H63D 複合異合子；C282Y 同合子：男性 28%、女性 1% 發展鐵過載（女性因月經保護）
  • 非 HFE 突變：Hemojuvelin、Hepcidin、Transferrin receptor 2、Ferroportin
  • 繼發性鐵過載：鐵負荷性貧血（海洋性貧血、骨髓增生不良）；靜脈輸血；慢性肝病（HBV、HCV、MASH、酒精）；飲食鐵過量

臨床與診斷

  • 早期：倦怠、關節痛（MCP 關節最典型）、性欲下降
  • 晚期（罕見）：青銅色皮膚（黑色素 + 鐵）、性腺功能低下（青少年型）、DM、關節病變、CHF、感染（Vibrio、Listeria、Yersinia 風險 ↑）、肝硬化（合併酒精 / 脂肪肝時 15% HCC 風險）
  • 診斷：鐵飽和度（Iron/TIBC × 100%）>45%；↑ Ferritin（急性期反應物，特異性差；年輕患者可能正常）→ 確認 HFE 基因型；若 HFE 陽性 + Ferritin >1000 或 ↑ LFT → 肝臟切片（定量鐵指數 + 纖維化分期）；MRI（黑色肝臟）

治療

  • 靜脈放血：每週 250 mL（~250 mg 鐵）直到鐵飽和度 <50% 且 Ferritin 50-100 μg/L；之後每 3-4 個月維持；PPI 可 ↓ 腸道鐵吸收
  • 避免 Vit C 補充（促進鐵吸收）；避免未熟海鮮（Vibrio 風險）；Deferoxamine（放血禁忌時）
  • 基因諮詢（一等親篩查）

Wilson 病
  • 遺傳：ATP7B 基因突變 → 銅轉運障礙 → 銅過載；常染色體隱性；1/30000
  • 發病年齡：通常 3-55 歲
  • 肝外表現：神經精神症狀（帕金森樣、運動障礙、行為改變）；Kayser-Fleischer 環（神經精神型 99%，但無神經症狀者 <50%）；Coombs 陰性溶血性貧血；腎臟
  • 診斷：24h 尿銅 ↑；血清 Ceruloplasmin ↓（Se 90%）；肝臟切片（肝臟銅含量）；ATP7B 基因測試（不清楚時）；ALF 時：Aϕ/Bili <4 + AST/ALT >2.2 診斷 Wilson 病優於尿銅或 Ceruloplasmin
  • 治療：螯合劑 D-penicillamine（補充 Vit B6，因 D-pen 使其失活）；替代：Trientine（毒性較低，效果相當，但較貴）；鋅（↓ 腸道銅吸收，可延緩疾病；螯合劑後 5 小時給予）；避免高銅食物；ALF 或對治療無反應 → 肝臟移植

α1-抗胰蛋白酶缺乏（α1-AT）
  • 機轉：異常 α1-AT → 在肝臟聚合（肝硬化）+ 在肺臟失去蛋白酶抑制活性（肺氣腫）；影響 1/3000 歐裔
  • 臨床表現：新生兒肝炎；兒童膽汁鬱積性黃疸；兒童 / 成人 ↑ ALT/AST 或肝硬化；全葉性肺氣腫；壞死性脂膜炎；ANCA 陽性血管炎
  • 診斷：血清 α1-AT 濃度（急性期反應物，需同時測 CRP）；蛋白酶抑制劑（Pi）表現型分型（金標準）；Z 型等位基因（ZZ 成人 63% 有慢性肝病）；肝臟切片：PAS 陽性細胞質包涵體（特徵性）
  • 治療：避免酒精；維持正常 BMI；嚴重者 → 肝臟移植；抗 AAT 療法研究中

原發性膽汁性膽管炎（PBC）
  • 機轉：小葉間膽管的自體免疫破壞
  • 流行病學：女性 40-60 歲；合併 Sjögren's（50%）、Raynaud's、硬皮病、乳糜瀉、甲狀腺疾病；X 染色體單體症；IL-12α 和 IL-12R 基因變異
  • 症狀：倦怠 / 睡眠障礙、搔癢、黃疸；50% 僅有 LFT 異常（無症狀）
  • 診斷：↑ Aϕ；↑ Bili；↑ IgM；↑ 膽固醇（主要是 HDL-C）；AMA 陽性（95%，Se + Sp 高）→ AMA 陽性時不需切片；0.5% 一般人口 AMA 陽性 + 正常 LFT → 10% 在 6 年後發展 PBC；AMA 陰性者做肝臟切片（常有 ANA、ASMA，預後與 AMA 陽性者相同）

治療

  • Ursodeoxycholic acid（UDCA）13-15 mg/kg bid：~25% 完全反應；↑ 存活、↓ 組織學進展、↓ 靜脈曲張風險；治療 3-6 個月後評估生化反應
  • PPAR 促效劑（UDCA 後二線）：Elafibranor（α + δ）或 Seladelpar（δ）→ UDCA 無效或不耐受時；不能用於失代償肝硬化；GI 副作用（Elafibranor）；↓ Aϕ
  • Bezafibrate：UDCA 反應不足的二線替代
  • 搔癢：Cholestyramine（UDCA 後 2-4 小時給，避免干擾吸收）；頑固性 → Naltrexone、Rifampin、血漿置換
  • ESLD：肝臟移植（~20% 復發，但不影響長期存活）
機轉說明
原發性硬化性膽管炎（PSC）
  • 機轉：肝內外膽管瀰漫性發炎 → 纖維化和多發性膽管狹窄；HLA-B8 和 DR3 / DR4 相關；常有自體抗體
  • 流行病學：男性 > 女性（20-50 歲）；~70% 合併 IBD（多為 UC）；1-4% UC 患者有 PSC；男性、無 IBD、小管型 PSC → 預後較好
  • 症狀：倦怠、搔癢、黃疸、發燒、RUQ 疼痛、IBD；50% 無症狀
  • 診斷：膽管攝影（MRCP ± ERCP）→ 多發性串珠狀膽管狹窄；排除繼發性原因；小管型 PSC（切片確診，MRCP 可能正常）；肝臟切片：「洋蔥皮狀」纖維化圍繞膽管
  • 治療：支持性治療 + 脂溶性維生素補充；無藥物改善存活；UDCA 可 ↓ Aϕ 和改善症狀，但效益不確定；有顯著狹窄 → 內視鏡氣球擴張 / 支架 / 手術切除
  • 膽管癌（CCA）：15% 終生風險；每年 MRCP 或超音波 + CA19-9 監測
  • 肝臟移植：~30% 復發（若合併 UC，結腸切除術可 ↓ 復發）
機轉說明
肝臟血管疾病

門靜脈血栓（PVT）

  • 病因：肝硬化或高凝狀態（癌症、感染、OCP、膠原血管疾病、IBD）
  • 急性：腹痛、腹脹、發燒、肝臟失代償；腸繫膜靜脈受累 → 腸道梗塞；發燒 → 考慮門靜脈炎（pylephlebitis）
  • 慢性：無症狀或門脈高壓（靜脈曲張出血、脾腫大）；「門靜脈海綿狀瘤」（慢性 PVT 的側支血管網路）
  • 診斷：超音波 + Doppler → MRA 或 CT 確認
  • 治療：急性非肝硬化 → LMWH → Warfarin 或 DOAC × 6 個月（無法逆轉原因則無限期）；急性肝硬化 + 完全主門靜脈阻塞或延伸至腸繫膜靜脈 → 建議抗凝（先篩查高風險靜脈曲張）；慢性非肝硬化 → 抗凝；慢性肝硬化 → 考慮肝臟移植

Budd-Chiari 症候群

  • 肝靜脈或下腔靜脈流出阻塞 → 竇狀隙充血 + 門脈高壓
  • 病因：~50% 骨髓增生性疾病（JAK2 突變）；高凝狀態（系統性、OCP、懷孕）；腫瘤侵犯（HCC、腎臟、腎上腺）；外傷；特發性
  • 症狀：肝腫大、RUQ 疼痛、腹水、靜脈側支擴張、急性肝衰竭
  • 診斷：肝靜脈 Doppler 超音波（Se/Sp 85%）；CT 或 MRI/MRV → 靜脈阻塞 + 尾狀葉肥大；肝靜脈攝影（金標準，「蜘蛛網」型態）
  • 治療：治療根本原因；抗凝（LMWH → Warfarin）；急性 → 考慮溶栓；短狹窄 → 血管成形術 + 支架；TIPS 或 DIPS（其他方法失敗後）；肝臟移植（ALF 或其他方法失敗）

肝竇阻塞症候群（SOS）

  • 肝小靜脈和竇狀隙阻塞（舊稱靜脈閉塞病）；毒性損傷
  • 病因：HSCT 後（15%）；化療（Cyclophosphamide、Cytarabine）；放射線；某些草藥
  • 症狀：痛性肝腫大、RUQ 疼痛、腹水、體重增加、↑ Bili
  • 治療：支持性治療 + 利尿劑；嚴重 → Defibrotide（↑ 存活但副作用多且昂貴）；預防：Defibrotide；高風險 HSCT → UDCA 或低劑量肝素
膽道疾病 Biliary Tract Disease
膽道疾病的三層遞進：膽石症（無症狀）→ 膽囊炎（膽囊發炎）→ 膽管炎（膽道感染）。有症狀的膽石症建議腹腔鏡膽囊切除術（越早越好）。膽管炎是急症——Charcot 三聯症（RUQ 疼痛 + 黃疸 + 發燒），合併休克和意識改變（Reynolds 五聯症）要緊急 ERCP 引流。
機轉說明
膽石症（Cholelithiasis）

危險因子與結石類型

  • 危險因子（5F）：Female、Forty、Fat、Fertile、Family history；也有 TPN、快速體重減輕、血脂異常、懷孕、OCP、雌激素、Octreotide；Statin ↓ 症狀性膽石症和膽囊切除術風險
  • 膽固醇結石（90%）：混合型（>50% 膽固醇，多發，較小）；純型（100% 膽固醇，較大，黃色）
  • 色素性結石（10%）：黑色型（非接合型膽紅素 + 鈣，慢性溶血 / 肝硬化 / CF / Gilbert 症候群）；棕色型（細菌去接合 + 膽汁鬱積，多在膽管內，與膽道狹窄 / 寄生蟲 / 術後有關）

臨床與診斷

  • ~80% 無症狀；有症狀者每年 1-4% 發生併發症
  • 膽道疼痛：陣發性 RUQ 或上腹疼痛；突發、持續（30 分鐘 - 3 小時）、緩慢消退；± 放射至右肩胛；油膩食物誘發；噁心嘔吐；無發燒
  • RUQ 超音波：Se/Sp >95%（>5 mm 結石）；禁食 ≥8 小時（膽囊充盈）
  • EUS：Se 94-98%（超音波陰性但有症狀時）
  • CT 敏感性低（結石可等密度）；KUB 更不敏感

治療

  • 有症狀 → 腹腔鏡膽囊切除術（CCY）：越早越好
  • 無症狀 CCY 適應症：瓷器膽囊（↑ 癌症風險）；膽囊息肉 >10 mm；結石 >3 cm；接受減重手術 / 心臟移植者；鐮刀型紅血球貧血
  • 手術高風險的替代方案：經皮引流；內視鏡經乳頭引流
  • UDCA：可試用於膽固醇結石 + 膽道疼痛或手術禁忌（約 3 個月起效）；也用於快速體重減輕期間預防結石形成
  • 止痛：NSAIDs 首選（效果 ≈ 鴉片類，且避免 ↑ Oddi 括約肌壓力）
緊急處置
膽囊炎（Cholecystitis）

分類

  • 急性結石性膽囊炎：結石嵌頓於膽囊管 → 膽囊腫脹 ± 繼發感染（50%）；~20% 有症狀膽石症患者在 2 年內發展膽囊炎
  • 無結石性膽囊炎：膽囊鬱積 + 缺血（無結石）→ 壞死性發炎；發生在危重症患者（術後、TPN、敗血症、外傷、燒傷）；感染：CMV、Candida、Crypto、Campylobacter、傷寒

診斷

  • 症狀：RUQ / 上腹痛 ± 放射至右肩 / 背；噁心嘔吐；發燒
  • Murphy sign：深吸氣時 RUQ 觸診引起疼痛加劇和吸氣暫停（最有診斷意義的身體檢查）
  • 血液：↑ WBC、↑ CRP；± 輕度 ↑ Bili / Aϕ / ALT / Amylase；若 AST/ALT >500、Bili >4 或 Amylase >1000 → 懷疑膽管結石
  • RUQ 超音波（首選）：膽囊壁增厚 >4 mm + 膽囊周圍積液 + 超音波 Murphy sign → 高特異性；結石本身敏感性高
  • HIDA 掃描（Se 80-90%）：最敏感的急性膽囊炎診斷工具；HIDA 進入膽管但不進入膽囊 = 陽性；10-20% 假陽性（慢性膽囊炎、長期禁食、肝病）
  • CT 或 MRI：超音波不確定或懷疑併發症時

治療

  • NPO + IVF + NGT（頑固性嘔吐）+ 止痛
  • 抗生素：E. coli、Klebsiella、Enterobacter 為主要病原體；[第二或三代 Cephalosporin 或 FQ] + MNZ；或 Pip-tazo
  • 腹腔鏡 CCY：症狀發作 72 小時內（優於等待 7-10 天）；術中膽管攝影或 ERCP 排除膽管結石（有黃疸、膽管炎或超音波顯示膽管結石者）
  • 初始不穩定者：EUS 引導經壁引流、ERCP 引導經膽囊管引流、或經皮膽囊造口術（無腹水或凝血障礙）作為橋接治療

併發症

  • 壞疽性膽囊炎：壞死 + 膿瘍 / 穿孔風險
  • 氣腫性膽囊炎：產氣菌感染（膽囊壁積氣）
  • 穿孔：~10%，膽囊周圍膿瘍形成（常因診斷延誤）
  • CCY 後：膽管漏、膽管損傷或殘留結石、囊管殘端、Oddi 括約肌功能障礙
數值判讀
膽管結石（Choledocholithiasis）
  • 15% 膽石症患者有膽管結石；也可在膽管內原發（膽汁鬱積）
  • 症狀：RUQ / 上腹疼痛（膽管阻塞）→ 黃疸、搔癢、噁心；極少無症狀
  • 血液：↑ Bili；↑ Aϕ；ALT 或 Amylase 一過性升高提示結石排出

診斷策略

  • RUQ 超音波：結石直接看到 50-80%；可見膽管擴張（>6 mm）
  • 高可能性（ERCP 首選）：超音波直接見到結石；膽管炎；Bili >4；膽管擴張 + Bili 1.8-4
  • 中等可能性（EUS 或 MRCP 先排除）：無結石 + 膽管擴張 + Bili 1.8-4；膽石性胰臟炎；年齡 >55 歲；↑ 非 Bili 的 LFT
  • 低可能性：ERCP 不必要
  • MRCP 常先做，即使可能性高也常先做 MRCP 再決定是否 ERCP

治療

  • ERCP + 乳頭切開術 + 取石（± 碎石術）
  • CCY 在 6 週內（若未治療，>15% 發展 CCY 適應症）
緊急處置
膽管炎（Cholangitis）
Reynolds 五聯症（Charcot 三聯症 + 休克 + 意識改變）→ 緊急 ERCP 膽道引流，不能等待。
  • 病因：膽管阻塞 → 膽汁鬱積 → 近端感染；膽管結石（~85%）；惡性或良性狹窄；Clonorchis sinensis / Opisthorchis（肝吸蟲）；復發性化膿性膽管炎
  • Charcot 三聯症：RUQ 疼痛 + 黃疸 + 發燒 / 寒顫（~70% 患者）
  • Reynolds 五聯症：Charcot 三聯症 + 休克 + 意識改變（~15% 患者）→ 緊急
  • 診斷：RUQ 超音波（膽管擴張）；↑ WBC + 膽汁鬱積性 LFT + ↑ Amylase；血培養（陽性）；ERCP 或 PCT 膽管攝影

治療

  • 抗生素（廣效覆蓋膽道病原體）：Pip-tazo 或 [Ceftriaxone 或 Ciprofloxacin] + MNZ；嚴重 → Carbapenem 或 Pip-tazo
  • ~80% 保守治療 + 抗生素有效 → 擇期膽道引流
  • ~20% 需緊急膽道引流：ERCP（乳頭切開術 + 取石 + 必要時支架）；ERCP 失敗 → 經皮肝臟膽管引流（PCT）
  • 若括約肌切開術無法進行（較大結石）→ 膽道支架或鼻膽管引流暫時減壓
  • CCY 在病情穩定後
