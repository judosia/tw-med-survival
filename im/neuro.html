意識改變 / 譫妄 / 昏迷 Altered Mental Status
意識改變的評估核心是區分譫妄（急性、可逆）vs 失智（慢性、進行性），並找出潛在可逆原因。GCS ≤8 → 考慮插管保護氣道。住院患者最常見的意識改變是譫妄（尤其老年 + 基礎認知障礙）。心跳停止後腦損傷的神經預後評估需在停止低溫治療 72 小時後，以多模態工具綜合判斷。
數值判讀
意識程度與 GCS

清醒程度

  • 清醒（Alert）：正常
  • 嗜睡（Drowsy）：呼叫可喚醒
  • 遲鈍（Obtunded）：重複刺激才可喚醒
  • 昏睡（Stuporous）：強烈疼痛刺激才有反應（優選斜方肌擠壓）
  • 昏迷（Comatose）：無法喚醒；腦幹網狀活化系統、丘腦或雙側大腦半球瀰漫性功能障礙
  • 昏迷鑑別（似昏迷但有意識）：閉鎖症候群（Locked-in）；不動性緘默症（Akinetic mutism）；緊張症（Catatonia）
  • GCS ≤8 → 考慮插管保護氣道

昏迷患者的瞳孔判讀

  • 對光有反應 → 正常
  • 針尖瞳孔 → 鴉片類；橋腦病變
  • 中間位固定 → 中腦病變
  • 固定散大 → 嚴重缺氧損傷；腦疝；抗膽鹼藥物

意識改變病因

神經性（通常有局灶性缺損）

  • 血管性：缺血性中風、ICH、靜脈竇血栓、PRES、垂體卒中
  • 癲癇：癲癇持續狀態、非驚厥性癲癇、發作後期
  • 感染：腦膜炎、腦炎、膿瘍
  • 外傷：TBI、瀰漫性軸索損傷
  • ↑ 顱內壓：腫塊、水腫、水腦、腦疝
  • 發炎：自體免疫腦炎、CNS 血管炎
  • 退化性：終末期失智（AD）或快速進展性失智（CJD）

全身性（通常無局灶性缺損 + 注意力受損）

  • 心血管：全腦缺血、低血壓或高血壓
  • 肺臟：↓ PaO₂、↑ PaCO₂
  • GI：肝衰竭、↑ NH₃、硫胺缺乏
  • 腎臟：尿毒症、透析、低 / 高鈉血症、低 / 高鈣血症
  • 血液：TTP/HUS、DIC、高黏稠度
  • 內分泌：低血糖、DKA/HHS、甲狀腺低下、腎上腺危象
  • 感染：肺炎、UTI、心內膜炎、敗血症
  • 低體溫 / 高體溫
  • 藥物：抗膽鹼藥、抗組織胺、精神科用藥、Digoxin
  • 毒素 / 戒斷：EtOH、鎮靜劑、鴉片類、CO
  • 精神科：緊張症、血清素症候群、NMS
用藥指引
譫妄（Delirium）
  • 定義：急性意識混亂（小時至數天）；核心特徵：注意力受損；也可有睡眠、自律神經、幻覺、過度活躍 / 低活躍（尤其老年或前次 CNS 損傷）
  • 失智：進行性認知損害（數年）；通常影響記憶、語言、視空間、執行功能；注意力相對保留；住院患者難以診斷
  • 評估工具：CAM（Confusion Assessment Method）；4AT；住院患者常用 CAM-ICU

治療（非藥物為主）

  • 治療潛在急性疾病；排除促發因素
  • 提供感覺和認知輸入（頻繁定向、眼鏡 / 助聽器）
  • ↓ / 預防感染和約束；移除不必要管路
  • 促進正確睡眠 / 清醒週期：白天陽光；減少夜間燈光、噪音、醫療介入；Melatonin
  • 藥物：低劑量抗精神病藥僅在有安全疑慮或必要症狀控制時使用（不縮短譫妄持續時間）；避免 BZD（除 EtOH 戒斷或癲癇）
注意！
心跳停止後腦損傷（缺氧性腦損傷）
  • 適應症：心跳停止後昏迷（無法遵從語言指令，GCS <9）
  • 目標體溫管理：避免發燒（目標 ≤37.5°C）持續 72 小時；部分單位目標低溫 33-36°C × 24 小時（ROSC 後立刻開始）；証據尚不確定（NEJM 2013、2019、2021、2023）
  • 低溫療法併發症（若目標 33-36°C）：心律不整（心搏過緩最常見）；低血壓（MAP >65）；凝血障礙；感染（每 12 小時培養監測）；冷卻期間高血糖、復溫後低血糖；冷卻期間低鉀、復溫後高鉀

神經預後評估（ROSC 後 / 復溫後 72 小時）

  • 多模態評估；避免鎮靜；預後評估前等待足夠時間
  • 不良預後預測因子：無瞳孔對光反應；SSEP 缺失；CT 顯示廣泛灰白質分化喪失 + 腦溝消失；MRI 瀰漫性擴散受限；EEG 抑制
  • 不可靠的預後指標：年齡；初始心律；ROSC 時間
  • 血清神經元特異性烯醇酶（NSE）第 1-3 天：輔助預後判斷，但閾值機構間差異大
判斷框架
初始評估與緊急處置

病史（目擊者和背景資訊極關鍵）

  • 時間過程；症狀（神經、疼痛、頭痛、感染、跌倒）；過去病史（基礎認知 / 失智、癲癇、癌症、心臟、精神科、感染 / 免疫狀態）；藥物（鎮靜劑、鴉片類、抗膽鹼藥、抗凝藥、抗癲癇藥、免疫抑制劑）；藥 / 酒精使用

緊急初始處置

  • 懷疑 CNS 感染 → 經驗性 Vancomycin + CTX；考慮 Acyclovir、Ampicillin、類固醇
  • 懷疑頸椎外傷 → C-spine 固定
  • Dextrose 50 g IV + Thiamine 100 mg IV（給葡萄糖前先給硫胺，預防 Wernicke 惡化）
  • 懷疑鴉片類 → Naloxone 0.01 mg/kg；懷疑 BZD → 考慮 Flumazenil 0.2 mg IV
  • 懷疑 ↑ ICP / 腦疝 → ↑ 床頭；過度通氣；滲透療法（Mannitol、高張 NS）；Dexamethasone（腫瘤水腫）；神經外科會診

檢查

  • 所有患者：床邊血糖、電解質、BUN/Cr、LFTs、CBC、毒物篩查、UA
  • 依臨床懷疑：血培養、藥物濃度、ABG、ESR、B12、HIV、ANA、NH₃、TSH、皮質醇刺激試驗
  • 影像：CT 頭部；MRI 腦部；若懷疑中風 / SAH → CTA 頭頸部
  • LP（確認無佔位病變後）：懷疑腦膜腦炎、SAH、自體免疫腦炎、炎症；HIV 陽性患者
  • EEG：評估非驚厥性癲癇持續狀態；毒性 / 代謝性腦病；↑ ICP 時最高風險期間（復溫期）
癲癇 + 酒精戒斷 Seizures & Alcohol Withdrawal
癲癇發作的第一步是和暈厥、PNES（心因性非癲癇發作）鑑別——目擊者描述最關鍵。癲癇持續狀態（Status Epilepticus）= 驚厥發作 ≥5 分鐘或 ≥2 次發作之間無恢復——立刻 BZD，之後二線 AED，若失敗 → 全身麻醉 + ICU。酒精戒斷譫妄（DT）最重要的治療是BZD（Diazepam 首選）；β-blocker 只掩蓋症狀不能預防 DT，要避免。

癲癇分類與鑑別診斷

分類

  • 全身性（自發作起即瀰漫大腦）：強直陣攣（Grand mal）= 前驅期（感覺異常、局灶運動）→ 發作期（凝視偏移、強直 → 陣攣、咬舌、尿失禁）→ 發作後期（混淆 / 定向障礙 / 嗜睡，可達數小時；Todd 麻痺 = 發作後局灶無力）；失神（Petit mal，兒童）；肌陣攣
  • 局灶性（局限大腦特定區域）：無意識受損（舊稱單純部分）= 局灶運動 / 感覺 / 自律神經症狀（常為先兆）；有意識受損（舊稱複雜部分）= 持續數分鐘 + 認知障礙 + 發作後期；進展為雙側驚厥（舊稱繼發性全身化）
  • 癲癇持續狀態：驚厥性發作 ≥5 分鐘 或 ≥2 次無中間恢復；危及生命
  • 非驚厥性癲癇持續狀態：意識改變（混淆到昏迷）但無運動表現；需 EEG 診斷

判斷框架
病因與評估

無局灶病變的誘發因素

  • 遺傳易感性 / 癲癇症候群；酒精戒斷；違禁藥物
  • 藥物（β-lactam、Bupropion、FQ、Tramadol、Meperidine、CsA）
  • 電解質（低鈉血症）及代謝異常（尿毒症、肝衰竭、低血糖）
  • 自體免疫腦炎；特發性（~60%）

有局灶病變

  • 腫瘤；外傷；中風；硬膜下血腫；PRES；海馬硬化；膿瘍；局灶皮質發育異常

評估

  • 病史（目擊者最重要）：前驅症狀；發作前異常行為；抽搐型態（頭偏轉、眼偏斜方向通常遠離發作焦點）；尿失禁；發作後期；近期事件（疾病 / 發燒、頭部外傷、睡眠不足、壓力）；既往癲癇或家族史；既往 CNS 感染、中風、頭部外傷；失智；藥物（新藥或不規律服藥）；酒精和違禁藥物
  • 身體檢查：皮膚顏色（蒼白、發紺）；側面咬舌；Todd 麻痺（暫時性局灶無力）
  • Lab：電解質、BUN/Cr、血糖、LFT、CK、乳酸、毒物篩查；AED 血中濃度（評估不規律服藥）
  • EEG：~30 分鐘常規 EEG；24 小時內 ↑ 診斷率；睡眠剝奪和重複檢查 ↑ 敏感性；正常 EEG 不能排除癲癇（50% 有癲癇者發作間期 EEG 正常）
  • MRI：排除結構性異常（前額葉和顳葉的細矢狀切面 ↑ 敏感性）
  • 緊急頭部 CT：若有神經缺損、頭痛、發燒、認知改變、外傷
  • LP：若無佔位病變 + 懷疑腦膜腦炎（發燒、↑ WBC、頸部僵硬）；自體免疫腦炎；所有 HIV 陽性患者

抗癲癇藥物（AED）選擇與副作用
  • AED 通常保留用於 ≥2 次無誘因發作；或單次發作 + 高復發風險（EEG / MRI 異常、夜間發作）；或潛在結構性異常
  • 首次無誘因發作立刻給 AED ↓ 2 年復發風險，但不改變長期預後
  • AED 選擇依：發作類型、副作用、成本、腎 / 肝清除、致畸性、藥物交互作用

治療

  • BZD（首選）：Diazepam 優先（長效；↓ 復發性戒斷風險）；Lorazepam（短效）；Chlordiazepoxide；Oxazepam（無活性代謝物，適用肝硬化）
  • 給藥方式：通常從 Diazepam 10-15 mg IV q10-15 分鐘直到適當鎮靜，然後依 CIWA-Ar 調整（每 1-4 小時評估，直到 <10 分 × 24 小時，然後每 4-8 小時）
  • Phenobarbital：嚴重戒斷的輔助或頑固性 DT
  • 避免 β-blocker（只掩蓋症狀，無法預防 DT）
  • 必要時機械性約束（直到化學鎮靜達到）
  • 補液；補充 K、Mg、PO₄
  • 硫胺（Thiamine）：200 mg IV qd × 3 天（預防）；若懷疑 Wernicke 腦病 → 500 mg IV tid × 2 天（在給葡萄糖前先給）
  • 輕度 / 無症狀預防（CIWA <8 分，但有長期大量飲酒史或有戒斷癲癇 / DT 病史）：Chlordiazepoxide 25-100 mg q6h × 24 小時，然後漸減
頭暈 Dizziness
頭暈是症狀群，需先區分：平衡障礙（Disequilibrium）、眩暈（Vertigo）、接近暈厥（Near syncope）。眩暈的關鍵區分是周邊（內耳 / CN VIII）vs 中樞（腦幹 / 小腦）——急性暈眩患者用 HINTS 試驗鑑別。「5D」= 後循環中風的線索：頭暈（Dizziness）、複視（Diplopia）、構音障礙（Dysarthria）、吞嚥困難（Dysphagia）、共濟失調（Dystaxia）。若 HINTS 試驗提示中樞性 → 立刻 MRI。

眩暈鑑別診斷

周邊性（內耳 / CN VIII）

  • BPPV（良性陣發性位置性眩暈）：耳石脫落進入半規管；短暫旋轉性眩暈（<1 分鐘）；由體位改變誘發；Dix-Hallpike 試驗陽性（旋轉性眼球震顫，延遲數秒後出現，重複後減弱）；治療：Epley 手法 / BBQ 翻滾手法
  • 梅尼爾氏病（Ménière's）：↑ 內淋巴壓力；發作性旋轉性眩暈（分鐘至小時）+ 噁心嘔吐 + 耳脹感 + 聽力喪失 + 耳鳴；治療：利尿劑、↓ 鹽分
  • 前庭神經炎：突發 + 步態共濟失調；嚴重症狀 24-48 小時後逐漸改善（常為病毒後）；合併聽力喪失 = 迷路炎；治療：前庭物理治療

中樞性（腦幹 / 小腦）

  • 後循環中風 / TIA：「5D」= 頭暈、複視、構音障礙、吞嚥困難、共濟失調；突發（TIA 數分鐘後緩解；中風持續）
  • 其他：偏頭痛；Chiari 畸形；癲癇；MS；腫瘤；藥物 / 藥物；腦震盪

試驗方法周邊性發現中樞性發現（→ 立刻 MRI）

Dix-Hallpike 試驗（BPPV 特異性診斷）

  • 患者由坐姿迅速躺下，頭部向一側傾斜 45°；誘發旋轉性眼球震顫（延遲數秒後出現，重複後減弱）→ 患側耳向下時陽性 = BPPV
  • 仰臥滾動試驗（Supine Roll test）：頭部轉動時誘發眼球震顫 → 患側耳向下（水平半規管，約 8%）

中風 Stroke
缺血性中風占所有中風 ~85%。治療時間就是大腦（Time is Brain）——tPA 在症狀發作 4.5 小時內給予；血管內取栓（EVT）在 6 小時內（部分患者可延伸至 24 小時）。IV tPA 禁忌包括：目前或既往 ICH、3 個月內頭部外傷或中風、顱內腫瘤 / AVM / 動脈瘤、近期顱內 / 脊椎手術、活動性出血、不可壓迫部位動脈穿刺、高血壓（>185/110）、血小板 <100k、INR >1.7、服用 Xa 抑制劑、血糖 <50。顱內出血（ICH）核心是逆轉凝血 + 嚴格控制血壓；SAH = 「一生最嚴重的頭痛」→ CT 陰性後 LP。

靜脈溶栓（IV tPA / TNK）

  • tPA：0.9 mg/kg（最高 90 mg），10% 以 1 分鐘快速推注，其餘 1 小時輸注；症狀發作 4.5 小時內考慮給予
  • Tenecteplase（TNK）：0.25 mg/kg IV 單次推注（最高 25 mg）；非劣效於 tPA，若患者適合血管內治療則優先使用
  • 0-3 小時：良好神經預後絕對 ↑ 12%；ICH 絕對 ↑ 5.8%；死亡率 ↓ 趨勢 4%
  • 3-4.5 小時：良好神經預後絕對 ↑ 7.4%；ICH 絕對 ↑ 1.8%；無死亡率效益

血管內取栓（EVT）

  • 症狀發作 6 小時內，pre-mRS 0-1，ICA 或 MCA 阻塞，NIHSS ≥6，ASPECTS ≥6
  • 可延伸至 6-24 小時（梗塞大小和臨床缺損不相符，或有缺血半暗帶）

  • 溶栓前：降至 <185/110；溶栓後 24 小時內維持 <180/105（IV Labetalol 或 Nicardipine）
  • 不溶栓：允許性高血壓（除非 >220/120 或有症狀性高血壓）；症狀性低血壓考慮升壓藥

其他急性處置

病因評估

  • 心臟：住院心電監測（AF）+ 延伸門診監測；TTE（排除血栓 / 贅生物）+ 氣泡試驗（PFO）
  • 血管影像：CTA 或 MRA 頭頸部；頸動脈超音波 Doppler（若 CTA/MRA 禁忌）
  • 實驗室：血脂、HbA1C、TSH、同半胱胺酸、Lp(a)；高凝狀態評估（<65 歲或隱源性中風；理想在開始抗凝前抽血）；ESR/CRP；血培養（若懷疑全身感染）
  • MRI：診斷不確定時（尤其後循環）或定義中風亞型 / 年齡 / 大小

抗血小板治療

  • ASA ↓ 死亡和再次中風；非栓塞性中風等同 Warfarin
  • Clopidogrel：略優於 ASA，ICH ↑ 略多
  • P2Y12（Clopidogrel 或 Ticagrelor）+ ASA × 1-3 個月（vs ASA 單藥）：↓ 缺血性中風風險；適用於輕度中風或高 ABCD2 的 TIA；90 天顱內動脈粥樣硬化引起的中風

抗凝治療

  • AF → 抗凝（見心臟科）；心源性 / 矛盾性栓塞；大型顱外夾層；高凝狀態；橋接 CEA
  • 開始時機（依中風大小）：輕中度 <48 小時；大面積 1-2 週；或使用 DOAC 在無出血的情況下 4 天內

長期血壓和血脂目標

  • SBP 目標 <130/80 mmHg；LDL-C <70 mg/dL（NEJM 2017；↓ 再次中風）
  • 頸動脈內膜切除術（CEA）：有症狀狹窄 70-99% → ↓ 再次中風 65%；有症狀狹窄 50-69% → 輕度效益；手術死亡率 ≤6% 時適應症明確
  • PFO 閉合：隱源性中風 + 高風險 PFO（RoPE 評分 ≥7 或 PASCAL 分類）→ ↓ 復發 ≥50%

顱內出血（ICH）+ 蜘蛛膜下腔出血（SAH）

ICH 病因

  • AVM、動脈瘤、靜脈竇血栓 → 腦實質出血（IPH）或 SAH
  • HTN（基底核、小腦、腦幹）；腦澱粉樣血管病（葉狀）；腫瘤（黑色素瘤、腎細胞癌、絨毛膜癌、甲狀腺癌）→ IPH
    • 外傷 → 所有部位（但外傷引起的 IPH 或 SAH 嚴格來說不是中風）

臨床表現

  • ↓ 意識、噁嘔、頭痛、進行性局灶神經缺損、癲癇發作
  • SAH：「雷劈樣頭痛」（一生中最嚴重、瞬間達最大強度）；用力時發作；頸部疼痛 / 僵硬；意識喪失
  • EDH：初始清醒期後意識改變（中間清醒期）

ICH 處置

  • 逆轉凝血：目標 INR <1.4；血小板 >100k（抗血小板患者出血擴大時 → 血小板輸注？有爭議）；尿毒性凝血障礙 → DDAVP
  • 血壓控制（動脈導管）：Nicardipine 或 Labetalol 點滴；SBP 目標前 24 小時 <140，之後 <160（NEJM 2013、2016）
  • SAH：血管內線圈或手術夾閉動脈瘤 / AVM（位置和共病決定方式）；Nimodipine 預防血管痙攣（TCD 監測）；癲癇預防
  • 手術清除：EDH；SDH（>1 cm 或快速增大）；頑固性癲癇；IPH：尚無明確效益
  • 靜脈竇血栓：開始抗凝；處理 ↑ ICP 和癲癇
  • SAH：CT 陰性但高度懷疑 → LP（黃疸色沉（xanthochromia）= 2-6 小時後可見）；2-6 週後再出血風險 50-60%
周邊神經病變 + GBS Peripheral Neuropathy & GBS
周邊神經病變依受累神經分布（單神經炎、多發性單神經炎、多發性神經病變）和病生理（軸索性 vs 脫髓鞘性）分類。最常見病因：糖尿病（~30%）、特發性（~30%）、酒精。GBS 是最常見的急性 / 亞急性麻痺疾病，典型表現為上升性無力 + 反射消失 + 白蛋白細胞分離（CSF ↑ 蛋白 + 無白血球增多）。GBS 的呼吸監測最重要：FVC 和 NIF（不能只靠 PaO₂ 或 SpO₂）；進展迅速 / 呼吸衰竭 → ICU + 血漿置換或 IVIg。

  • 呼吸衰竭（25%）需要機械通氣；自律神經不穩定和心律不整（60%）

診斷

  • LP（前幾天可正常）：白蛋白細胞分離 = ↑ 蛋白（無細胞增多，WBC <10）；前 1 週 50% 升高，第 3 週 75% 升高；WBC >50 不太可能是 GBS
  • EMG / NCS：↓ 傳導速度、傳導阻滯、F 波異常；前 2 週可正常
  • FVC 和 NIF：評估呼吸衰竭風險（不能只靠 PaO₂ 或 SpO₂ 評估）

治療

  • 血漿置換（PLEX）或 IVIg 效果相當（兩者聯合無額外效益）；類固醇無效
  • 快速進展或呼吸衰竭 → ICU 支持性照護
  • 監測自律神經功能：血壓不穩定、心律不整、尿滯留、腸阻塞
  • 插管指徵（「20/30/40 法則」）：FVC <20 mL/kg；MIP（最大吸氣壓）<-30 cmH₂O；MEP（最大呼氣壓）<40 cmH₂O → 預防性插管
  • 預後：大多數在 1 年內接近恢復基礎；死亡率 3-5%；殘餘缺損：疼痛、疲勞；Erasmus GBS 預後評分可協助
重症肌無力 + 肌病變 Myasthenia Gravis & Myopathies
重症肌無力（MG）= 神經肌肉接合處的自體免疫疾病，抗體攻擊 AChR（80%）或 MuSK（4%）。核心特徵：波動性無力 + 易疲勞性（重複使用後加重，休息後改善），眼部症狀（眼瞼下垂、複視）最常見初始表現。MG 危象= 症狀惡化 + 呼吸危害 → FVC <20 mL/kg 或 NIF <-30 cmH₂O → 預防性插管。肌病變以近端對稱性無力為主（上下樓梯、從椅子站起），CK 是最重要的篩查工具。

  • 流行病學：150-250/百萬；所有年齡；高峰：女性 20-30 歲；男性 60-70 歲
  • 15% AChR MG 合併胸腺瘤；30% 胸腺瘤患者發展 AChR MG

臨床表現

  • 波動性無力 + 易疲勞性：重複使用後加重；休息後改善
  • 顱神經（最常早期受累）：眼瞼下垂（ptosis）；複視（diplopia）→ 60% 以眼部症狀初發；15% 僅限眼部；15% 有吞嚥困難 / 構音障礙
  • 肢體無力：近端 > 遠端；深腱反射保留；無明顯萎縮
  • MuSK MG（女 >> 男）：較嚴重的肢體 / 臉部 / 吞嚥困難；肌肉萎縮
  • 惡化誘因：上呼吸道感染、手術、懷孕 / 產後、藥物（Mg、Aminoglycoside、Macrolide、FQ、Procainamide、Phenytoin、D-Penicillamine、β-blocker）；Prednisone 可在初始時急性惡化
  • MG 危象：症狀惡化 + 呼吸危害
  • 膽鹼酯酶危象：抗膽鹼酯酶藥物過量 → 流涎、腹絞痛、腹瀉

診斷

  • 床邊試驗：持續向上凝視 >45 秒後眼瞼下垂加重；冰袋試驗（眼瞼上放冰袋 2-5 分鐘 → 眼瞼下垂改善，Se 77%、Sp 98%）
  • EMG：重複神經刺激 ↓ 反應（vs Lambert-Eaton：刺激後 ↑ 反應）；單纖維 EMG（最敏感）
  • Anti-AChR 結合抗體（Se 80%，眼肌型 50%，Sp >90%）；Anti-MuSK 抗體
  • 胸部 CT 或 MRI 評估胸腺（65% 增生；10% 胸腺瘤）

頭痛 Headache
頭痛的第一步是區分原發性（偏頭痛、叢集性、緊張型）vs 繼發性（危及生命）。警示徵象（Red Flags）需立刻影像：「雷劈樣頭痛」（SAH）；最嚴重的頭痛；進行性加重；姿勢相關（↑ ICP）；伴發燒 / 頸僵；年齡 >65 歲；免疫抑制；視乳頭水腫；新發神經缺損。偏頭痛是最常見的原發性頭痛（女性 15%，男性 6%）；急性期 Triptan 效果最佳（但有 CAD / 中風禁忌）；預防首選 Topiramate、Propranolol 或 Anti-CGRP。

多發性硬化症 Multiple Sclerosis
多發性硬化症（MS）是 CNS 白質的慢性自體免疫脫髓鞘疾病，女：男 = 3:1，高峰發病 20-40 歲。診斷靠時間和空間的多發性（DIS + DIT）——MRI 是最敏感工具。急性發作 = IV 甲基 Prednisolone 1 g/天 × 3-5 天（縮短發作期但不改變長期預後）。疾病修飾治療（DMT）是長期管理核心——早期開始可 ↓ 復發率和殘障進展。Lhermitte 徵（頸部前屈時電流感沿脊椎向下）= 頸髓脫髓鞘的特徵性症狀。

Uhthoff 現象

  • 體溫升高（運動、發燒、熱水澡）→ 暫時性症狀惡化；非真正復發，體溫正常後恢復

  • 空間多發性（DIS）：≥2 個 CNS 區域受累（腦室旁、皮質 / 近皮質、幕下、脊髓）
  • 時間多發性（DIT）：≥2 次臨床發作（間隔 >30 天）；或 MRI 同時有強化和非強化病灶；或 CSF 寡克隆帶（OCB）
  • MRI（最敏感）：T2/FLAIR 高信號白質病灶（腦室旁「手指狀 Dawson 指」）；急性病灶 Gd 強化（血腦屏障破壞）；脊髓病灶通常 <2 個椎體節段

  • VEP（視覺誘發電位）：P100 延遲 = 既往亞臨床視神經炎的客觀證據

MS 亞型

  • 復發緩解型（RRMS，85%）：明確復發 + 緩解期（可有殘餘缺損）
  • 繼發進展型（SPMS）：RRMS 後轉為持續進展（± 疊加復發）
  • 原發進展型（PPMS，10-15%）：從一開始就持續進展，無明確復發；男 ≈ 女；發病年齡較大；脊髓受累為主
  • 臨床孤立症候群（CIS）：第一次脫髓鞘事件，尚不符合 MS 診斷；高 MRI 病灶負荷 ↑ MS 轉化風險

背痛 + 脊椎疾病 Back Pain & Spinal Disease
背痛是最常見的住院和門診主訴之一。大多數背痛是肌肉骨骼性（非特異性），無需影像（前 6 週），但必須排除Red Flags。馬尾症候群（鞍部麻醉 + 膀胱 / 直腸功能障礙 + 下肢無力）= 外科急症。脊髓壓迫：急性 = 弛緩性麻痺 + 反射消失（「脊髓休克」）；亞急性 / 慢性 = 痙攣性麻痺 + 反射亢進（上運動神經元徵）。神經性跛行 vs 血管性跛行鑑別很重要——前者彎腰前傾改善、後者站立休息改善。

Red Flags（需立刻影像）

  • 急性改變（疼痛、無力）；上運動神經元徵（反射亢進、上翹腳趾）
  • 馬尾症候群：鞍部麻醉 + 膀胱 / 直腸或性功能障礙 + 直腸括約肌張力 ↓ + 薦椎反射消失 → 外科急症
  • 呼吸困難（平躺時）= C3-C5 受累
  • 休息或夜間疼痛（腫瘤 / 感染）；體重減輕；發燒；免疫抑制；IVDU；既往癌症；年齡 >50

身體檢查

  • Spurling 徵（頸椎神經根病變）：頸部向下加壓 + 伸展 + 同側旋轉 → 神經根痛；Se 30%、Sp 93%
  • 直腿抬高（坐骨神經痛 = 腰薦椎神經根病變）：30-70° 誘發神經根痛；同側 Se 92%、Sp 28%；交叉（對側抬腿）Se 28%、Sp 90%
  • FABER 試驗（骶髂關節症候群）：髖關節外旋時嚴重疼痛；Se 70%、Sp 100%
  • 神經學檢查：完整運動（含括約肌張力）；步態；感覺（溫痛覺、位置覺、振動覺；± 會陰部感覺）；反射（含球海綿體肌、肛門收縮 S4）

影像和檢查

  • 無 Red Flag 且非放射性背痛 → 影像陽性率低（假陽性率高，偶發性退化發現）；前 6 週保守治療
  • MRI（首選）：脊髓、神經根、軟組織評估；有症狀後 MRI 陰性 → 考慮 EMG / NCS
  • CT 脊髓攝影：MRI 禁忌時替代
  • X 光和 CT：骨骼結構評估（敏感度對脊髓 / 神經根低）；骨掃描
  • EMG / NCS：區分神經根病變、神經叢病變和周邊神經病變
  • 實驗室（依懷疑）：CBC + ESR/CRP；Ca；PO₄；CSF；血培養

脊髓壓迫（Spinal Cord Compression）
所有外傷患者 → 立刻頸椎固定。懷疑脊髓壓迫 → 緊急全脊椎 MRI + 神經外科 / 神經科會診。

病因

  • 腫瘤（椎骨轉移、硬膜內腦膜瘤 / 神經纖維瘤）；硬膜外膿瘍 / 血腫；血管畸形（硬膜 AV 瘻管）；退化性疾病（頸椎病）；外傷

臨床特徵

  • 急性（脊髓休克）：弛緩性下肢輕癱 + 反射消失
  • 亞急性 / 慢性：痙攣性下肢輕癱 + 反射亢進（上翹腳趾 ± 踝陣攣）
  • 後柱功能障礙（腿部）：振動覺和 / 或位置覺喪失
  • 感覺喪失低於病變平面：軀幹感覺平面 ± 雙側腿部症狀 → 定位於脊髓

處置

  • 所有外傷 → 頸椎固定（頸托、長背板）
  • 緊急全脊椎 MRI（含 Gd）或 CT 脊髓攝影
  • 緊急神經外科和 / 或神經科會診；轉移性疾病 → 緊急放射線 ± 手術
  • 懷疑硬膜外膿瘍 → 廣效抗生素 ± 手術
  • 類固醇（依病因）：腫瘤 → Dexamethasone 16 mg/天 IV（8 mg bid）緩慢漸減；外傷 → Methylprednisolone 30 mg/kg IV 超過 15 分鐘，然後 5.4 mg/kg/h × 24 小時（3 小時內開始）或 × 48 小時（3-8 小時後開始）
