急性腹痛評估 Acute Abdominal Pain
急性腹痛評估的核心是區分需要緊急手術（外科急腹症）vs 可以保守觀察 vs 需要進一步檢查。腹膜刺激徵象（反彈痛、肌衛、腹壁僵硬）= 手術探查的重要指標。老年患者和免疫抑制患者症狀常不典型——體溫、WBC 和腹膜徵象均可能偏低。女性育齡期腹痛必須先排除異位妊娠（hCG）。

需立刻手術評估的警示徵象

  • 腹膜炎徵象：反彈痛（Rebound tenderness）；肌衛（Guarding）；腹壁板狀硬（Rigidity）
  • 腸音消失（穿孔 / 腹膜炎）；或腸音高亢金屬音（高位腸阻塞）
  • 血流動力學不穩定 + 腹痛 → 主動脈破裂、腸系膜缺血、腸穿孔出血
  • 老年患者：症狀和體徵均可不典型，閾值更低
判斷框架
評估流程 + 影像選擇

初步評估

  • 病史重點：疼痛性質（刀割樣 vs 絞痛 vs 鈍痛）；開始方式（突發 vs 漸進）；位置和放射；時間過程；加重和緩解因素；飲食；嘔吐（先痛後吐 = 外科；先吐後痛 = 內科）；排便習慣；最後月經；LMP（育齡期女性）；近期手術史；藥物（NSAIDs → 潰瘍）
  • 身體檢查：生命徵象；視診（腹部膨隆、疤痕、蠕動波）；聽診（腸音）；叩診（鼓音 vs 濁音）；觸診（由無痛處開始，最後觸痛處）；肛診（不可省略）；骨盆腔診察（育齡期女性）

實驗室檢查

  • CBC；CMP（電解質、肝腎功能）；Lipase（胰臟炎）；LFT；凝血；UA
  • 育齡期女性：β-hCG（尿液 or 血清）必做
  • Lactate（懷疑腸系膜缺血 / 敗血症）；血型配血（外科前）
  • 血培養（發燒 + 腹痛）

急性闌尾炎 Acute Appendicitis
急性闌尾炎是最常見需要急診手術的腹部疾病，終生罹病率約 7%。典型表現：臍周痛轉移至 RLQ + 厭食 + 噁心嘔吐 + 發燒。Alvarado 評分（MANTRELS）協助風險分層。CT 腹盆腔是診斷金標準（Se ~98%）；超音波適合兒童和孕婦（避免輻射）。穿孔闌尾炎：先抗生素引流，術前計分，數週後再切除（interval appendectomy）。部分非複雜性闌尾炎可抗生素保守治療，但復發率約 25-30%。
判斷框架
臨床評估 + Alvarado 評分

臨床表現

  • 典型進展：臍周鈍痛（內臟痛）→ 數小時後轉移至 RLQ（體腔壁痛，壁腹膜受累）→ 厭食 + 噁心嘔吐 → 發燒（通常低燒；高燒 >39°C 懷疑穿孔）
  • 非典型（30%）：骨盆位闌尾 → 恥骨上痛 / 直腸症狀；後位闌尾 → 腰背痛；老年 / 免疫抑制 → 症狀輕微；孕婦 → 闌尾位置上移

重要體徵

  • McBurney 點壓痛：臍和 ASIS 連線外 1/3 處壓痛（最常見）
  • Rovsing 徵：按壓 LLQ → RLQ 疼痛加重（腹膜刺激）
  • Psoas 徵：右大腿後伸 → RLQ 疼痛（後位闌尾）
  • Obturator 徵：右髖屈曲 + 內旋 → RLQ / 恥骨上疼痛（骨盆位闌尾）
  • Dunphy 徵：咳嗽或深呼吸時 RLQ 疼痛加重（腹膜刺激）

用藥指引
影像診斷 + 治療策略

影像選擇

  • CT 腹盆腔（含顯影劑）：首選（Se ~98%、Sp ~98%）；闌尾直徑 >6 mm + 周圍脂肪模糊 + 闌尾石；穿孔 → 游離氣體 / 膿瘍 / 蜂窩組織炎
  • 超音波：兒童和孕婦首選（避免輻射）；Se 86%、Sp 81%；操作者依賴；腸氣干擾；陽性（闌尾直徑 >6 mm、不可壓縮）有診斷意義；陰性不能排除
  • MRI：孕婦（超音波陰性但仍懷疑）；無輻射；耗時

治療

  • 非複雜性闌尾炎（首選）：腹腔鏡闌尾切除術（Laparoscopic appendectomy）→↓ 住院天數、傷口感染；術前 Cefazolin 單劑預防；術後抗生素不需常規使用
  • 抗生素保守治療（部分適選患者）：非複雜性闌尾炎 → Ertapenem 或 Cefotaxime + Metronidazole × 3-5 天；10 年內 40% 需手術；適合拒絕手術者或高手術風險患者
  • 複雜性闌尾炎（穿孔 + 膿瘍 <3 cm）：IV 抗生素（Pip/Tazo 或 Cefepime + Metronidazole）× 4-7 天；膿瘍 >3 cm → CT 引導引流；4-8 週後 interval appendectomy（選擇性）
  • 複雜性（穿孔 + 瀰漫性腹膜炎）：立刻手術

穿孔闌尾炎特徵

  • 高燒（>39°C）；WBC >15,000；腹膜炎徵象；CT：游離氣體 / 膿瘍 / 瀰漫脂肪浸潤
  • 穿孔後疼痛可暫時緩解（腹壓釋放）→ 再度加重（腹膜炎）
  • 闌尾石（Appendicolith）存在 → 穿孔風險↑；抗生素保守治療失敗率↑ → 通常建議手術
腸阻塞 Bowel Obstruction
腸阻塞分為小腸阻塞（SBO，75%）和大腸阻塞（LBO，25%），病因和處置不同。SBO 最常見原因是沾黏（60%，常見於先前腹部手術後）；LBO 最常見是大腸癌（60%）。關鍵鑑別：絞勒性阻塞（Strangulation）= 腸壁缺血 → 壞死 → 死亡率極高，需立刻手術。臨床線索：腹痛突然加劇 + 腹膜炎徵象 + 發燒 + 白血球升高 + Lactate ↑。「絞痛後消失的疼痛」= 警示，可能是腸壞死。
機轉說明
小腸阻塞（SBO）

病因

  • 沾黏（60%）：先前腹部手術（最常見）；腹膜炎後；子宮內膜異位症
  • 疝氣嵌頓（20%）：腹股溝疝氣、股疝氣、腹壁疝氣
  • 腫瘤（10%）：小腸原發性癌症（少見）；腹腔轉移癌（卵巢癌等）；淋巴瘤
  • 其他：Crohn 氏病（狹窄）；腸套疊（兒童）；膽石腸阻塞（Gallstone ileus）；腸扭轉（Volvulus）

臨床表現

  • 4 大症狀：腹痛（絞痛性，陣發）；噁心嘔吐（近端阻塞較嚴重）；腹脹；停止排便排氣
  • 高位 SBO（近端小腸）：嘔吐較早且頻繁；腹脹不明顯；嘔吐物為膽汁性
  • 低位 SBO（遠端小腸）：腹脹明顯；嘔吐物為糞便樣；疼痛較劇烈
  • 腸音：早期高亢（「梯形」蠕動聲）；晚期 / 絞勒性 → 腸音消失

影像診斷

  • 腹部 X 光（站立 + 仰臥）：擴張小腸（>3 cm）；氣液平面（「階梯狀」）；結腸無氣；Valvulae conniventes（小腸皺褶，橫越整個管腔）
  • CT 腹盆腔：首選（Se 93%、Sp 95%）；找到轉換點（Transition point）= 阻塞位置；腸壁增厚 / 強化缺失 = 缺血；腸繫膜脂肪條紋 = 絞勒性
  • 水溶性顯影劑（Gastrografin）通過試驗：口服後 24 小時看到顯影劑到達結腸 → 預測可保守治療成功（Se 96%、Sp 98%）

治療

  • 保守治療（沾黏性 SBO 首選）：NPO；NGT 減壓；IVF；電解質矯正；密切監測（每 4-6 小時腹部評估）；72 小時內改善 → 繼續觀察；Gastrografin 試驗
  • 立刻手術指徵：絞勒性阻塞；腹膜炎徵象；閉袢阻塞（Closed-loop obstruction）；疝氣嵌頓；腫瘤；保守治療 48-72 小時無改善
  • 手術方式：腹腔鏡或開腹沾黏鬆解；腸切除（若壞死）；疝氣修補
  • 閉袢阻塞（兩端均阻塞）→ 腸壓急速上升 → 快速進展到缺血壞死；CT 特徵：「U 形」或「C 形」擴張腸袢；立刻手術
緊急處置
大腸阻塞（LBO）+ 腸扭轉（Volvulus）

大腸阻塞（LBO）病因

  • 大腸癌（60%）：最常見；乙狀結腸和直腸最多；漸進性便祕 + 血便
  • 乙狀結腸扭轉（Sigmoid Volvulus，15%）：老年、慢性便秘、精神科病患（抗精神病藥物 → 腸蠕動↓）
  • 憩室炎狹窄；糞石阻塞；Ogilvie 症候群（假性阻塞）

LBO 特徵

  • 腹脹（通常更明顯）；便秘；嘔吐（較晚出現，糞便樣）；腹痛（不如 SBO 劇烈）
  • X 光：擴張結腸（>6 cm；盲腸 >9 cm = ↑ 穿孔風險）；「Coffee bean sign」= 乙狀結腸扭轉
  • 迴盲瓣勝任（Competent ileocecal valve）：形成閉袢阻塞 → 盲腸壓力急速升高 → 穿孔

LBO 處置

  • 腫瘤引起：內視鏡自擴張金屬支架（SEMS）→ 橋接至選擇性手術（較緊急腫瘤切除更安全）；或緊急手術（Hartmann 手術）
  • 乙狀結腸扭轉（無腹膜炎）：硬式乙狀結腸鏡減壓（成功率 ~85%）→ 擇期乙狀結腸切除（防復發）
  • 盲腸扭轉：通常需手術（右半結腸切除 or 固定術）
  • 腹膜炎 / 穿孔 → 立刻手術

Ogilvie 症候群（急性結腸假性阻塞）

  • 無機械性阻塞的急性結腸擴張；好發重症患者、手術後、電解質異常、藥物（鴉片類）
  • 治療：矯正誘因；Neostigmine 2 mg IV（禁忌：心動過緩、機械性阻塞、妊娠）→ 成功率 ~90%；頑固性 → 結腸鏡減壓
  • 盲腸 >12 cm → 穿孔高風險 → 緊急處置
腹膜炎 + 腸穿孔 Peritonitis & Bowel Perforation
腹膜炎是腹膜的發炎，分為原發性（自發性細菌性腹膜炎，SBP）和繼發性（腹腔器官穿孔 / 感染蔓延）。腸穿孔最常見原因：消化性潰瘍（最常見）、闌尾炎穿孔、憩室炎穿孔、腸缺血壞死、外傷。典型表現：突發劇烈腹痛（「刀刺樣」）+ 腹壁板狀硬 + 橫膈下游離氣體（直立 X 光）——任何疑似腸穿孔患者需立刻外科會診。SBP 是肝硬化腹水的嚴重併發症，死亡率 ~20%，診斷靠腹水 PMN >250/mm³。
緊急處置
腸穿孔（Bowel Perforation）
橫膈下游離氣體 + 腹膜炎徵象 → 立刻外科會診 + 禁食 + 廣效抗生素 + 備手術。

診斷

  • 直立腹部 X 光：橫膈下游離氣體（新月形）= 穿孔；Se 75%（陰性不排除穿孔）
  • CT 腹盆腔（首選）：Se >95%；找到穿孔位置；評估腹腔污染程度；引導手術策略
  • CBC + 完整代謝、Lactate、凝血、血型配血

初始處置

  • NPO；NGT 減壓；大口徑 IV × 2；積極補液（LR 或 NS）
  • 廣效抗生素（立刻）：Pip/Tazo 3.375 g IV q6h；或 Cefepime + Metronidazole；或 Meropenem（重症 / HAI）
  • Foley 導尿（監測 UOP）；NGT
  • 外科會診（立刻）：消化性潰瘍穿孔 → 手術（修補 ± 迷走神經切斷 ± 胃切除）；闌尾 / 憩室穿孔 + 瀰漫性腹膜炎 → 手術；腸缺血壞死 → 立刻手術

消化性潰瘍穿孔特殊考量

  • 症狀 <24 小時 + 穩定患者 → 修補術（Graham patch）
  • 老年 / 高風險手術 + 症狀 <12 小時 → 部分患者可試保守（NPO + NG + 抗生素）；CT 確認無持續滲漏
  • 術後：PPI + H. pylori 根除（若陽性）
用藥指引
自發性細菌性腹膜炎（SBP）
  • 定義：肝硬化腹水患者的細菌性腹水感染（無腸穿孔 / 腹腔感染源）
  • 病原體：E. coli（最常見）；Klebsiella；S. pneumoniae；腸球菌；厭氧菌少見
  • 發病機轉：腸道細菌移位（Bacterial translocation）→ 腸繫膜淋巴結 → 全身菌血症 → 腹水感染

診斷

  • 腹水 PMN（嗜中性球）≥250/mm³ = SBP 診斷標準（不需等培養結果）
  • 腹水培養陽性率僅 40-60%（因細菌量少）→ 床邊注入培養瓶 ↑ 敏感性
  • 臨床表現：發燒（69%）；腹痛（59%）；腹膜炎徵象（39%）；意識改變（54%）；或完全無症狀
  • 肝硬化腹水患者若有以下任一→ 診斷性腹腔穿刺：新發腹水；住院患者；發燒或腹痛；腎功能惡化；肝性腦病加重；GI 出血

治療

  • 抗生素（確認 PMN ≥250 立刻開始）：Cefotaxime 2 g IV q8h × 5 天（首選）；或 Ceftriaxone 1 g IV qd × 5 天；FQ（Ciprofloxacin）次選（↑ FQ 抗藥性）
  • Albumin IV（↓ 腎衰竭和死亡率）：診斷當天 1.5 g/kg IV；第 3 天 1 g/kg IV；Cr >1 mg/dL 或 BUN >30 或 Bili >4 時效益最明顯
  • 48 小時後追蹤腹腔穿刺：若 PMN 未下降 >25% → 考慮繼發性腹膜炎（腸穿孔）→ CT 評估

預防

  • 一級預防（高風險：腹水蛋白 <1.5 g/dL + 腎功能或肝功能嚴重受損）：Norfloxacin 400 mg qd 或 TMP-SMX
  • 二級預防（SBP 後）：Norfloxacin 400 mg qd 長期 → ↓ SBP 復發率（60-70%）；Ciprofloxacin 週一 / 四替代
  • GI 出血時：Ceftriaxone 1 g IV qd × 7 天（↓ SBP 發生率）
  • SBP 死亡率 ~20%；腎衰竭是最重要的死亡預測因子；Albumin 輸注是降低腎衰竭的關鍵
膽道疾病 Biliary Disease
膽道疾病的臨床譜從良性（膽絞痛）到嚴重（急性膽管炎合併敗血症）。Charcot 三聯症（RUQ 痛 + 發燒 / 寒顫 + 黃疸）= 急性膽管炎；Reynolds 五聯症（Charcot + 意識改變 + 低血壓）= 嚴重膽管炎，需緊急引流。急性膽囊炎診斷靠 Murphy 徵 + 超音波（首選影像）；治療靠腹腔鏡膽囊切除術（症狀 72 小時內早期手術優於延遲）。Tokyo Guidelines（TG18）是膽囊炎和膽管炎的國際分級標準。
用藥指引
急性膽囊炎（Acute Cholecystitis）

病因與類型

  • 結石性膽囊炎（90%）：膽石阻塞膽囊管 → 膽囊壁發炎；危險因素：5F（Female、Fat、Forty、Fertile、Fair）
  • 無石性膽囊炎（10%）：重症患者（長期 TPN、大手術後、燒傷、多重外傷）；膽囊壁缺血 + 膽汁積滯；死亡率更高（~30%）
  • 膽絞痛（Biliary Colic）：膽石一過性阻塞膽囊管 → 陣發 RUQ / 上腹痛（餐後 30-60 分鐘，持續 <6 小時）→ 自行緩解；無發燒；無 Murphy 徵

診斷（Tokyo Guidelines TG18）

  • 診斷標準（需 A 項 ≥1 + B 項 ≥1 + C 項確認）：
  • A. 局部症狀：Murphy 徵；RUQ 壓痛 / 腫塊 / 疼痛
  • B. 全身症狀：發燒；WBC ↑；CRP ↑
  • C. 影像確認：超音波（膽囊壁增厚 >4 mm、周圍積液、Murphy 徵）；CT；MRI

嚴重度分級（TG18）

分級標準治療時機
Grade I（輕度）無器官衰竭；無局部併發症早期腹腔鏡膽囊切除（症狀 <72 小時內）
Grade II（中度）WBC >18,000；RUQ 壓痛腫塊；症狀 >72 小時；局部嚴重發炎（壞疽 / 氣腫 / 肝膿瘍 / 膽汁性腹膜炎）早期手術（若技術可行）；或抗生素 + 膽囊引流後延期手術
Grade III（重度）器官衰竭（循環、神經、呼吸、腎、肝、凝血）緊急 / 早期膽囊引流（PTGBA 或內視鏡）→ 病況穩定後延期手術

抗生素（依嚴重度）

  • Grade I（社區輕中度）：Cefazolin 或 Cefotaxime；或 Ampicillin-sulbactam
  • Grade II-III（重度 / 院內）：Pip/Tazo 或 Meropenem（ESBL 風險）；+ Vancomycin（MRSA 高風險）
  • 術後抗生素：Grade I 術後通常不需要；Grade II-III 術後 4-7 天
緊急處置
急性膽管炎（Acute Cholangitis）
Reynolds 五聯症（Charcot + 意識改變 + 低血壓）= 嚴重膽管炎 → 立刻抗生素 + 緊急膽管引流（ERCP）。

病因與診斷

  • 病因：膽管結石（最常見）；膽管狹窄（術後 / 惡性）；ERCP 後；Oddi 括約肌功能障礙
  • Charcot 三聯症：RUQ 痛 + 發燒 / 寒顫 + 黃疸（三者同時出現率僅 50-70%）
  • Reynolds 五聯症：Charcot + 意識改變 + 低血壓 = 嚴重膽管炎（敗血症）
  • 實驗室：↑ Bili（直接為主）；↑ ALP、GGT；↑ ALT / AST（通常 <500）；WBC ↑；血培養（菌血症 40-60%）
  • 超音波：膽管擴張（>8 mm）；膽管結石；首選影像（快速可得）
  • MRCP / CT：超音波診斷不確定時；評估膽管解剖

嚴重度分級（TG18）與處置

分級標準引流時機
Grade I（輕度）無器官衰竭；對抗生素有反應選擇性 ERCP（72 小時內）；抗生素 48 小時後無反應 → 升為 Grade II
Grade II（中度）WBC >12,000 或 <4,000；高燒（>39°C）；年齡 >75；黃疸（Bili >5）；低白蛋白血症早期 ERCP（72 小時內）
Grade III（重度）器官衰竭（循環、神經、呼吸、腎、肝、凝血）緊急 ERCP（<24 小時）+ ICU 支持

抗生素 + 引流

  • 立刻廣效抗生素：Pip/Tazo 或 Ceftriaxone + Metronidazole；重症 / 院內 → Meropenem；依培養結果降階
  • ERCP（內視鏡逆行性膽胰管攝影）：取石 + 括約肌切開術；膽管引流（支架 / 鼻膽管）；失敗 → PTC（經皮穿肝膽管引流）
  • 膽囊結石合併膽管炎：ERCP 成功 → 擇期腹腔鏡膽囊切除術（通常 2-4 週後）
  • 急性膽管炎菌血症率高 → 血培養兩套（抗生素前）；膽汁培養（ERCP 時）指導降階治療
疝氣 Hernias
疝氣是腹腔內容物（腸管、網膜）通過腹壁薄弱點突出。嵌頓（Incarceration）= 無法還納；絞勒（Strangulation）= 血流受阻 → 腸壁缺血壞死——絞勒性疝氣是外科急症，死亡率 >20%（未及時手術）。臨床重點：腹股溝疝氣 vs 股疝氣的鑑別——股疝氣（Femoral Hernia）在鼠蹊韌帶下方，女性較常見，絞勒風險更高（50% vs 腹股溝疝氣 5-10%）。無症狀 / 可還納的腹股溝疝氣可擇期手術，但絞勒性 → 立刻手術。

腹股溝疝氣 vs 股疝氣解剖鑑別

  • 腹股溝疝氣：腫塊在鼠蹊韌帶上方；可進入陰囊；男性多
  • 股疝氣：腫塊在鼠蹊韌帶下方（股三角區）；不進入陰囊；女性多；常誤診為淋巴結腫大
  • 診斷不確定 → 超音波（高敏感度）；CT（評估嵌頓 / 絞勒）

  • Littre 疝氣：Meckel 憩室進入疝囊
  • Maydl 疝氣（W 疝氣）：兩段腸袢進入疝囊（W 形）→ 中間段腸管（在腹腔內）最先缺血

嵌頓疝氣徒手還納（Taxis）

  • 條件：無腹膜炎徵象；無皮膚變色；症狀 <4-6 小時；患者相對穩定
  • 方法：給予鎮痛（Morphine 或 Midazolam）+ 頭低腳高臥位（Trendelenburg）→ 持續輕柔向下壓迫 + 引導疝囊頸部（不要猛力推擠）
  • 失敗 / 絞勒徵象 → 立刻手術
  • 還納成功後 → 仍需住院觀察（腸道活性評估）→ 48 小時內擇期手術（防再嵌頓）

擇期手術修補

  • 腹腔鏡（首選）：TEP（完全腹膜外）或 TAPP（經腹膜）；↓ 術後疼痛；↓ 恢復時間；適合雙側或復發性疝氣
  • 開放式修補：Lichtenstein（無張力網片修補）= 最常見開放手術；Bassini / McVay（有張力，較少用）
  • 股疝氣：所有股疝氣均建議手術（高絞勒風險）；腹腔鏡 > 開放式（↓ 復發率）
  • 無症狀腹股溝疝氣：男性可觀察（急症風險低）；女性股疝氣建議手術（高絞勒風險）
消化道出血外科處置 GI Bleeding – Surgical Management
大多數消化道出血（80-90%）可由內視鏡止血，但10-20% 需要外科或介入放射治療。外科手術的適應症：內視鏡止血失敗（2 次嘗試後）、大量出血血流動力學不穩、反覆出血超過 24-48 小時。下消化道大量出血（LGIB）先用 CTA 定位出血點，再選擇介入放射栓塞（首選）或手術。Dieulafoy 病變（黏膜下大動脈異常破裂）= 少見但致命，內視鏡常難以發現，需高度懷疑。

LGIB 處置流程

  • 先排除 UGIB：鼻胃管灌洗（有血 = UGIB；膽汁無血 = LGIB 可能性高）；或上消化道內視鏡
  • 血流動力學穩定 → 結腸鏡：最佳診斷 + 治療工具；需腸道準備（PEG 4L）；急診結腸鏡（12-24 小時內）；電燒 / 注射止血 / 套扎
  • 大量出血 / 血流動力學不穩定 → CTA（首選定位）：出血速率 >0.5 mL/min 即可顯示；找到出血點 → IR 栓塞（首選）
  • 核醫紅血球掃描（Tc-99m RBC scan）：可偵測 0.1 mL/min 的慢性出血；定位較不精準；現已被 CTA 大幅取代

介入放射（IR）+ 手術

  • 動脈栓塞（TAE）：LGIB 首選介入；成功率 ~85%；腸缺血風險 <5%（結腸側支循環比小腸豐富）
  • 手術適應症（緊急）：TAE 失敗或不可行；大量出血 + 血流動力學持續不穩定；需輸血 >6-8 單位 / 24 小時；腹膜炎
  • 手術術式：已定位 → 節段性腸切除；未定位（盲目手術）→ 次全結腸切除（↑ 死亡率 ↑ 再出血）；盡量術前定位
  • 憩室出血 80% 自發性止血；但再出血率高（25-40%）→ 若憩室位置確認 → 選擇性節段切除可考慮（第二次發作後）

術後 3-5 天 Water（泌尿） 尿路感染（UTI）；留置導尿管相關 UA + 尿液培養；盡早移除 Foley 導尿管；依培養抗生素治療
術後 5-7 天 Wound（傷口） 手術部位感染（SSI）；表淺（傷口紅腫膿）vs 深層（筋膜下）vs 器官腔間隙感染 傷口評估；表淺 SSI → 開放引流（最重要）；深部 → CT 評估；抗生素
術後 7 天後 Walking（DVT / PE） 深部靜脈血栓（DVT）→ 肺栓塞（PE） 下肢超音波；CTA 胸部（PE）；預防：早期下床、彈力襪、LMWH 預防性抗凝
任何時間 Wonder drug（藥物熱） 藥物過敏反應（β-lactam 最常見）；輸血反應；惡性高體溫 審視藥物；停可疑藥物；藥物熱特點：無白血球增高；停藥後消退
留置管路 Walking（留置導管） 中央靜脈導管相關菌血症（CLABSI）；腹腔引流管感染 血培養（中央 + 周邊）；移除或更換導管；Vancomycin 覆蓋 GPC

術後早期（24 小時內）發燒特殊考量

  • 惡性高體溫（MH）：Succinylcholine / 揮發性麻醉劑誘發；RYR1 基因突變；↑↑ 體溫 + 肌肉強直 + ↑ CK + 代謝性酸中毒；治療：Dantrolene 2.5 mg/kg IV（立刻）+ 降溫
  • 輸血反應：溶血性輸血反應（ABO 不相容）→ 發燒 + 寒顫 + 血紅蛋白尿 + 腰痛 → 立刻停輸血；非溶血性發燒反應（最常見）→ 退燒藥
  • 鏈球菌 / 梭狀桿菌傷口感染：術後 24-48 小時即出現高燒 + 全身毒性 → 緊急傷口探查
用藥指引
術後腸麻痺 + 出血 + 吻合口滲漏 + 其他

術後腸麻痺（Postoperative Ileus）

  • 定義：腸蠕動功能暫時性受損；全身麻醉 + 腸道手術後正常生理反應
  • 正常恢復時間：小腸 24-48 小時；胃 24-48 小時；結腸 3-5 天
  • 延長性腸麻痺（>5 天）誘因：電解質異常（低鉀、低鎂）；鴉片類用量；腹腔感染；腸系膜缺血；再次手術
  • vs 機械性腸阻塞：腸麻痺 = 全腸道均無蠕動 + X 光整體腸脹氣；機械性 = 局部脹氣 + 轉換點 + 氣液平面不等
  • 處置：NPO；NGT 減壓；矯正電解質；早期下床；Chewing Gum（↑ 腸蠕動）；限制鴉片類；避免過度補液；Neostigmine（結腸麻痺 / Ogilvie）

術後出血

  • 早期（<24 小時）：技術問題（血管結紮滑脫、縫線鬆動）→ 通常需再次手術探查
  • 晚期（24-72 小時）：凝血異常（DIC、血小板減少）；抗凝藥物；感染侵蝕血管
  • 臨床監測：引流管顏色和量；腹圍；Hgb；血壓和心率；腹膜後出血 → 腰背痛 + Grey-Turner 徵（腰部瘀血）+ Cullen 徵（臍周瘀血）

吻合口滲漏（Anastomotic Leak）

  • 時間：通常術後 3-7 天（結腸 5-7 天）
  • 臨床表現：發燒；心搏過速；腹痛加重；引流管有糞便樣液體；腹膜炎徵象；WBC ↑
  • 診斷：CT 腹盆腔（游離氣體、積液、吻合口周圍滲漏）；水溶性顯影劑灌腸
  • 處置：小且局限性 → 引流 + 抗生素（保守）；大或瀰漫性腹膜炎 → 再次手術（拆除吻合 + 造口）
  • 危險因素：腸道準備不足；吻合口張力；血流供應不良；DM；類固醇；化療；貧血；低蛋白血症

其他常見術後問題

  • 術後噁心嘔吐（PONV）：預防：Ondansetron + Dexamethasone（術中）+ Scopolamine 貼片；治療：Ondansetron；Metoclopramide；Promethazine
  • 術後尿滯留：脊椎麻醉後最常見；鴉片類；前列腺肥大；→ 導尿管；α-blocker（Tamsulosin）
  • 術後認知功能障礙（POCD）：老年患者常見；全身麻醉後數天至數週；評估和矯正代謝因素；鼓勵早期活動
  • Clavien-Dindo 術後併發症分類：Grade I（無需治療）→ Grade II（藥物治療）→ Grade III（手術 / 內視鏡 / 介入放射）→ Grade IV（器官衰竭）→ Grade V（死亡）
  • 術前 RCRI（改良心臟風險指數）評估圍術期心臟事件風險：缺血性心臟病、充血性心衰竭、腦血管疾病、DM（需胰島素）、CKD（Cr >2）、高風險手術 → 每項 +1 分；≥3 分 = 高風險
