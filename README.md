## 📖 About

This is a peer-built, open-source survival guide for:
- **醫學生** — Medical students on clinical rotations
- **PGY (Post-Graduate Year)** — First and second year residents
- **Interns** starting their first hospital experience in Taiwan

Built by those who've been through it, for those just starting out.

## 🚀 Getting Started (Deploying Your Own)

### Option 1: GitHub Pages (Recommended — Free)

1. **Fork this repository**
   ```
   Click "Fork" in the top right
   ```

2. **Go to Settings → Pages**
   ```
   Source: Deploy from branch
   Branch: main / root
   ```

3. **Your site will be live at:**
   ```
   https://YOUR-USERNAME.github.io/taiwan-med-survival
   ```

### Option 2: Run Locally

```bash
git clone https://github.com/yourusername/taiwan-med-survival.git
cd taiwan-med-survival

# No build step needed — it's pure HTML/CSS/JS
open index.html
# or
python3 -m http.server 8080
# then visit http://localhost:8080
```

## 🤝 Contributing

This guide only gets better with more voices. If you have tips, corrections, or experiences to share, please contribute!

### Ways to Contribute

- **Add content**: Clinical tips, culture insights, admin wisdom
- **Fix errors**: Medical information can always be more accurate
- **Translate**: Help with English/Chinese bilingual sections
- **Design**: Improve accessibility or mobile experience

### How to Submit

1. Fork the repo
2. Create a branch: `git checkout -b add/your-tip-here`
3. Edit `index.html` or open a discussion
4. Submit a Pull Request

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed guidelines.
